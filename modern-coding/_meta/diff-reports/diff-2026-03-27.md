## 核心新发现详情

### 1. SDD Triangle 模型

**来源**: Drew Breunig @ MLOps Community "Coding Agents" Conference, 2026年3月3日

**核心洞察**:
```
         Spec（规范）
           △
          ╱ ╲
         ╱   ╲
        ╱     ╲
       ╱       ╲
      ╱─────────╲
   Tests ←──────→ Code
```

**关键学习**:
1. 实现代码有助于改进规范 —— 这是反馈循环，非直线
2. 测试和规范并非免费，是宝贵资产
3. 架构选择决定并行效率

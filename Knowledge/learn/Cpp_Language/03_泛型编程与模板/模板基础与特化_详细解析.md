# 模板基础与特化详细解析

> **核心结论**：C++模板是编译期多态的基石。理解模板参数推导、两阶段名称查找、特化优先级规则是掌握泛型编程的关键。**默认优先使用函数模板重载而非特化**，CTAD(C++17)能简化代码但需注意推导失败场景。

---

## 1. Why — 为什么模板如此重要

**结论先行**：模板让C++在零运行时开销的前提下实现泛型编程，是STL和标准库的基石。

### 1.1 泛型编程的核心价值

```cpp
// 没有模板：为每种类型写重复代码
int max_int(int a, int b) { return a > b ? a : b; }
double max_double(double a, double b) { return a > b ? a : b; }

// 有模板：一份代码，无限类型
// C++11/14/17/20/23 通用
template<typename T>
T max(T a, T b) { return a > b ? a : b; }
```

**模板的优势**：

| 特性 | 描述 | 对比方案 |
|-----|------|---------|
| **零开销抽象** | 编译期实例化，无运行时开销 | 虚函数有间接调用开销 |
| **类型安全** | 编译期类型检查 | void* 需要手动类型转换 |
| **代码复用** | 一份模板代码适配多种类型 | 宏定义无类型检查 |

### 1.2 模板在标准库中的应用

```cpp
// STL容器全是类模板
std::vector<int> vec;           // vector<T>
std::map<std::string, int> mp;  // map<Key, T>

// 算法是函数模板
std::sort(vec.begin(), vec.end());  // 适配各种迭代器类型

// 智能指针是类模板
std::shared_ptr<MyClass> ptr;   // shared_ptr<T>
```

## 2. What — 模板的核心概念

### 2.1 函数模板详解

#### 2.1.1 基本语法与实例化

```cpp
// C++98/03/11/14/17/20/23 通用
#include <iostream>

// 函数模板定义
template<typename T>
T add(T a, T b) {
    return a + b;
}

// 使用方式
int main() {
    // 隐式实例化：编译器推导 T = int
    int result1 = add(1, 2);
    
    // 显式实例化：明确指定类型
    double result2 = add<double>(1.5, 2.5);
    
    // 隐式实例化：T = const char*，注意：这里只是指针相加！
    const char* s = add("Hello, ", "World!");  // 危险！
    
    std::cout << "result1: " << result1 << std::endl;
    std::cout << "result2: " << result2 << std::endl;
    
    return 0;
}
```

#### 2.1.2 参数推导规则

```cpp
// C++11/14/17/20/23 通用
#include <iostream>
#include <type_traits>

template<typename T>
void deduce(T param) {  // 按值传递
    std::cout << "T = " << typeid(T).name() << std::endl;
    std::cout << "param type = " << typeid(param).name() << std::endl;
}

template<typename T>
void deduce_ref(T& param) {  // 按引用传递
    std::cout << "T = " << typeid(T).name() << std::endl;
    std::cout << "param type = " << typeid(param).name() << std::endl;
}

template<typename T>
void deduce_const_ref(const T& param) {  // 按const引用传递
    std::cout << "T = " << typeid(T).name() << std::endl;
}

template<typename T>
void deduce_rvalue_ref(T&& param) {  // 万能引用 (C++11)
    std::cout << "T = " << typeid(T).name() << std::endl;
}

int main() {
    int x = 10;
    const int cx = 20;
    
    // 按值传递：忽略顶层const
    deduce(x);    // T = int
    deduce(cx);   // T = int (const被忽略)
    
    // 按引用传递：保留cv限定符
    deduce_ref(x);   // T = int
    deduce_ref(cx);  // T = const int
    
    // 万能引用：引用折叠规则
    deduce_rvalue_ref(x);   // T = int& (左值 → T& &&折叠为T&)
    deduce_rvalue_ref(42);  // T = int  (右值 → T&&)
    
    return 0;
}
```

**参数推导规则总结**：

| 形参类型 | 实参 int | 实参 const int | 实参 int& | 实参 const int& |
|---------|---------|---------------|----------|----------------|
| T | int | int | int | int |
| T& | int | const int | int | const int |
| const T& | int | int | int | int |
| T&& | int& | const int& | int& | const int& |
| const T&& | int | int | int | int |

#### 2.1.3 函数模板重载解析

```cpp
// C++11/14/17/20/23 通用
#include <iostream>
#include <cstring>

// 通用模板
template<typename T>
T max(T a, T b) {
    std::cout << "通用模板" << std::endl;
    return a > b ? a : b;
}

// 特化版本（针对指针）
template<typename T>
T* max(T* a, T* b) {
    std::cout << "指针特化" << std::endl;
    return *a > *b ? a : b;
}

// 非模板重载（const char*）
const char* max(const char* a, const char* b) {
    std::cout << "非模板重载" << std::endl;
    return std::strcmp(a, b) > 0 ? a : b;
}

int main() {
    int a = 1, b = 2;
    max(a, b);  // 调用通用模板
    
    int *pa = &a, *pb = &b;
    max(pa, pb);  // 调用指针特化版本
    
    max("Hello", "World");  // 调用非模板重载
    
    return 0;
}
```

**重载解析优先级**：

1. 完全匹配的非模板函数
2. 完全匹配的模板特化
3. 需要类型转换的非模板函数
4. 需要类型转换的模板函数

### 2.2 类模板详解

#### 2.2.1 基本类模板

```cpp
// C++11/14/17/20/23 通用
#include <iostream>
#include <stdexcept>

template<typename T, size_t N>
class Array {
public:
    // 类型别名
    using value_type = T;
    using size_type = size_t;
    using reference = T&;
    using const_reference = const T&;
    
    // 成员函数
    reference operator[](size_type index) {
        if (index >= N) throw std::out_of_range("Index out of range");
        return data_[index];
    }
    
    const_reference operator[](size_type index) const {
        if (index >= N) throw std::out_of_range("Index out of range");
        return data_[index];
    }
    
    size_type size() const { return N; }
    
    T* begin() { return data_; }
    T* end() { return data_ + N; }
    const T* begin() const { return data_; }
    const T* end() const { return data_ + N; }
    
private:
    T data_[N];
};

int main() {
    Array<int, 5> arr;
    arr[0] = 10;
    arr[1] = 20;
    
    for (auto& elem : arr) {
        std::cout << elem << " ";
    }
    std::cout << std::endl;
    
    return 0;
}
```

#### 2.2.2 成员函数模板

```cpp
// C++11/14/17/20/23 通用
#include <iostream>
#include <utility>
#include <type_traits>

template<typename T>
class Container {
public:
    explicit Container(T val) : value_(std::move(val)) {}
    
    // 成员函数模板：支持不同类型赋值
    template<typename U>
    void assign(U&& val) {
        value_ = std::forward<U>(val);
    }
    
    // 成员函数模板：支持不同类型构造
    template<typename... Args>
    void emplace(Args&&... args) {
        value_ = T(std::forward<Args>(args)...);
    }
    
    const T& get() const { return value_; }
    
private:
    T value_;
};

// 嵌套类模板
template<typename T>
class Outer {
public:
    template<typename U>
    class Inner {
    public:
        void print() {
            std::cout << "Outer<T>::Inner<U>" << std::endl;
        }
    };
};

int main() {
    Container<int> c(42);
    c.assign(100);
    std::cout << c.get() << std::endl;
    
    // 使用嵌套类模板
    Outer<int>::Inner<double> inner;
    inner.print();
    
    return 0;
}
```

#### 2.2.3 友元模板

```cpp
// C++11/14/17/20/23 通用
#include <iostream>

// 前置声明
template<typename T> class Box;

template<typename T>
std::ostream& operator<<(std::ostream& os, const Box<T>& box);

template<typename T>
class Box {
public:
    explicit Box(T val) : value_(val) {}
    
    // 友元函数模板：每个实例化类型有自己的友元
    friend std::ostream& operator<< <T>(std::ostream& os, const Box<T>& box);
    
    // 友元函数模板：所有实例都是友元（C++11起）
    template<typename U>
    friend void printBox(const Box<U>& box) {
        std::cout << "Box value: " << box.value_ << std::endl;
    }
    
private:
    T value_;
};

// 友元函数模板定义
template<typename T>
std::ostream& operator<<(std::ostream& os, const Box<T>& box) {
    os << "Box(" << box.value_ << ")";
    return os;
}

int main() {
    Box<int> intBox(42);
    Box<double> doubleBox(3.14);
    
    std::cout << intBox << std::endl;
    std::cout << doubleBox << std::endl;
    
    printBox(intBox);
    printBox(doubleBox);
    
    return 0;
}
```

### 2.3 模板特化

#### 2.3.1 全特化（Full Specialization）

```cpp
// C++11/14/17/20/23 通用
#include <iostream>
#include <cstring>

// 通用模板
template<typename T>
class Storage {
public:
    void store(T val) {
        std::cout << "通用模板存储: " << val << std::endl;
        value_ = val;
    }
    T get() const { return value_; }
private:
    T value_;
};

// 全特化：针对 const char*
template<>
class Storage<const char*> {
public:
    void store(const char* val) {
        std::cout << "const char* 特化: 深拷贝字符串" << std::endl;
        size_t len = std::strlen(val) + 1;
        value_ = new char[len];
        std::strcpy(value_, val);
    }
    
    const char* get() const { return value_; }
    
    ~Storage() {
        delete[] value_;
    }
    
    // 需要自定义拷贝/移动操作（Rule of Five）
    Storage() : value_(nullptr) {}
    Storage(const Storage& other) { /* 深拷贝 */ }
    Storage(Storage&& other) noexcept { /* 移动 */ }
    Storage& operator=(const Storage& other) { /* 深拷贝 */ return *this; }
    Storage& operator=(Storage&& other) noexcept { /* 移动 */ return *this; }
    
private:
    char* value_;
};

// 函数模板全特化
template<typename T>
void process(T val) {
    std::cout << "通用处理: " << val << std::endl;
}

template<>
void process<int>(int val) {
    std::cout << "int 特化处理: " << val << std::endl;
}

int main() {
    Storage<int> intStorage;
    intStorage.store(42);
    
    Storage<const char*> strStorage;
    strStorage.store("Hello");
    std::cout << "Stored: " << strStorage.get() << std::endl;
    
    process(3.14);  // 通用模板
    process(42);    // int特化
    
    return 0;
}
```

#### 2.3.2 偏特化（Partial Specialization）

```cpp
// C++11/14/17/20/23 通用
#include <iostream>
#include <type_traits>

// 通用模板
template<typename T, typename U>
class Pair {
public:
    void identify() {
        std::cout << "通用 Pair<T, U>" << std::endl;
    }
};

// 偏特化1：两个类型相同
template<typename T>
class Pair<T, T> {
public:
    void identify() {
        std::cout << "偏特化 Pair<T, T> - 相同类型" << std::endl;
    }
};

// 偏特化2：第一个类型是指针
template<typename T, typename U>
class Pair<T*, U> {
public:
    void identify() {
        std::cout << "偏特化 Pair<T*, U> - 第一个类型是指针" << std::endl;
    }
};

// 偏特化3：第二个类型是引用
template<typename T, typename U>
class Pair<T, U&> {
public:
    void identify() {
        std::cout << "偏特化 Pair<T, U&> - 第二个类型是引用" << std::endl;
    }
};

// 偏特化4：两个类型都是指针
template<typename T, typename U>
class Pair<T*, U*> {
public:
    void identify() {
        std::cout << "偏特化 Pair<T*, U*> - 两个都是指针" << std::endl;
    }
};

// 函数模板没有偏特化！但可以用重载实现
template<typename T, typename U>
void process_pair(T a, U b) {
    std::cout << "通用 process_pair" << std::endl;
}

// 用重载模拟偏特化
template<typename T>
void process_pair(T* a, T* b) {
    std::cout << "重载 process_pair<T*, T*>" << std::endl;
}

int main() {
    Pair<int, double> p1;      // 通用
    Pair<int, int> p2;         // 偏特化1
    Pair<int*, double> p3;     // 偏特化2
    Pair<int, double&> p4;     // 偏特化3
    Pair<int*, double*> p5;    // 偏特化4（最特化匹配）
    
    p1.identify();
    p2.identify();
    p3.identify();
    p4.identify();
    p5.identify();
    
    int x = 1, y = 2;
    process_pair(x, y);        // 通用
    process_pair(&x, &y);      // 重载（模拟偏特化）
    
    return 0;
}
```

#### 2.3.3 特化匹配优先级

```cpp
// C++11/14/17/20/23 通用
#include <iostream>

// 匹配优先级（从高到低）：
// 1. 全特化
// 2. 最匹配的偏特化
// 3. 通用模板

template<typename T, typename U, typename V>
class Triple {
public:
    static void identify() { std::cout << "通用 Triple<T, U, V>" << std::endl; }
};

// 偏特化1：T = U
template<typename T, typename V>
class Triple<T, T, V> {
public:
    static void identify() { std::cout << "偏特化 Triple<T, T, V>" << std::endl; }
};

// 偏特化2：T = U = V
template<typename T>
class Triple<T, T, T> {
public:
    static void identify() { std::cout << "偏特化 Triple<T, T, T>" << std::endl; }
};

// 偏特化3：T = int
template<typename U, typename V>
class Triple<int, U, V> {
public:
    static void identify() { std::cout << "偏特化 Triple<int, U, V>" << std::endl; }
};

// 全特化
template<>
class Triple<int, int, int> {
public:
    static void identify() { std::cout << "全特化 Triple<int, int, int>" << std::endl; }
};

int main() {
    Triple<double, int, char>::identify();     // 通用
    Triple<double, double, char>::identify();  // 偏特化1
    Triple<double, double, double>::identify(); // 偏特化2（比偏特化1更特化）
    Triple<int, double, char>::identify();     // 偏特化3
    Triple<int, int, int>::identify();         // 全特化（最高优先级）
    
    return 0;
}
```

### 2.4 模板参数详解

#### 2.4.1 类型参数

```cpp
// C++11/14/17/20/23 通用
#include <iostream>

// typename 和 class 等价（C++17前）
template<typename T>  // 推荐
template<class T>     // 等价，但不表示必须是类

// 默认模板参数
template<typename T = int, typename U = double>
class DefaultParams {
public:

                  << ", sizeof(U)=" << sizeof(U) << std::endl;
    }

// 从右向左默认
template<typename T, typename U = T>  // OK
template<typename T = int, typename U>  // 错误：U没有默认值

int main() {
    DefaultParams<> d1;      // T=int, U=double
    DefaultParams<float> d2; // T=float, U=double
    DefaultParams<char, long> d3; // T=char, U=long
    
    d1.print();
    d2.print();
    d3.print();
    
    return 0;
}
```

### 2.5 C++17 类模板参数推导（CTAD）

#### 2.5.1 CTAD 基础

```cpp
// C++17 起支持
#include <iostream>
#include <utility>

// 定义类模板
template<typename T, typename U>
class Pair {
public:
    Pair(T first, U second) : first_(first), second_(second) {}
    
    T first() const { return first_; }
    U second() const { return second_; }
    
private:
    T first_;
    U second_;
};

// C++17前：必须显式指定类型
// Pair<int, double> p1(1, 2.5);

// C++17起：编译器自动推导
// Pair p2(1, 2.5);  // 推导为 Pair<int, double>

// 聚合类的CTAD
template<typename T, typename U>
struct AggPair {
    T first;
    U second;
};

// C++17起需要推导指引（Deduction Guide）
template<typename T, typename U>
AggPair(T, U) -> AggPair<T, U>;

// 拷贝推导指引（C++17）
template<typename T, typename U>
AggPair(const AggPair<T, U>&) -> AggPair<T, U>;

int main() {
    // CTAD 自动推导
    Pair p1(1, 2.5);           // Pair<int, double>
    Pair p2("hello", 42);      // Pair<const char*, int>
    
    // 聚合类CTAD
    AggPair ap1{1, 2.5};       // AggPair<int, double>
    
    // 花括号初始化
    Pair p3{1, 2.5};           // 同样支持
    
    std::cout << "p1: " << p1.first() << ", " << p1.second() << std::endl;
    
    return 0;
}
```

#### 2.5.2 推导指引（Deduction Guide）

```cpp
// C++17 起支持
#include <iostream>
#include <memory>
#include <vector>

// 类模板
template<typename T>
class Container {
public:
    Container(T val) : value_(val) {}
    Container(const Container& other) : value_(other.value_) {}
    
    T get() const { return value_; }
    
private:
    T value_;
};

// 显式推导指引：告诉编译器如何推导
template<typename T>
Container(T) -> Container<T>;

// 更复杂的推导指引
template<typename T>
Container(const Container<T>&) -> Container<T>;

// 用于std::vector的推导指引示例
template<typename Iter>
Container(Iter, Iter) -> Container<std::vector<typename std::iterator_traits<Iter>::value_type>>;

// 自定义类型转换推导
template<typename T>
class Wrapper {
public:
    template<typename U>
    Wrapper(U&& val) : value_(std::forward<U>(val)) {}
    
    T get() const { return value_; }
private:
    T value_;
};

// 推导指引：从std::unique_ptr推导底层类型
template<typename T>
Wrapper(std::unique_ptr<T>) -> Wrapper<T*>;

int main() {

    Container c2(3.14);         // Container<double>
    Container c3(c1);           // Container<int>（拷贝）
    
    auto ptr = std::make_unique<int>(100);

}
```

```cpp
        this->template template_func(x);

    obj.template template_func<int>(42);  // 正确

    public:

    typename Outer<T>::template Inner<U> obj;
    obj.print();
}
```

## 3. How — 实战代码示例

### 3.1 完整示例：类型安全的通用容器

```cpp
// C++17 完整示例
#include <iostream>
#include <stdexcept>
#include <utility>
#include <type_traits>

template<typename T, size_t N>
class FixedArray {
public:
    // 类型定义
    using value_type = T;
    using size_type = size_t;
    using reference = T&;
    using const_reference = const T&;
    using pointer = T*;
    using const_pointer = const T*;
    using iterator = T*;
    using const_iterator = const T*;
    
    // 默认构造
    FixedArray() = default;
    
    // 初始化列表构造
    FixedArray(std::initializer_list<T> init) {
        size_t i = 0;
        for (auto& val : init) {
            if (i >= N) break;
            data_[i++] = val;
        }
    }
    
    // 拷贝构造
    FixedArray(const FixedArray& other) {
        for (size_t i = 0; i < N; ++i) {
            data_[i] = other.data_[i];
        }
    }
    
    // 移动构造
    FixedArray(FixedArray&& other) noexcept {
        for (size_t i = 0; i < N; ++i) {
            data_[i] = std::move(other.data_[i]);
        }
    }
    
    // 拷贝赋值
    FixedArray& operator=(const FixedArray& other) {
        if (this != &other) {
            for (size_t i = 0; i < N; ++i) {
                data_[i] = other.data_[i];
            }
        }
        return *this;
    }
    
    // 移动赋值
    FixedArray& operator=(FixedArray&& other) noexcept {
        if (this != &other) {
            for (size_t i = 0; i < N; ++i) {
                data_[i] = std::move(other.data_[i]);
            }
        }
        return *this;
    }
    
    // 元素访问
    reference operator[](size_type index) {
        if (index >= N) throw std::out_of_range("Index out of range");
        return data_[index];
    }
    
    const_reference operator[](size_type index) const {
        if (index >= N) throw std::out_of_range("Index out of range");
        return data_[index];
    }
    
    reference at(size_type index) {
        if (index >= N) throw std::out_of_range("Index out of range");
        return data_[index];
    }
    
    const_reference at(size_type index) const {
        if (index >= N) throw std::out_of_range("Index out of range");
        return data_[index];
    }
    
    reference front() { return data_[0]; }
    const_reference front() const { return data_[0]; }
    reference back() { return data_[N-1]; }
    const_reference back() const { return data_[N-1]; }
    
    pointer data() { return data_; }
    const_pointer data() const { return data_; }
    
    // 迭代器
    iterator begin() { return data_; }
    iterator end() { return data_ + N; }
    const_iterator begin() const { return data_; }
    const_iterator end() const { return data_ + N; }
    const_iterator cbegin() const { return data_; }
    const_iterator cend() const { return data_ + N; }
    
    // 容量
    constexpr size_type size() const { return N; }
    constexpr size_type max_size() const { return N; }
    constexpr bool empty() const { return N == 0; }
    
    // 操作
    void fill(const T& value) {
        for (size_t i = 0; i < N; ++i) {
            data_[i] = value;
        }
    }
    
    void swap(FixedArray& other) noexcept {
        using std::swap;
        for (size_t i = 0; i < N; ++i) {
            swap(data_[i], other.data_[i]);
        }
    }
    
private:
    T data_[N];
};

// C++17 CTAD推导指引
template<typename T, typename... Rest>
FixedArray(T, Rest...) -> FixedArray<T, 1 + sizeof...(Rest)>;

// 测试
int main() {
    // CTAD自动推导
    FixedArray arr1{1, 2, 3, 4, 5};  // FixedArray<int, 5>
    
    // 显式指定
    FixedArray<double, 3> arr2{1.1, 2.2, 3.3};
    
    // 元素访问
    std::cout << "arr1[0] = " << arr1[0] << std::endl;
    std::cout << "arr1.size() = " << arr1.size() << std::endl;
    
    // 迭代
    for (const auto& elem : arr1) {
        std::cout << elem << " ";
    }
    std::cout << std::endl;
    
    // 填充
    arr2.fill(0.0);
    for (const auto& elem : arr2) {
        std::cout << elem << " ";
    }
    std::cout << std::endl;
    
    return 0;
}
```

## 4. 面试要点

### 4.1 高频面试题

#### Q1: 函数模板特化和重载有什么区别？什么时候用哪个？

**标准答案**：

```cpp
// 重载（推荐）
template<typename T> void process(T val);
template<typename T> void process(T* ptr);  // 重载
void process(const char* str);              // 非模板重载

// 特化（不推荐用于函数模板）
template<> void process<int>(int val);  // 全特化
```

**区别**：
1. **重载参与重载解析**，特化不参与
2. **重载可以有不同的返回类型**，特化必须与主模板返回类型兼容
3. **重载更灵活**，可以添加/删除参数

**建议**：函数模板优先使用重载，类模板使用特化。

#### Q2: 解释两阶段名称查找（Two-Phase Name Lookup）

**标准答案**：

```cpp
template<typename T>
void func(T x) {
    // 第一阶段：定义时查找
    global_func();  // 查找非依赖名称
    
    // 第二阶段：实例化时查找
    x.dependent_func();  // 查找依赖名称
    
    // 问题：依赖基类的成员
    // base_func();  // 错误！
    this->base_func();  // OK
}
```

**两阶段**：
1. **定义时（Definition Time）**：查找不依赖模板参数的名称
2. **实例化时（Instantiation Time）**：查找依赖模板参数的名称

**陷阱**：依赖基类的名称在第一阶段不可见，需要用 `this->` 或 `Base<T>::` 修饰。

#### Q3: CTAD 在什么情况下会失败？

**标准答案**：

```cpp
// 失败场景1：构造函数模板干扰
template<typename T>
class Bad {
public:
    Bad(T val) {}
    template<typename U> Bad(U val) {}  // 干扰CTAD
};
// 修复：添加推导指引
template<typename T> Bad(T) -> Bad<T>;

// 失败场景2：std::initializer_list歧义
template<typename T>
class Vec {
    Vec(size_t n, T val);           // (A)
    Vec(std::initializer_list<T>);  // (B)
};
// Vec v(10, 20);  // 歧义：(A)还是(B)？

// 失败场景3：非类型参数无法推导
template<typename T, size_t N>
class Array { ... };
// Array a{1,2,3};  // N无法推导
```

**解决方案**：显式指定模板参数或添加推导指引。

#### Q4: typename 和 class 在模板参数中有什么区别？

**标准答案**：

```cpp
// C++17前：等价
template<typename T> void f1();
template<class T> void f2();

// typename 的其他用途：告诉编译器这是类型
template<typename T>
void func() {
    typename T::iterator it;  // 必须加typename
}

// C++17：模板模板参数可以用typename
template<typename T, template<typename> typename Container>  // C++17起
class Stack;
```

**区别**：
- `typename` 语义更清晰，强调"类型"
- `class` 历史遗留，不表示必须是class类型
- `typename` 还可用于依赖名称消歧义

#### Q5: 模板特化的匹配优先级是怎样的？

**标准答案**：

```cpp
template<typename T, typename U> class A {};  // 1. 通用模板
template<typename T> class A<T, T> {};        // 2. 偏特化
template<> class A<int, int> {};              // 3. 全特化

// 匹配优先级：
// 全特化 > 偏特化 > 通用模板
```

**最特化原则**：选择约束最多、最具体的特化版本。

## 5. 最佳实践

### 5.1 模板设计原则

```cpp
// 1. 优先使用函数模板重载而非特化
template<typename T> void process(T val);           // 主模板
template<typename T> void process(T* ptr);          // 重载（推荐）
// template<> void process<int>(int val);           // 特化（不推荐）

// 2. 类模板使用偏特化处理特殊情况
template<typename T> class Container { ... };       // 通用
template<typename T> class Container<T*> { ... };   // 指针偏特化

// 3. 使用CRTP替代虚函数实现静态多态
template<typename Derived>
class Interface {
public:
    void algorithm() {
        static_cast<Derived*>(this)->implementation();
    }
};

// 4. 模板参数默认从右向左
template<typename T, typename U = int, typename V = double>  // OK
template<typename T = int, typename U, typename V>           // 错误
```

### 5.2 代码组织建议

```cpp
// 模板定义通常放在头文件中
template<typename T>
class MyTemplate {
public:
    void func();  // 声明
};

// 定义可以在类外，但必须在同一翻译单元
template<typename T>
void MyTemplate<T>::func() {  // 注意语法
    // 实现
}

// 显式实例化声明（减少编译时间）
// 在头文件中
extern template class MyTemplate<int>;
extern template void MyTemplate<double>::func();

// 在.cpp文件中
template class MyTemplate<int>;
template void MyTemplate<double>::func();
```

## 6. 常见陷阱与UB

### 6.1 依赖名称查找陷阱

```cpp
template<typename T>
class Base {
public:
    void func() {}
    int member;
};

template<typename T>
class Derived : public Base<T> {
public:
    void call() {
        // 陷阱1：直接调用基类成员
        // func();        // 错误：找不到func
        // int x = member; // 错误：找不到member
        
        // 正确做法
        this->func();           // OK
        int x = this->member;   // OK
        Base<T>::func();        // OK
    }
};
```

### 6.2 模板特化作用域陷阱

```cpp
// 陷阱：在命名空间外特化
namespace ns {
    template<typename T> class Foo {};
}

// 错误：必须在原命名空间内特化
// template<> class ns::Foo<int> {};

// 正确
template<> class ns::Foo<int> {};  // C++20前需要namespace ns {}

// C++20起推荐
namespace ns {
    template<> class Foo<int> {};
}
```

### 6.3 ODR（One Definition Rule）违规

```cpp
// 陷阱：在不同翻译单元定义不同的模板特化
// file1.cpp
template<typename T> T max(T a, T b) { return a > b ? a : b; }

// file2.cpp  
template<typename T> T max(T a, T b) { return a < b ? a : b; }  // 不同实现！

// 如果两个都实例化max<int>，行为未定义
```

### 6.4 模板递归实例化深度

```cpp
// 陷阱：无限递归实例化
template<int N>
struct Factorial {
    static constexpr int value = N * Factorial<N>::value;  // 错误：无限递归
};

// 正确：需要特化终止条件
template<int N>
struct Factorial {
    static constexpr int value = N * Factorial<N - 1>::value;
};

template<>
struct Factorial<0> {
    static constexpr int value = 1;
};
```

## 8. 总结

```
模板基础与特化核心要点
│
├── 函数模板
│   ├── 参数推导：按值/引用/万能引用的不同规则
│   ├── 重载解析：优先使用重载而非特化
│   └── 两阶段查找：定义时 + 实例化时
│
├── 类模板
│   ├── 成员函数模板、友元模板
│   ├── 全特化：完全定制某个类型
│   └── 偏特化：部分约束匹配
│
├── 模板参数
│   ├── 类型参数：typename/class
│   ├── 非类型参数：整数、指针、引用、C++20浮点/类类型
│   └── 模板模板参数：容器适配器的基础
│
├── C++17 CTAD
│   ├── 自动推导类模板参数
│   ├── 推导指引控制推导行为
│   └── 注意构造函数模板干扰
│
└── 依赖名称处理
    ├── typename：标识依赖类型名
    ├── template：标识依赖模板名
    └── this->：访问依赖基类成员
```

**核心原则**：
1. 函数模板用重载，类模板用特化
2. 依赖名称用 `typename`/`template`/`this->` 消歧义
3. C++17起善用CTAD简化代码
4. 理解两阶段查找避免编译错误

# 变参模板与折叠表达式详细解析

> **核心结论**：变参模板（Variadic Templates）是C++11引入的强大特性，C++17的折叠表达式（Fold Expressions）极大简化了参数包展开语法。掌握包展开规则和完美转发是编写现代C++库代码的关键技能。

---

## 1. Why — 为什么需要变参模板

**结论先行**：变参模板让模板可以接受任意数量和类型的参数，是`std::tuple`、`std::function`、`emplace`等核心设施的基础。

### 1.1 变参模板的价值

```cpp
// C++98/03：需要为不同参数数量写多个重载
void print() {}
void print(int a) { cout << a; }
void print(int a, int b) { cout << a << " " << b; }
void print(int a, int b, int c) { cout << a << " " << b << " " << c; }
// ... 无限重复

// C++11起：一个模板搞定任意数量参数
template<typename... Args>
void print(Args... args) {
    // 处理任意数量的args
}
```

### 1.2 变参模板的应用场景

| 场景 | 示例 | 说明 |
|-----|------|------|
| **通用工厂函数** | `std::make_unique`, `std::make_shared` | 完美转发参数 |
| **容器emplace** | `vector::emplace_back` | 原位构造 |
| **元组** | `std::tuple` | 异构类型集合 |
| **函数包装** | `std::function`, `std::bind` | 类型擦除 |
| **格式化输出** | `std::format`, `printf` | 类型安全格式化 |

## 2. What — 变参模板核心概念

### 2.1 Parameter Pack（参数包）

```cpp
// C++11/14/17/20/23 通用
#include <iostream>

// ============== 类型参数包 ==============
template<typename... Types>  // Types 是类型参数包
class Tuple {
    // Types 展开为：int, double, char 等
};

// 使用
Tuple<int> t1;                    // Types = {int}
Tuple<int, double> t2;            // Types = {int, double}
Tuple<int, double, char> t3;      // Types = {int, double, char}
Tuple<> t4;                       // Types = {} (空包)

// ============== 非类型参数包 ==============
template<int... Values>  // Values 是非类型参数包
struct Sum {
    // Values 展开为：1, 2, 3 等
};

Sum<1> s1;                    // Values = {1}
Sum<1, 2, 3> s2;              // Values = {1, 2, 3}
Sum<> s3;                     // Values = {} (空包)

// ============== 模板模板参数包 ==============
template<template<typename> class... Containers>  // Containers 是模板模板参数包
struct ContainerList {
    // Containers 展开为：vector, list, deque 等
};

template<typename T>
using ContainerListInstance = ContainerList<std::vector, std::list, std::deque>;

// ============== 函数参数包 ==============
template<typename... Args>  // Args 是类型参数包
void func(Args... args) {   // args 是函数参数包
    // args 展开为具体的参数值
}

func();                     // args = {}
func(1);                    // args = {1}
func(1, 2.5, "hello");      // args = {1, 2.5, "hello"}

// ============== 参数包的大小 ==============
template<typename... Args>
void print_size() {
    std::cout << "参数包大小: " << sizeof...(Args) << std::endl;
}

template<typename... Args>
void print_args_size(Args... args) {
    std::cout << "参数包大小: " << sizeof...(args) << std::endl;
}

int main() {
    print_size<int, double, char>();  // 输出: 3
    print_args_size(1, 2, 3, 4, 5);   // 输出: 5
    return 0;
}
```

### 2.2 包展开（Pack Expansion）

```cpp
// C++11/14/17/20/23 通用
#include <iostream>
#include <vector>
#include <memory>

// ============== 基本展开语法 ==============
// 模式... 表示展开参数包

template<typename... Args>
void print(Args... args) {
    // args... 展开为: arg1, arg2, arg3, ...
}

// ============== 各种展开上下文 ==============

// 1. 函数参数列表
template<typename... Args>
void func1(Args... args);  // void func1(T1 a1, T2 a2, ...)

// 2. 初始化列表
template<typename... Args>
void func2(Args... args) {
    int arr[] = {args...};  // int arr[] = {arg1, arg2, ...}
}

// 3. 模板参数列表
template<typename... Args>
class MyClass {
    std::tuple<Args...> data_;  // std::tuple<T1, T2, ...>
};

// 4. 基类列表
template<typename... Bases>
class Derived : public Bases... {  // class Derived : public B1, public B2, ...
public:
    Derived(Bases... bases) : Bases(bases)... {}  // B1(b1), B2(b2), ...
};

// 5. 成员初始化列表
template<typename... Args>
class Container {
public:
    Container(Args... args) : args_(args...) {}  // 初始化成员
private:
    std::tuple<Args...> args_;
};

// 6. Lambda捕获（C++14）
template<typename... Args>
auto make_lambda(Args... args) {
    return [args...]() {  // 捕获所有参数
        return (args + ...);  // C++17折叠表达式
    };
}

// 7. sizeof... 运算符
template<typename... Args>
constexpr std::size_t count = sizeof...(Args);  // 编译期常量

// ============== 模式展开 ==============
// 可以对每个参数应用模式

template<typename... Args>
void print_doubled(Args... args) {
    // 每个参数乘以2
    int arr[] = {(args * 2)...};  // {(arg1*2), (arg2*2), ...}
}

// 类型转换模式
template<typename... Args>
auto sum_int(Args... args) {
    return (static_cast<int>(args) + ...);  // 强制转为int后求和
}

// 函数调用模式
template<typename... Args>
void print_all(Args... args) {
    // C++17前需要递归或逗号表达式技巧
    // (std::cout << args << " ", ...);  // C++17折叠表达式
}

int main() {
    func2(1, 2, 3, 4, 5);  // arr = {1, 2, 3, 4, 5}
    
    struct Base1 { int x; Base1(int x) : x(x) {} };
    struct Base2 { int y; Base2(int y) : y(y) {} };
    Derived<Base1, Base2> d(Base1(1), Base2(2));
    
    std::cout << "count<int, double, char> = " << count<int, double, char> << std::endl;
    
    return 0;
}
```

### 2.3 递归展开（C++11/14）

```cpp
// C++11/14 完整示例
#include <iostream>

// ============== 递归终止条件 ==============
// 必须提供空参数包的特化或重载

// 终止条件：空参数包
void print() {
    std::cout << std::endl;
}

// 递归展开：处理第一个参数，递归处理剩余参数
template<typename T, typename... Rest>
void print(T first, Rest... rest) {
    std::cout << first;
    if (sizeof...(rest) > 0) {
        std::cout << ", ";
    }
    print(rest...);  // 递归调用，参数包逐渐减小
}

// ============== 递归求和 ==============
// 终止条件
constexpr int sum() {
    return 0;
}

// 递归展开
template<typename T, typename... Rest>
constexpr auto sum(T first, Rest... rest) {
    return first + sum(rest...);
}

// ============== 类型列表递归处理 ==============
template<typename... Types>
struct TypeList;

// 终止条件：空列表
template<>
struct TypeList<> {
    static constexpr std::size_t size = 0;
};

// 递归展开
template<typename First, typename... Rest>
struct TypeList<First, Rest...> {
    using head = First;
    using tail = TypeList<Rest...>;
    static constexpr std::size_t size = 1 + sizeof...(Rest);
};

// ============== 递归构建tuple ==============
template<typename... Args>
struct MyTuple;

template<>
struct MyTuple<> {};

template<typename Head, typename... Tail>
struct MyTuple<Head, Tail...> : MyTuple<Tail...> {
    Head value;
    
    MyTuple(Head h, Tail... t) : MyTuple<Tail...>(t...), value(h) {}
};

// ============== 递归完美转发 ==============
// C++11实现make_unique
template<typename T, typename... Args>
std::unique_ptr<T> make_unique(Args&&... args) {
    return std::unique_ptr<T>(new T(std::forward<Args>(args)...));
}

// ============== 逗号表达式技巧（C++11） ==============
// 在单条语句中展开参数包
template<typename... Args>
void print_cpp11(Args... args) {
    // 使用初始化列表和逗号表达式
    int dummy[] = {0, ((void)(std::cout << args << " "), 0)...};
    (void)dummy;  // 抑制未使用变量警告
    std::cout << std::endl;
}

// 更通用的foreach实现
template<typename Func, typename... Args>
void for_each(Func&& func, Args&&... args) {
    int dummy[] = {0, (func(std::forward<Args>(args)), 0)...};
    (void)dummy;
}

// ============== 测试 ==============
int main() {
    std::cout << "=== print ===" << std::endl;
    print(1, 2.5, "hello", 'c');
    
    std::cout << "=== sum ===" << std::endl;
    std::cout << "sum(1, 2, 3, 4, 5) = " << sum(1, 2, 3, 4, 5) << std::endl;
    
    std::cout << "=== TypeList ===" << std::endl;
    using MyTypes = TypeList<int, double, char>;
    std::cout << "size = " << MyTypes::size << std::endl;
    
    std::cout << "=== print_cpp11 ===" << std::endl;
    print_cpp11(1, 2.5, "hello");
    
    std::cout << "=== for_each ===" << std::endl;
    for_each([](auto x) { std::cout << x << " "; }, 1, 2.5, "hello");
    std::cout << std::endl;
    
    return 0;
}
```

### 2.4 C++17 折叠表达式

```cpp
// C++17 完整示例
#include <iostream>
#include <string>

// ============== 折叠表达式基本语法 ==============
// (pack op ...)      一元右折叠: (args1 op (args2 op (... op argsN)))
// (... op pack)      一元左折叠: (((args1 op args2) op ...) op argsN)
// (pack op ... op init) 二元右折叠: (args1 op (args2 op (... op (argsN op init))))
// (init op ... op pack) 二元左折叠: ((((init op args1) op args2) op ...) op argsN)

// ============== 一元右折叠 ==============
template<typename... Args>
auto sum_right(Args... args) {
    return (args + ...);  // (args1 + (args2 + (... + argsN)))
}

// 等价于：
// sum_right(1, 2, 3) → (1 + (2 + 3)) → 6

// ============== 一元左折叠 ==============
template<typename... Args>
auto sum_left(Args... args) {
    return (... + args);  // (((args1 + args2) + ...) + argsN)
}

// 等价于：
// sum_left(1, 2, 3) → ((1 + 2) + 3) → 6

// 注意：对于+、*等满足结合律的运算，左右折叠结果相同
// 但对于-、/等不满足结合律的运算，结果不同！

template<typename... Args>
auto sub_right(Args... args) {
    return (args - ...);  // (args1 - (args2 - (... - argsN)))
}
// sub_right(10, 3, 2) → (10 - (3 - 2)) → 9

template<typename... Args>
auto sub_left(Args... args) {
    return (... - args);  // (((args1 - args2) - ...) - argsN)
}
// sub_left(10, 3, 2) → ((10 - 3) - 2) → 5

// ============== 二元右折叠 ==============
template<typename... Args>
auto sum_with_init_right(Args... args) {
    return (args + ... + 0);  // (args1 + (args2 + (... + (argsN + 0))))
}

// 支持空参数包！
template<typename... Args>
auto sum_safe(Args... args) {
    return (args + ... + 0);  // 空包时返回0
}

// ============== 二元左折叠 ==============
template<typename... Args>
auto sum_with_init_left(Args... args) {
    return (0 + ... + args);  // ((((0 + args1) + args2) + ...) + argsN)
}

// ============== 常用运算符折叠 ==============

// 逻辑与：检查所有条件为真
template<typename... Args>
bool all(Args... args) {
    return (args && ...);  // (args1 && (args2 && (... && argsN)))
}

// 逻辑或：检查任一条件为真
template<typename... Args>
bool any(Args... args) {
    return (args || ...);  // (args1 || (args2 || (... || argsN)))
}

// 逗号运算符：顺序执行
template<typename... Args>
void print_fold(Args... args) {
    (std::cout << ... << args) << std::endl;  // 二元左折叠
}

template<typename... Args>
void print_with_space(Args... args) {
    ((std::cout << args << " "), ...) << std::endl;  // 逗号折叠
}

// 函数调用折叠
template<typename Func, typename... Args>
void call_all(Func&& func, Args&&... args) {
    (func(std::forward<Args>(args)), ...);  // 顺序调用所有函数
}

// ============== 复杂模式折叠 ==============

// 对参数应用函数后折叠
template<typename Func, typename... Args>
auto map_fold(Func&& func, Args... args) {
    return (func(args) + ...);  // func(arg1) + func(arg2) + ...
}

// 检查所有参数是否满足条件
template<typename T, typename... Args>
bool all_same_type() {
    return (std::is_same_v<T, Args> && ...);
}

// 检查所有参数是否可转换为某类型
template<typename T, typename... Args>
bool all_convertible() {
    return (std::is_convertible_v<Args, T> && ...);
}

// ============== 测试 ==============
int main() {
    std::cout << "=== sum ===" << std::endl;
    std::cout << "sum_right(1, 2, 3, 4, 5) = " << sum_right(1, 2, 3, 4, 5) << std::endl;
    std::cout << "sum_left(1, 2, 3, 4, 5) = " << sum_left(1, 2, 3, 4, 5) << std::endl;
    std::cout << "sum_safe() = " << sum_safe() << std::endl;  // 空包返回0
    
    std::cout << "\n=== sub ===" << std::endl;
    std::cout << "sub_right(10, 3, 2) = " << sub_right(10, 3, 2) << std::endl;  // 9
    std::cout << "sub_left(10, 3, 2) = " << sub_left(10, 3, 2) << std::endl;    // 5
    
    std::cout << "\n=== all/any ===" << std::endl;
    std::cout << "all(true, true, true) = " << all(true, true, true) << std::endl;
    std::cout << "all(true, false, true) = " << all(true, false, true) << std::endl;
    std::cout << "any(false, true, false) = " << any(false, true, false) << std::endl;
    std::cout << "any(false, false, false) = " << any(false, false, false) << std::endl;
    
    std::cout << "\n=== print ===" << std::endl;
    print_fold(1, 2.5, "hello");
    print_with_space(1, 2.5, "hello");
    
    std::cout << "\n=== map_fold ===" << std::endl;
    std::cout << "map_fold(square, 1, 2, 3) = " 
              << map_fold([](int x) { return x * x; }, 1, 2, 3) << std::endl;
    
    std::cout << "\n=== type checks ===" << std::endl;
    std::cout << "all_same_type<int, int, int>() = " 
              << all_same_type<int, int, int>() << std::endl;
    std::cout << "all_same_type<int, double>() = " 
              << all_same_type<int, double>() << std::endl;
    
    return 0;
}
```

### 2.5 sizeof... 运算符

```cpp
// C++11/14/17/20/23 通用
#include <iostream>
#include <type_traits>

// ============== sizeof... 基本用法 ==============
template<typename... Args>
void print_pack_size() {
    // sizeof...(Args) 返回类型参数包的大小
    std::cout << "类型参数包大小: " << sizeof...(Args) << std::endl;
}

template<typename... Args>
void print_args_size(Args... args) {
    // sizeof...(args) 返回函数参数包的大小
    std::cout << "函数参数包大小: " << sizeof...(args) << std::endl;
}

// ============== 编译期使用 ==============
template<typename... Args>
struct PackSize {
    static constexpr std::size_t value = sizeof...(Args);
};

// ============== 基于大小的特化 ==============
template<typename... Args>
struct Processor {
    static void process() {
        std::cout << "通用处理，参数数量: " << sizeof...(Args) << std::endl;
    }
};

// 特化：空参数包
template<>
struct Processor<> {
    static void process() {
        std::cout << "空参数包特化" << std::endl;
    }
};

// 特化：单参数
template<typename T>
struct Processor<T> {
    static void process() {
        std::cout << "单参数特化" << std::endl;
    }
};

// 特化：双参数
template<typename T, typename U>
struct Processor<T, U> {
    static void process() {
        std::cout << "双参数特化" << std::endl;
    }
};

// ============== 编译期if（C++17） ==============
template<typename... Args>
void conditional_process() {
    if constexpr (sizeof...(Args) == 0) {
        std::cout << "空参数处理" << std::endl;
    } else if constexpr (sizeof...(Args) == 1) {
        std::cout << "单参数处理" << std::endl;
    } else {
        std::cout << "多参数处理: " << sizeof...(Args) << "个" << std::endl;
    }
}

// ============== 与SFINAE结合 ==============
template<typename... Args>
std::enable_if_t<sizeof...(Args) == 0>
empty_only() {
    std::cout << "只有空参数包才能调用我" << std::endl;
}

template<typename... Args>
std::enable_if_t<(sizeof...(Args) > 0)>
non_empty_only() {
    std::cout << "非空参数包，数量: " << sizeof...(Args) << std::endl;
}

// ============== 测试 ==============
int main() {
    print_pack_size<>();
    print_pack_size<int>();
    print_pack_size<int, double, char>();
    
    print_args_size();
    print_args_size(1, 2, 3);
    
    std::cout << "PackSize<int, double>::value = " 
              << PackSize<int, double>::value << std::endl;
    
    Processor<>::process();
    Processor<int>::process();
    Processor<int, double>::process();
    Processor<int, double, char>::process();
    
    conditional_process<>();
    conditional_process<int>();
    conditional_process<int, double, char>();
    
    empty_only<>();
    non_empty_only<int>();
    non_empty_only<int, double>();
    
    return 0;
}
```

### 2.6 std::tuple 与变参模板

```cpp
// C++17 完整示例
#include <iostream>
#include <tuple>
#include <utility>
#include <type_traits>

// ============== tuple的基本使用 ==============
void basic_tuple_usage() {
    // 创建tuple
    std::tuple<int, double, std::string> t1(1, 2.5, "hello");
    
    // C++17 CTAD
    std::tuple t2(1, 2.5, "hello");  // 自动推导类型
    
    // 访问元素
    std::cout << "t1: " << std::get<0>(t1) << ", "
              << std::get<1>(t1) << ", "
              << std::get<2>(t1) << std::endl;
    
    // 通过类型访问（类型必须唯一）
    std::cout << "get<int>: " << std::get<int>(t1) << std::endl;
    
    // 修改元素
    std::get<0>(t1) = 100;
    
    // tuple大小
    constexpr std::size_t size = std::tuple_size_v<decltype(t1)>;
    std::cout << "tuple size: " << size << std::endl;
    
    // 元素类型
    using FirstType = std::tuple_element_t<0, decltype(t1)>;
    static_assert(std::is_same_v<FirstType, int>);
}

// ============== tuple的构造 ==============
template<typename... Args>
std::tuple<std::decay_t<Args>...> make_tuple_wrapper(Args&&... args) {
    return std::tuple<std::decay_t<Args>...>(std::forward<Args>(args)...);
}

// ============== std::apply ==============
void test_apply() {
    auto t = std::make_tuple(1, 2.5, std::string("hello"));
    
    // 将tuple展开为函数参数
    auto result = std::apply([](int a, double b, const std::string& c) {
        return a + b + c.length();
    }, t);
    
    std::cout << "apply result: " << result << std::endl;
}

// ============== std::make_from_tuple ==============
struct Point {
    int x, y;
    Point(int x, int y) : x(x), y(y) {}
};

void test_make_from_tuple() {
    auto coords = std::make_tuple(10, 20);
    Point p = std::make_from_tuple<Point>(coords);
    std::cout << "Point: (" << p.x << ", " << p.y << ")" << std::endl;
}

// ============== tuple_cat ==============
void test_tuple_cat() {
    auto t1 = std::make_tuple(1, 2);
    auto t2 = std::make_tuple(3.5, std::string("hello"));
    auto t3 = std::make_tuple('a');
    
    auto combined = std::tuple_cat(t1, t2, t3);
    
    std::cout << "combined tuple size: " << std::tuple_size_v<decltype(combined)> << std::endl;
    std::apply([](auto... args) {
        ((std::cout << args << " "), ...);
        std::cout << std::endl;
    }, combined);
}

// ============== 自定义tuple遍历 ==============
// 递归方式遍历
template<std::size_t I = 0, typename Func, typename... Args>
std::enable_if_t<I == sizeof...(Args)>
for_each_tuple(const std::tuple<Args...>&, Func&&) {}

template<std::size_t I = 0, typename Func, typename... Args>
std::enable_if_t<I < sizeof...(Args)>
for_each_tuple(const std::tuple<Args...>& t, Func&& func) {
    func(std::get<I>(t));
    for_each_tuple<I + 1>(t, std::forward<Func>(func));
}

// C++17折叠表达式方式（需要index_sequence辅助）
template<typename Func, typename... Args, std::size_t... Is>
void for_each_tuple_impl(const std::tuple<Args...>& t, Func&& func, 
                         std::index_sequence<Is...>) {
    (func(std::get<Is>(t)), ...);
}

template<typename Func, typename... Args>
void for_each_tuple_fold(const std::tuple<Args...>& t, Func&& func) {
    for_each_tuple_impl(t, std::forward<Func>(func), 
                        std::index_sequence_for<Args...>{});
}

// ============== 测试 ==============
int main() {
    std::cout << "=== basic_tuple_usage ===" << std::endl;
    basic_tuple_usage();
    
    std::cout << "\n=== make_tuple_wrapper ===" << std::endl;
    auto t = make_tuple_wrapper(1, 2.5, "hello");
    std::apply([](auto... args) {
        ((std::cout << args << " "), ...);
        std::cout << std::endl;
    }, t);
    
    std::cout << "\n=== test_apply ===" << std::endl;
    test_apply();
```

### 3.1 完整示例：类型安全的printf实现

```cpp
// C++17 完整示例
#include <iostream>
#include <sstream>
#include <stdexcept>
#include <string>
#include <type_traits>
#include <vector>

// ============== 格式化说明符解析 ==============
struct FormatSpec {
    int width = 0;
    int precision = -1;
    char fill = ' ';
    char align = '>';  // >右对齐, <左对齐, ^居中
    char type = 's';   // s字符串, d整数, f浮点, x十六进制
    bool alternate = false;
    bool sign = false;
    bool zero_pad = false;
};

// ============== 类型到格式字符的映射 ==============
template<typename T>
struct default_format {
    static constexpr char value = 's';
};

template<>
struct default_format<int> {
    static constexpr char value = 'd';
};

template<>
struct default_format<long> {
    static constexpr char value = 'd';
};

template<>
struct default_format<long long> {
    static constexpr char value = 'd';
};

template<>
struct default_format<float> {
    static constexpr char value = 'f';
};

template<>
struct default_format<double> {
    static constexpr char value = 'f';
};

template<>
struct default_format<char> {
    static constexpr char value = 'c';
};

template<>
struct default_format<bool> {
    static constexpr char value = 'd';
};

// ============== 格式化单个值 ==============
template<typename T>
std::string format_value(const T& val, const FormatSpec& spec) {
    std::ostringstream oss;
    
    // 设置填充和对齐
    if (spec.zero_pad && spec.align == '>') {
        oss << std::setfill('0') << std::setw(spec.width);
    } else {
        oss << std::setfill(spec.fill) << std::setw(spec.width);
    }
    
    if (spec.align == '<') oss << std::left;
    else if (spec.align == '>') oss << std::right;
    
    // 根据类型格式化
    char type = spec.type == 's' ? default_format<T>::value : spec.type;
    
    switch (type) {
        case 'd':
        case 'i':
            if constexpr (std::is_integral_v<T>) {
                if (spec.sign) oss << std::showpos;
                oss << val;
            } else {
                oss << val;
            }
            break;
            
        case 'x':
        case 'X':
            if constexpr (std::is_integral_v<T>) {
                if (spec.alternate) oss << "0x";
                oss << std::hex;
                if (type == 'X') oss << std::uppercase;
                oss << val;
            }
            break;
            
        case 'f':
        case 'F':
            if constexpr (std::is_floating_point_v<T>) {
                if (spec.precision >= 0) {
                    oss << std::fixed << std::setprecision(spec.precision);
                }
                oss << val;
            }
            break;
            
        case 'e':
        case 'E':
            if constexpr (std::is_floating_point_v<T>) {
                if (spec.precision >= 0) {
                    oss << std::scientific << std::setprecision(spec.precision);
                }
                oss << val;
            }
            break;
            
        case 'c':
            if constexpr (std::is_same_v<T, char>) {
                oss << val;
            }
            break;
            
        case 's':
        default:
            oss << val;
            break;
    }
    
    return oss.str();
}

// ============== 解析格式说明符 ==============
FormatSpec parse_spec(const char*& fmt) {
    FormatSpec spec;
    
    if (*fmt != ':') return spec;
    ++fmt;
    
    // 解析填充和对齐
    if (fmt[1] == '<' || fmt[1] == '>' || fmt[1] == '^') {
        spec.fill = *fmt;
        spec.align = fmt[1];
        fmt += 2;
    } else if (*fmt == '<' || *fmt == '>' || *fmt == '^') {
        spec.align = *fmt;
        ++fmt;
    }
    
    // 解析符号
    if (*fmt == '+') {
        spec.sign = true;
        ++fmt;
    }
    
    // 解析alternate形式
    if (*fmt == '#') {
        spec.alternate = true;
        ++fmt;
    }
    
    // 解析零填充
    if (*fmt == '0') {
        spec.zero_pad = true;
        spec.fill = '0';
        ++fmt;
    }
    
    // 解析宽度
    while (*fmt >= '0' && *fmt <= '9') {
        spec.width = spec.width * 10 + (*fmt - '0');
        ++fmt;
    }
    
    // 解析精度
    if (*fmt == '.') {
        ++fmt;
        spec.precision = 0;
        while (*fmt >= '0' && *fmt <= '9') {
            spec.precision = spec.precision * 10 + (*fmt - '0');
            ++fmt;
        }
    }
    
    // 解析类型
    if (*fmt && *fmt != '}') {
        spec.type = *fmt;
        ++fmt;
    }
    
    return spec;
}

// ============== 核心format函数 ==============
// 递归终止
template<typename... Args>
std::string format_impl(const char* fmt) {
    // 检查是否有未处理的格式标记
    while (*fmt) {
        if (*fmt == '{') {
            if (fmt[1] == '{') {
                fmt += 2;
            } else {
                throw std::runtime_error("格式字符串参数不足");
            }
        } else if (*fmt == '}') {
            if (fmt[1] == '}') {
                fmt += 2;
            } else {
                throw std::runtime_error("格式字符串中有孤立的'}'");
            }
        } else {
            ++fmt;
        }
    }
    return "";
}

// 递归展开
template<typename T, typename... Rest>
std::string format_impl(const char* fmt, T&& val, Rest&&... rest) {
    std::string result;
    
    while (*fmt) {
        if (*fmt == '{') {
            if (fmt[1] == '{') {
                result += '{';
                fmt += 2;
            } else {
                ++fmt;  // 跳过'{'
                
                // 解析格式说明符
                FormatSpec spec = parse_spec(fmt);
                
                if (*fmt != '}') {
                    throw std::runtime_error("格式字符串中缺少'}'");
                }
                ++fmt;  // 跳过'}'
                
                // 格式化当前值
                result += format_value(std::forward<T>(val), spec);
                
                // 递归处理剩余参数
                result += format_impl(fmt, std::forward<Rest>(rest)...);
                return result;
            }
        } else if (*fmt == '}') {
            if (fmt[1] == '}') {
                result += '}';
                fmt += 2;
            } else {
                throw std::runtime_error("格式字符串中有孤立的'}'");
            }
        } else {
            result += *fmt;
            ++fmt;
        }
    }
    
    throw std::runtime_error("格式字符串参数过多");
}

// 对外接口
template<typename... Args>
std::string format(const char* fmt, Args&&... args) {
    return format_impl(fmt, std::forward<Args>(args)...);
}

// ============== 测试 ==============
int main() {
    std::cout << "=== 基本格式化 ===" << std::endl;
    std::cout << format("Hello, {}!", "World") << std::endl;
    std::cout << format("Value: {}", 42) << std::endl;
    std::cout << format("Pi: {}", 3.14159) << std::endl;
    
    std::cout << "\n=== 宽度对齐 ===" << std::endl;
    std::cout << format("|[{:10}]|", "hello") << std::endl;     // 默认右对齐
    std::cout << format("|[{:<10}]|", "hello") << std::endl;    // 左对齐
    std::cout << format("|[{:>10}]|", "hello") << std::endl;    // 右对齐
    std::cout << format("|[{:^10}]|", "hello") << std::endl;    // 居中
    std::cout << format("|[{:*^10}]|", "hi") << std::endl;      // 自定义填充
    
    std::cout << "\n=== 数字格式化 ===" << std::endl;
    std::cout << format("|[{:10d}]|", 42) << std::endl;
    std::cout << format("|[{:010d}]|", 42) << std::endl;        // 零填充
    std::cout << format("|[{:+d}]|", 42) << std::endl;          // 显示正号
    std::cout << format("0x{:x}", 255) << std::endl;            // 十六进制
    std::cout << format("0x{:X}", 255) << std::endl;            // 大写十六进制
    
    std::cout << "\n=== 浮点数 ===" << std::endl;
    std::cout << format("{:.2f}", 3.14159) << std::endl;        // 2位小数
    std::cout << format("{:.4f}", 3.14159) << std::endl;        // 4位小数
    
    std::cout << "\n=== 多参数 ===" << std::endl;
    std::cout << format("{} + {} = {}", 1, 2, 3) << std::endl;
    std::cout << format("Name: {}, Age: {}, Score: {:.1f}", 
                        "Alice", 25, 95.5) << std::endl;
    
    std::cout << "\n=== 转义 ===" << std::endl;
    std::cout << format("{{}}") << std::endl;  // 输出 {}
    
    return 0;
}
```

### 4.1 高频面试题

#### Q1: 如何实现一个类型安全的make_unique？

**标准答案**：

```cpp
template<typename T, typename... Args>
std::unique_ptr<T> make_unique(Args&&... args) {
    return std::unique_ptr<T>(new T(std::forward<Args>(args)...));
}

// 关键点：
// 1. Args&&... 万能引用保持值类别
// 2. std::forward<Args>(args)... 完美转发
// 3. new T(...) 原位构造
```

#### Q2: 折叠表达式的四种形式是什么？

**标准答案**：

```cpp
// 一元右折叠: (args1 op (args2 op (... op argsN)))
(args + ...)      // 1 + (2 + 3)

// 一元左折叠: (((args1 op args2) op ...) op argsN)
(... + args)      // (1 + 2) + 3

// 二元右折叠: (args1 op (args2 op (... op (argsN op init))))
(args + ... + 0)  // 1 + (2 + (3 + 0))

// 二元左折叠: ((((init op args1) op args2) op ...) op argsN)
(0 + ... + args)  // (((0 + 1) + 2) + 3)
```

**注意**：二元折叠支持空参数包，一元折叠不支持（除非运算符有单位元）。

#### Q3: 包展开有哪些上下文？

**标准答案**：

```cpp
// 1. 函数参数列表
void func(Args... args);

// 2. 初始化列表
int arr[] = {args...};

// 3. 模板参数列表
template<Args...> class C;

// 4. 基类列表
class D : public Bases... {};

// 5. 成员初始化列表
D(Bases... bases) : Bases(bases)... {}

// 6. Lambda捕获
[args...]() { };

// 7. 折叠表达式
(args + ...);
```

#### Q4: 如何用递归展开实现print函数？

**标准答案**：

```cpp
// 终止条件
void print() {
    std::cout << std::endl;
}

// 递归展开
template<typename T, typename... Rest>
void print(T first, Rest... rest) {
    std::cout << first;
    if (sizeof...(rest) > 0) std::cout << " ";
    print(rest...);
}

// C++17折叠表达式版本
template<typename... Args>
void print_fold(Args... args) {
    (std::cout << ... << args) << std::endl;

// 带分隔符版本
template<typename... Args>
void print_sep(const char* sep, Args... args) {

    ((std::cout << args << (++n < sizeof...(Args) ? sep : "")), ...);
    std::cout << std::endl;
}
```

## 5. 最佳实践

### 5.1 变参模板设计原则

```cpp
// 1. 优先使用折叠表达式（C++17）
// 好：简洁清晰
template<typename... Args>
auto sum(Args... args) {
    return (args + ...);
}

// 不好：递归复杂
template<typename T>
T sum(T val) { return val; }

template<typename T, typename... Rest>
T sum(T first, Rest... rest) {
    return first + sum(rest...);
}

// 2. 完美转发时使用万能引用
template<typename... Args>
void forward_all(Args&&... args) {
    target(std::forward<Args>(args)...);
}

// 3. 使用sizeof...处理空包
template<typename... Args>
void process(Args... args) {
    if constexpr (sizeof...(args) == 0) {
        // 空包处理
    } else {
        // 非空处理
    }
}

// 4. 二元折叠处理可能为空的包
template<typename... Args>
auto sum_safe(Args... args) {
    return (args + ... + 0);  // 空包返回0
}
```

## 6. 常见陷阱与UB

### 6.1 参数包展开陷阱

```cpp
// 陷阱1：运算符优先级

template<typename... Args>
auto good_fold(Args... args) {
    return (args && ... && true);  // 空包返回true

template<typename... Args>
```

## 8. 总结

```
变参模板与折叠表达式核心要点
│
├── Parameter Pack（参数包）
│   ├── 类型参数包：typename... Args
│   ├── 非类型参数包：int... Values
│   ├── 模板模板参数包：template<typename> class... Containers
│   └── 函数参数包：Args... args
│
├── 包展开（Pack Expansion）
│   ├── 模式... 语法
│   ├── 展开上下文：函数参数、初始化列表、模板参数等
│   └── sizeof... 获取包大小
│
├── 递归展开（C++11/14）
│   ├── 必须提供终止条件
│   ├── 参数包逐渐减小
│   └── 逗号表达式技巧
│
├── C++17折叠表达式
│   ├── 一元右折叠：(args op ...)
│   ├── 一元左折叠：(... op args)
│   ├── 二元右折叠：(args op ... op init)
│   ├── 二元左折叠：(init op ... op args)
│   └── 支持空参数包（二元）
│
├── std::tuple
│   ├── 异构类型容器
│   ├── std::apply 展开调用
│   ├── std::make_from_tuple 构造对象
│   └── tuple_cat 连接
│
└── 完美转发
    ├── Args&&... 万能引用
    ├── std::forward<Args>(args)... 保持值类别
    ├── make_unique实现
    └── emplace系列函数原理
```

**核心原则**：
1. C++17起优先使用折叠表达式，代码更简洁
2. 完美转发使用 `Args&&...` + `std::forward<Args>(args)...`
3. emplace系列函数通过原位构造提升性能
4. 二元折叠表达式支持空参数包，更安全

## 1. ISP调试方法论

### 1.1 系统化调试流程

ISP（Image Signal Processor）调试是一个复杂的系统工程，需要遵循结构化的方法论。

```

│  ──────────────────────────────                             │

### 1.3 根因分析方法

```
5 Whys根因分析法示例：

问题现象：室内暖色光源下拍摄的照片偏蓝

Why 1: 为什么照片偏蓝？
→ 因为白平衡算法将暖色光源误判为日光

Why 2: 为什么白平衡会误判？
→ 因为AWB算法依赖的色温估计不准确

Why 3: 为什么色温估计不准确？
→ 因为场景中存在大面积暖色物体干扰

Why 4: 为什么暖色物体会干扰？
→ 因为AWB算法没有排除饱和色区域

Why 5: 为什么没有排除饱和色区域？
→ 因为饱和度阈值设置过高

根因：AWB算法的饱和度阈值参数设置不合理
解决：调低饱和度阈值，排除更多有色物体
```

### 2.1 RAW域可视化

```
RAW数据可视化方法：

┌─────────────────────────────────────────────────────────────┐
│  1. 单通道灰度显示                                           │
│  ─────────────────────                                      │
│  • 直接将RAW值映射到0-255灰度                                 │
│  • 可观察整体亮度分布                                         │
│  • 需要处理10/12/14bit到8bit的映射                           │
│                                                             │
│  raw_8bit = (raw_data >> (bit_depth - 8)).astype(uint8)     │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│  2. Bayer通道分离显示                                        │
│  ─────────────────────                                      │
│  • 将RGGB四个通道分别显示                                    │
│  • 便于检查各通道异常                                         │
│                                                             │
│  ┌─────┬─────┐                                             │
│  │  R  │  Gr │   →  四张独立灰度图                          │
│  ├─────┼─────┤                                             │
│  │  Gb │  B  │                                             │
│  └─────┴─────┘                                             │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│  3. 伪彩色显示                                               │
│  ─────────────────────                                      │
│  • 使用色彩映射增强可视化                                     │
│  • 热力图便于观察亮度分布                                     │
│  • 常用: jet, viridis, hot等colormap                        │
└─────────────────────────────────────────────────────────────┘
```

```python
import numpy as np
import matplotlib.pyplot as plt

def visualize_raw(raw_data, bit_depth=10, bayer_pattern='RGGB'):
    """
    RAW数据可视化
    """
    fig, axes = plt.subplots(2, 3, figsize=(15, 10))
    
    # 1. 原始灰度图
    raw_8bit = (raw_data / (2**bit_depth - 1) * 255).astype(np.uint8)
    axes[0, 0].imshow(raw_8bit, cmap='gray')
    axes[0, 0].set_title('RAW Grayscale')
    
    # 2. 热力图
    axes[0, 1].imshow(raw_8bit, cmap='hot')
    axes[0, 1].set_title('RAW Heatmap')
    
    # 3. 直方图
    axes[0, 2].hist(raw_data.flatten(), bins=256, color='gray')
    axes[0, 2].set_title('RAW Histogram')
    
    # 4-7. Bayer通道分离
    h, w = raw_data.shape
    channels = {
        'R': raw_data[0::2, 0::2],
        'Gr': raw_data[0::2, 1::2],
        'Gb': raw_data[1::2, 0::2],
        'B': raw_data[1::2, 1::2]
    }
    
    for idx, (name, ch) in enumerate(channels.items()):
        if idx < 3:
            ax = axes[1, idx]
        ch_8bit = (ch / (2**bit_depth - 1) * 255).astype(np.uint8)
        ax.imshow(ch_8bit, cmap='gray')
        ax.set_title(f'Channel {name}')
    
    plt.tight_layout()
    return fig
```

### 2.2 频谱分析（2D FFT）

```
频谱分析用于检测：

┌─────────────────────────────────────────────────────────────┐
│  • 周期性噪声（如传感器读出噪声）                              │
│  • 条带干扰（banding）                                       │
│  • 混叠伪影（aliasing artifacts）                            │
│  • 图像锐度/模糊程度                                         │
└─────────────────────────────────────────────────────────────┘

2D FFT分析流程：

原始图像        FFT变换        频谱图
┌─────────┐    ┌─────────┐    ┌─────────┐
│         │    │         │    │    ╋    │
│  Image  │ →  │  FFT2D  │ →  │   中心   │
│         │    │         │    │  为低频  │
└─────────┘    └─────────┘    └─────────┘

频谱特征解读：
• 中心亮点：DC分量（平均亮度）
• 水平/垂直线：对应方向的周期性噪声
• 高频能量分布：图像锐度指标
• 尖锐峰值：特定频率干扰
```

```python
import numpy as np
import cv2

def analyze_frequency(image):
    """
    2D FFT频谱分析
    """
    # 转灰度
    if len(image.shape) == 3:
        gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
    else:
        gray = image
    
    # 2D FFT
    f = np.fft.fft2(gray.astype(np.float32))
    fshift = np.fft.fftshift(f)  # 将零频移到中心
    
    # 幅度谱（对数变换便于显示）
    magnitude = 20 * np.log(np.abs(fshift) + 1)
    
    # 相位谱
    phase = np.angle(fshift)
    
    # 功率谱
    power = np.abs(fshift) ** 2
    
    return {
        'magnitude': magnitude,
        'phase': phase,
        'power': power,
        'fshift': fshift
    }

def detect_periodic_noise(fft_result, threshold=0.8):
    """
    检测周期性噪声
    找出频谱中的异常峰值
    """
    magnitude = fft_result['magnitude']
    h, w = magnitude.shape
    center = (h // 2, w // 2)
    
    # 排除DC分量
    mask = np.ones_like(magnitude)
    mask[center[0]-5:center[0]+5, center[1]-5:center[1]+5] = 0
    
    masked_mag = magnitude * mask
    
    # 找异常峰值
    max_val = np.max(masked_mag)
    mean_val = np.mean(masked_mag[masked_mag > 0])
    
    if max_val > mean_val * threshold:
        # 存在异常峰值，可能有周期性噪声
        peak_locations = np.where(masked_mag > max_val * 0.9)
        return True, peak_locations
    
    return False, None
```

### 2.3 差异图与热力图

```
差异图分析方法：

┌─────────────────────────────────────────────────────────────┐
│                    差异图生成                                │
│  ────────────────────────                                   │
│  diff = |image_A - image_B|                                 │
│                                                             │
│  用于：                                                      │
│  • 对比处理前后效果                                          │
│  • 对比不同算法结果                                          │
│  • 检测参数变化影响                                          │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│                    热力图增强                                │
│  ────────────────────────                                   │
│  • 将差异值映射到颜色                                        │
│  • 快速定位差异较大区域                                       │
│  • 支持多种colormap                                         │
│                                                             │
│  颜色编码：                                                  │
│  ┌────────────────────────────────────────┐                 │
│  │ 蓝──────青──────绿──────黄──────红     │                 │
│  │  ↑                               ↑     │                 │
│  │ 小差异                         大差异   │                 │
│  └────────────────────────────────────────┘                 │
└─────────────────────────────────────────────────────────────┘
```

```python
import numpy as np
import cv2

def generate_diff_map(img_a, img_b, amplify=5, colormap=cv2.COLORMAP_JET):
    """
    生成差异热力图
    
    参数:
        img_a, img_b: 两张对比图像
        amplify: 差异放大倍数
        colormap: 颜色映射方案
    """
    # 确保相同尺寸
    assert img_a.shape == img_b.shape
    
    # 计算绝对差异
    diff = cv2.absdiff(img_a, img_b)
    
    # 转灰度
    if len(diff.shape) == 3:
        diff_gray = cv2.cvtColor(diff, cv2.COLOR_BGR2GRAY)
    else:
        diff_gray = diff
    
    # 放大差异便于观察
    diff_amplified = np.clip(diff_gray * amplify, 0, 255).astype(np.uint8)
    
    # 应用颜色映射
    heatmap = cv2.applyColorMap(diff_amplified, colormap)
    
    # 统计信息
    stats = {
        'mean_diff': np.mean(diff_gray),
        'max_diff': np.max(diff_gray),
        'diff_ratio': np.sum(diff_gray > 10) / diff_gray.size
    }
    
    return heatmap, diff_amplified, stats

def overlay_diff_on_image(base_img, diff_heatmap, alpha=0.5):
    """
    将差异热力图叠加到原图上
    """
    if len(base_img.shape) == 2:
        base_img = cv2.cvtColor(base_img, cv2.COLOR_GRAY2BGR)
    
    overlay = cv2.addWeighted(base_img, 1-alpha, diff_heatmap, alpha, 0)
    return overlay
```

```python
import numpy as np
import cv2
import matplotlib.pyplot as plt

def analyze_histogram(image, mask=None):
    """
    综合直方图分析
    """
    fig, axes = plt.subplots(2, 2, figsize=(12, 10))
    
    # 1. 亮度直方图
    if len(image.shape) == 3:
        gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
    else:
        gray = image
    
    axes[0, 0].hist(gray.flatten(), bins=256, range=(0, 256), 
                    color='gray', alpha=0.7)
    axes[0, 0].set_title('Luminance Histogram')
    axes[0, 0].axvline(x=np.mean(gray), color='r', linestyle='--', 
                       label=f'Mean: {np.mean(gray):.1f}')
    axes[0, 0].legend()
    
    # 2. RGB三通道直方图
    if len(image.shape) == 3:
        colors = ('b', 'g', 'r')
        for i, color in enumerate(colors):
            hist = cv2.calcHist([image], [i], mask, [256], [0, 256])
            axes[0, 1].plot(hist, color=color)
        axes[0, 1].set_title('RGB Histogram')
    
    # 3. 累积直方图
    hist_cum = np.cumsum(cv2.calcHist([gray], [0], mask, [256], [0, 256]))
    hist_cum = hist_cum / hist_cum[-1]  # 归一化
    axes[1, 0].plot(hist_cum, color='gray')
    axes[1, 0].set_title('Cumulative Histogram')
    axes[1, 0].axhline(y=0.5, color='r', linestyle='--')
    
    # 4. 统计信息
    stats_text = f"""
    Luminance Statistics:
    ─────────────────────
    Mean: {np.mean(gray):.1f}
    Std: {np.std(gray):.1f}
    Min: {np.min(gray)}
    Max: {np.max(gray)}
    
    Percentiles:
    1%: {np.percentile(gray, 1):.1f}
    5%: {np.percentile(gray, 5):.1f}
    50%: {np.percentile(gray, 50):.1f}
    95%: {np.percentile(gray, 95):.1f}
    99%: {np.percentile(gray, 99):.1f}
    
    Clipping:
    Underexposed (<5): {np.sum(gray < 5) / gray.size * 100:.2f}%
    Overexposed (>250): {np.sum(gray > 250) / gray.size * 100:.2f}%
    """
    axes[1, 1].text(0.1, 0.5, stats_text, fontsize=10, 
                    family='monospace', verticalalignment='center')
    axes[1, 1].axis('off')
    
    plt.tight_layout()
    return fig
```

                         ↓

                         ↓

│  Step 7: 降噪参数 (NR)                                       │
│  ─────────────────────────                                  │

## 5. 常见问题诊断

### 5.1 色偏问题排查

```
色偏诊断决策树：

整体色偏？
    │
    ├─ 是 ─→ 检查白平衡
    │         │
    │         ├─ AWB算法问题？
    │         │    ├─ 场景干扰 → 调整AWB统计区域/阈值
    │         │    ├─ 色温估计错误 → 检查色温曲线
    │         │    └─ 增益计算错误 → 检查WB增益表
    │         │
    │         └─ CCM问题？
    │              └─ 重新标定CCM
    │
    └─ 局部色偏 ─→ 定位问题区域
                    │
                    ├─ 边缘色偏 → 镜头色差校正
                    │             (紫边/绿边)
                    │
                    ├─ 高光色偏 → Gamma曲线/饱和度处理
                    │
                    └─ 特定颜色偏差 → CCM调优

色偏量化分析：
1. 拍摄灰卡/ColorChecker
2. 测量R/G/B通道均值
3. 计算偏离度：
   • R偏移 = (R - G) / G
   • B偏移 = (B - G) / G
4. 理想值：R偏移 ≈ B偏移 ≈ 0
```

噪声类型识别：

│  ────────────────────                                       │

│  类型：                                                      │
│  • 列噪声/行噪声 → 读出电路问题                              │

```
清晰度问题排查：

模糊类型判断：
    │
    ├─ 整体模糊
    │      │
    │      ├─ 对焦问题？
    │      │    └─ 检查AF算法/对焦位置
    │      │

```python
# evaluate_quality.py 示例

import json
import os
from pathlib import Path
from skimage.metrics import structural_similarity as ssim
from skimage.metrics import peak_signal_noise_ratio as psnr
import cv2

def evaluate_folder(ref_dir, test_dir):
    """批量评估质量"""
    results = []
    
    for ref_path in Path(ref_dir).glob('*.png'):
        test_path = Path(test_dir) / ref_path.name
        
        if test_path.exists():
            ref_img = cv2.imread(str(ref_path))
            test_img = cv2.imread(str(test_path))
            
            ref_gray = cv2.cvtColor(ref_img, cv2.COLOR_BGR2GRAY)
            test_gray = cv2.cvtColor(test_img, cv2.COLOR_BGR2GRAY)
            
            result = {
                'filename': ref_path.name,
                'psnr': psnr(ref_img, test_img),
                'ssim': ssim(ref_gray, test_gray)
            }
            results.append(result)
    
    # 计算统计
    summary = {
        'total_images': len(results),
        'avg_psnr': sum(r['psnr'] for r in results) / len(results),
        'avg_ssim': sum(r['ssim'] for r in results) / len(results),
        'min_psnr': min(r['psnr'] for r in results),
        'min_ssim': min(r['ssim'] for r in results),
        'details': results
    }
    
    return summary

def check_threshold(report, psnr_th=45, ssim_th=0.98):
    """检查是否通过阈值"""
    passed = True
    failures = []
    
    for item in report['details']:
        if item['psnr'] < psnr_th:
            passed = False
            failures.append(f"{item['filename']}: PSNR={item['psnr']:.2f} < {psnr_th}")
        if item['ssim'] < ssim_th:
            passed = False
            failures.append(f"{item['filename']}: SSIM={item['ssim']:.4f} < {ssim_th}")
    
    return passed, failures
```

import time
import psutil
import numpy as np

def benchmark_isp(isp_func, test_images, warmup=10, iterations=100):
    """ISP性能基准测试"""
    
    # 预热
    for _ in range(warmup):
        _ = isp_func(test_images[0])
    
    # 性能测试
    times = []
    memory_usage = []
    
    for i in range(iterations):
        img = test_images[i % len(test_images)]
        
        # 记录内存
        mem_before = psutil.Process().memory_info().rss
        
        # 计时
        start = time.perf_counter()
        _ = isp_func(img)
        end = time.perf_counter()
        
        mem_after = psutil.Process().memory_info().rss
        
        times.append((end - start) * 1000)  # ms
        memory_usage.append(mem_after - mem_before)
    
    return {
        'avg_time_ms': np.mean(times),
        'std_time_ms': np.std(times),
        'p95_time_ms': np.percentile(times, 95),
        'p99_time_ms': np.percentile(times, 99),
        'fps': 1000 / np.mean(times),
        'peak_memory_mb': max(memory_usage) / 1024 / 1024
    }
```

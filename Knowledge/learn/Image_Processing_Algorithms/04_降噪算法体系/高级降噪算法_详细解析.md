## 1. 非局部均值（NLM）

### 1.1 Patch相似度思想

非局部均值（Non-Local Means）的核心思想是：**图像中存在大量重复的结构模式**，可以利用这些相似区域进行联合去噪。

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         NLM 核心思想                                         │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│   传统局部滤波：只使用空间邻近的像素                                         │
│   ┌─────────────────────────────────┐                                       │
│   │         ┌───┐                   │                                       │
│   │         │ P │ ← 只用周围像素    │                                       │
│   │         └───┘                   │                                       │
│   └─────────────────────────────────┘                                       │
│                                                                              │
│   NLM：使用整幅图像中所有相似区域                                            │
│   ┌─────────────────────────────────┐                                       │
│   │  ┌───┐           ┌───┐         │                                       │
│   │  │ A │    ┌───┐  │ C │         │  A, B, C 都是相似的patch              │
│   │  └───┘    │ B │  └───┘         │  虽然空间距离远，但都参与计算          │
│   │           └───┘                 │                                       │
│   │      ┌───┐                      │                                       │
│   │      │ P │ ← 目标像素          │                                       │
│   │      └───┘                      │                                       │
│   └─────────────────────────────────┘                                       │
│                                                                              │
│   直观理解：                                                                 │
│   • 重复纹理：砖墙、天空、织物上相似区域多                                   │
│   • 结构边缘：找到其他位置的相似边缘                                         │
│   • 噪声独立：相似patch的噪声不同，平均后降噪                                │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 1.3 权重计算公式

对于目标像素 $p$，其滤波结果为：

$$
\hat{I}(p) = \frac{1}{Z(p)} \sum_{q \in S(p)} w(p, q) \cdot I(q)
$$

其中权重基于patch相似度：

$$
w(p, q) = \exp\left( -\frac{\|P_p - P_q\|_2^2}{h^2} \right)
$$

- $P_p, P_q$：以 $p, q$ 为中心的patch
- $h$：滤波参数（控制平滑强度）
- $Z(p) = \sum_q w(p, q)$：归一化因子

### 1.4 实现代码

```python
import numpy as np
from scipy.ndimage import uniform_filter

def nlm_denoise(image, patch_size=7, search_size=21, h=10):
    """
    非局部均值降噪
    
    Args:
        image: 输入灰度图像
        patch_size: patch大小 (2f+1)
        search_size: 搜索窗口大小 (2r+1)
        h: 滤波参数
    
    Returns:
        降噪后图像
    """
    image = image.astype(np.float64)
    h, w = image.shape
    
    f = patch_size // 2  # patch半径
    r = search_size // 2  # 搜索半径
    
    # 填充图像
    padded = np.pad(image, ((r+f, r+f), (r+f, r+f)), mode='reflect')
    
    result = np.zeros_like(image)
    
    for y in range(h):
        for x in range(w):
            # 目标patch
            py, px = y + r + f, x + r + f
            target_patch = padded[py-f:py+f+1, px-f:px+f+1]
            
            # 搜索窗口内计算权重
            weights = []
            values = []
            
            for dy in range(-r, r+1):
                for dx in range(-r, r+1):
                    qy, qx = py + dy, px + dx
                    compare_patch = padded[qy-f:qy+f+1, qx-f:qx+f+1]
                    
                    # 计算patch距离
                    dist = np.sum((target_patch - compare_patch) ** 2)
                    weight = np.exp(-dist / (h * h * patch_size * patch_size))
                    
                    weights.append(weight)
                    values.append(padded[qy, qx])
            
            # 加权平均
            weights = np.array(weights)
            values = np.array(values)
            result[y, x] = np.sum(weights * values) / np.sum(weights)
    
    return np.clip(result, 0, 255).astype(np.uint8)
```

### 1.5 快速NLM实现

通过积分图和FFT加速：

```python
def fast_nlm_denoise(image, patch_size=7, search_size=21, h=10):
    """
    快速NLM实现（积分图加速）
    """
    from scipy.ndimage import convolve
    
    image = image.astype(np.float64)
    H, W = image.shape
    
    f = patch_size // 2
    r = search_size // 2
    
    padded = np.pad(image, ((r+f, r+f), (r+f, r+f)), mode='reflect')
    
    # 预计算：用均匀滤波器近似patch距离
    result = np.zeros_like(image)
    weight_sum = np.zeros_like(image)
    
    # 均匀核（用于计算patch内平均）
    patch_kernel = np.ones((patch_size, patch_size)) / (patch_size ** 2)
    
    for dy in range(-r, r+1):
        for dx in range(-r, r+1):
            # 位移后的图像
            shifted = np.roll(np.roll(padded, -dy, axis=0), -dx, axis=1)
            shifted = shifted[r+f:r+f+H, r+f:r+f+W]
            
            # 计算差的平方
            diff_sq = (image - shifted) ** 2
            
            # 在patch上求和（使用卷积近似）
            patch_dist = convolve(diff_sq, patch_kernel, mode='reflect')
            
            # 计算权重
            weights = np.exp(-patch_dist / (h * h))
            
            result += weights * shifted
            weight_sum += weights
    
    return np.clip(result / weight_sum, 0, 255).astype(np.uint8)
```

## 2. BM3D算法

### 2.1 三步框架：块匹配→3D变换→协同滤波

BM3D（Block-Matching and 3D filtering）是目前公认的**降噪金标准**：

```

├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│   ┌─────────────────────────────────────────────────────────────────┐       │
│   │                     第一阶段：基础估计                           │       │
│   │                                                                  │       │
│   │  1. 块匹配 (Block Matching)                                     │       │
│   │     • 对每个参考块，在搜索窗口找相似块                           │       │
│   │     • 相似块堆叠成3D组                                          │       │
│   │                                                                  │       │

│   │     • 转换到稀疏表示域                                          │       │

│   │                                                                  │       │
│   │  2. 维纳滤波                                                     │       │
```

### 2.2 块匹配详解

```python
def block_matching(image, ref_pos, block_size=8, search_size=39, 
                   max_matched=16, threshold=2500):
    """
    BM3D块匹配
    
    Args:
        image: 输入图像
        ref_pos: 参考块位置 (y, x)
        block_size: 块大小
        search_size: 搜索窗口大小
        max_matched: 最大匹配块数
        threshold: 距离阈值
    
    Returns:
        matched_positions: 相似块位置列表
    """
    h, w = image.shape
    ref_y, ref_x = ref_pos
    r = search_size // 2
    
    # 提取参考块
    ref_block = image[ref_y:ref_y+block_size, ref_x:ref_x+block_size]
    
    # 搜索相似块
    distances = []
    
    for dy in range(-r, r+1, 3):  # 步长3加速
        for dx in range(-r, r+1, 3):
            qy, qx = ref_y + dy, ref_x + dx
            
            if (0 <= qy < h - block_size and 
                0 <= qx < w - block_size):
                
                compare_block = image[qy:qy+block_size, qx:qx+block_size]
                
                # L2距离
                dist = np.sum((ref_block - compare_block) ** 2)
                
                if dist < threshold:
                    distances.append((dist, (qy, qx)))
    
    # 排序并选择top-K
    distances.sort(key=lambda x: x[0])
    matched = [pos for _, pos in distances[:max_matched]]
    
    return matched
```

### 2.3 3D变换与协同滤波

```python
import numpy as np
from scipy.fftpack import dct, idct

def bm3d_collaborative_filter(image, matched_positions, block_size=8, 
                               sigma=25, mode='hard'):
    """
    BM3D协同滤波
    
    Args:
        image: 输入图像
        matched_positions: 匹配块位置列表
        block_size: 块大小
        sigma: 噪声标准差
        mode: 'hard' (硬阈值) 或 'wiener' (维纳滤波)
    """
    n_matched = len(matched_positions)
    
    # 提取所有匹配块，组成3D数组
    group_3d = np.zeros((n_matched, block_size, block_size))
    for i, (y, x) in enumerate(matched_positions):
        group_3d[i] = image[y:y+block_size, x:x+block_size]
    
    # === 3D变换 ===
    # 步骤1: 对每个块做2D DCT
    for i in range(n_matched):
        group_3d[i] = dct(dct(group_3d[i], axis=0, norm='ortho'), 
                         axis=1, norm='ortho')
    
    # 步骤2: 沿第三维（堆叠方向）做1D变换（Haar小波）
    group_3d = haar_transform_1d(group_3d, axis=0)
    
    # === 滤波 ===
    if mode == 'hard':
        # 硬阈值
        threshold = 2.7 * sigma
        group_3d[np.abs(group_3d) < threshold] = 0
        
        # 计算非零系数数量（用于加权）
        n_nonzero = np.sum(group_3d != 0)
        weight = 1.0 / max(n_nonzero, 1)
        
    elif mode == 'wiener':
```

## 3. 小波域降噪

### 3.1 小波分解原理

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                        小波分解结构                                          │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│   原始图像                一级分解                 二级分解                  │
│   ┌─────────────┐        ┌──────┬──────┐         ┌───┬───┬──────┐         │
│   │             │        │  LL  │  HL  │         │LL2│HL2│      │         │
│   │             │   →    │      │      │    →    ├───┼───┤  HL  │         │
│   │   Image     │        ├──────┼──────┤         │LH2│HH2│      │         │
│   │             │        │  LH  │  HH  │         ├───┴───┼──────┤         │
│   │             │        │      │      │         │  LH   │  HH  │         │
│   └─────────────┘        └──────┴──────┘         └───────┴──────┘         │
│                                                                              │
│   LL: 低频近似（缩小版图像）                                                 │
│   HL: 水平高频（垂直边缘）                                                   │

│                                                                              │
│   噪声特性：                                                                 │
│   • 噪声主要分布在高频子带（HL, LH, HH）                                    │
│   • 信号能量集中在低频（LL）                                                 │
│   • 可以通过阈值处理高频子带来去噪                                           │
│                                                                              │

```

### 3.2 阈值方法

#### 硬阈值

$$
\hat{w} = \begin{cases}
w & |w| \geq T \\
0 & |w| < T
\end{cases}
$$

#### 软阈值

$$
\hat{w} = \begin{cases}
\text{sign}(w)(|w| - T) & |w| \geq T \\
0 & |w| < T
\end{cases}
$$

```python
import pywt
import numpy as np

def wavelet_denoise(image, wavelet='db4', level=3, threshold_type='soft'):
    """
    小波域降噪
    
    Args:
        image: 输入灰度图像
        wavelet: 小波基
        level: 分解层数
        threshold_type: 'soft' 或 'hard'
    """
    # 小波分解
    coeffs = pywt.wavedec2(image, wavelet, level=level)
    
    # 估计噪声标准差（使用最细尺度的对角子带）
    detail = coeffs[-1][2]  # HH子带
    sigma = np.median(np.abs(detail)) / 0.6745
    
    # 对每个高频子带进行阈值处理
    denoised_coeffs = [coeffs[0]]  # 保留低频
    
    for i in range(1, len(coeffs)):
        # 每层包含 (LH, HL, HH)
        denoised_detail = []
        for detail in coeffs[i]:
            # VisuShrink阈值
            threshold = sigma * np.sqrt(2 * np.log(detail.size))
            
            if threshold_type == 'soft':
                denoised = pywt.threshold(detail, threshold, mode='soft')
            else:
                denoised = pywt.threshold(detail, threshold, mode='hard')
            
            denoised_detail.append(denoised)
        
        denoised_coeffs.append(tuple(denoised_detail))
    
    # 小波重构
    result = pywt.waverec2(denoised_coeffs, wavelet)
    
    return np.clip(result, 0, 255).astype(np.uint8)
```

### 3.3 BayesShrink与VisuShrink

```
┌─────────────────────────────────────────────────────────────────┐
│                   阈值选择方法                                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│   VisuShrink（全局阈值）：                                       │
│   T = σ × √(2 log N)                                            │
│   • σ: 噪声标准差估计                                            │
│   • N: 图像像素数                                                │
│   • 简单但可能过度平滑                                           │
│                                                                  │
│   BayesShrink（自适应阈值）：                                    │
│   T_b = σ² / σ_s                                                │
│   • σ²: 噪声方差                                                 │
│   • σ_s: 信号标准差估计                                          │
│   σ_s = √(max(σ_y² - σ², 0))                                    │
│   • σ_y: 观测信号的标准差                                        │
│   • 每个子带独立计算阈值                                         │
│   • 保留更多细节                                                 │
│                                                                  │
│   性能对比（σ=25高斯噪声）：                                     │
│   VisuShrink: PSNR ~27 dB                                       │
│   BayesShrink: PSNR ~28.5 dB                                    │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

## 4. 深度学习降噪

### 4.1 DnCNN：残差学习降噪

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                          DnCNN 网络架构                                      │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│   核心思想：学习残差（噪声）而非直接学习干净图像                             │
│                                                                              │
│   Noisy ──→ [CNN] ──→ Residual ──→ (Noisy - Residual) = Clean              │
│                                                                              │
│   网络结构（17层）：                                                         │
│   ┌─────────────────────────────────────────────────────────────────┐       │
│   │  Input                                                           │       │
│   │    ↓                                                             │       │
│   │  [Conv 64×3×3] → [ReLU]                  第1层                   │       │
│   │    ↓                                                             │       │
│   │  [Conv 64×3×3] → [BN] → [ReLU]           第2-16层（重复15次）    │       │
│   │    ↓                                                             │       │
│   │  [Conv 1×3×3]                            第17层（输出残差）      │       │
│   │    ↓                                                             │       │
│   │  Output (Residual/Noise)                                        │       │
│   └─────────────────────────────────────────────────────────────────┘       │
│                                                                              │
│   关键设计：                                                                 │
│   • 残差学习：网络输出噪声而非图像，更容易学习                               │
│   • Batch Normalization：加速收敛，允许更高学习率                           │
│   • 无池化层：保持空间分辨率                                                 │
│   • 感受野：(17-1)×(3-1)+3 = 35像素                                        │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

```python
import torch
import torch.nn as nn

class DnCNN(nn.Module):
    """DnCNN降噪网络"""
    
    def __init__(self, channels=1, num_layers=17, features=64):
        super(DnCNN, self).__init__()
        
        layers = []
        
        # 第一层：Conv + ReLU
        layers.append(nn.Conv2d(channels, features, kernel_size=3, padding=1))
        layers.append(nn.ReLU(inplace=True))
        
        # 中间层：Conv + BN + ReLU
        for _ in range(num_layers - 2):
            layers.append(nn.Conv2d(features, features, kernel_size=3, padding=1))
            layers.append(nn.BatchNorm2d(features))
            layers.append(nn.ReLU(inplace=True))
        
        # 最后一层：Conv
        layers.append(nn.Conv2d(features, channels, kernel_size=3, padding=1))
        
        self.dncnn = nn.Sequential(*layers)
    
    def forward(self, x):
        # 输出残差（噪声）
        noise = self.dncnn(x)
        # 从输入中减去噪声
        return x - noise
```

### 4.2 FFDNet：灵活降噪水平

FFDNet允许用户指定降噪强度：

```python
class FFDNet(nn.Module):
    """FFDNet: 可调节降噪强度"""
    
    def __init__(self, channels=1, features=64, num_layers=15):
        super(FFDNet, self).__init__()
        
        # 输入：图像 + 噪声水平图
        input_channels = channels * 4 + 1  # 下采样后4通道 + sigma map
        
        layers = []
        layers.append(nn.Conv2d(input_channels, features, 3, padding=1))
        layers.append(nn.ReLU(inplace=True))
        
        for _ in range(num_layers - 2):
            layers.append(nn.Conv2d(features, features, 3, padding=1))
            layers.append(nn.BatchNorm2d(features))
            layers.append(nn.ReLU(inplace=True))
        
        layers.append(nn.Conv2d(features, channels * 4, 3, padding=1))
        
        self.net = nn.Sequential(*layers)
    
    def forward(self, x, sigma):
        """
        Args:
            x: 输入图像
            sigma: 噪声水平 (可以是标量或空间变化的map)
        """
        b, c, h, w = x.shape
        
        # 下采样（像素重排）
        x_down = pixel_unshuffle(x, 2)  # (b, c*4, h/2, w/2)
        
        # 构建sigma map
        sigma_map = torch.ones(b, 1, h//2, w//2, device=x.device) * sigma
        
        # 拼接
        x_input = torch.cat([x_down, sigma_map], dim=1)
        
        # 网络处理
        out = self.net(x_input)
        
        # 上采样
        out = pixel_shuffle(out, 2)
        
        return x - out  # 残差学习
```

### 4.4 盲降噪：Noise2Noise/Noise2Void

```python
# Noise2Noise 训练范式
# 不需要干净标签，只需要两张独立噪声的同场景图像

class Noise2NoiseTrainer:
    """Noise2Noise训练器"""
    
    def __init__(self, model, lr=1e-4):
        self.model = model
        self.optimizer = torch.optim.Adam(model.parameters(), lr=lr)
        self.criterion = nn.L1Loss()
    
    def train_step(self, noisy1, noisy2):
        """
        Args:
            noisy1: 第一张带噪图像
            noisy2: 同场景的第二张带噪图像（独立噪声）
        """
        self.optimizer.zero_grad()
        
        # 用noisy1预测noisy2
        pred = self.model(noisy1)
        loss = self.criterion(pred, noisy2)
        
        loss.backward()
        self.optimizer.step()
        
        return loss.item()

# Noise2Void 训练范式
# 只需要单张带噪图像，使用盲点网络

class Noise2VoidLoss:
    """Noise2Void损失函数"""
    
    def __init__(self, mask_ratio=0.5):
        self.mask_ratio = mask_ratio
    
    def __call__(self, pred, target, mask):
        """
        只计算被遮挡位置的损失
        网络无法看到这些位置的输入
        """
        loss = ((pred - target) ** 2 * mask).sum() / mask.sum()
        return loss
```

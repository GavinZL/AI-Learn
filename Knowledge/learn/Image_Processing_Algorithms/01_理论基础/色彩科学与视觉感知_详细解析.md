## 1. 人眼视觉系统

### 1.1 视锥与视杆细胞特性

#### 视网膜结构

人眼视网膜包含两种主要的光感受器：

```

│   │  ║ ║ ║ ║ ▌▌▌▌▌▌▌▌▌▌▌▌▌▌▌▌▌▌║ ║ ║ ║        │    │
│   │  ║ ║ ║ ║ ▌▌▌▌▌▌▌▌▌▌▌▌▌▌▌▌▌▌║ ║ ║ ║        │    │
```

#### 视锥细胞的三种类型

| 类型 | 峰值波长 | 颜色感知 | 相对数量 |
|------|---------|---------|---------|
| **L锥 (Long)** | ~564 nm | 红色敏感 | 约64% |
| **M锥 (Medium)** | ~533 nm | 绿色敏感 | 约32% |
| **S锥 (Short)** | ~437 nm | 蓝色敏感 | 约2% |

**直观类比**：三种视锥细胞就像三个颜色传感器，每个对特定波长范围的光最敏感。大脑通过比较这三个传感器的信号强度来"计算"出我们感知到的颜色。

#### 视锥与视杆的功能对比

| 特性 | 视锥细胞 | 视杆细胞 |
|------|---------|---------|
| **功能** | 明视觉（白天） | 暗视觉（夜间） |
| **颜色感知** | 有（三色视觉） | 无（单色） |
| **灵敏度** | 低（需强光） | 高（可感知单光子） |
| **空间分辨率** | 高 | 低 |
| **响应速度** | 快 | 慢 |
| **饱和阈值** | 高 | 低（强光下饱和） |

#### 光度函数V(λ)

明视觉下眼睛对不同波长光的相对灵敏度：

$$
V(\lambda) = \bar{y}(\lambda)
$$

峰值在555nm（黄绿光），这解释了为什么黄绿色在相同物理功率下看起来最亮。

```python
import numpy as np

# CIE 1931 光度函数 V(λ) 简化数据
def luminous_efficiency(wavelength_nm):
    """
    返回给定波长的明视觉光效率
    wavelength_nm: 波长（纳米）
    """
    # 近似公式（高斯拟合）
    peak = 555  # nm
    sigma = 50  # nm
    V = np.exp(-0.5 * ((wavelength_nm - peak) / sigma) ** 2)
    return V

# 示例：比较不同波长的感知亮度
wavelengths = [450, 500, 555, 600, 650]  # 蓝、青、绿黄、橙、红
for wl in wavelengths:
    eff = luminous_efficiency(wl)
    print(f"{wl}nm: 相对灵敏度 = {eff:.3f}")
```

### 1.3 亮度感知（Weber-Fechner定律）

#### Weber定律

人眼对亮度变化的感知是相对的，而非绝对的：

$$
\frac{\Delta I}{I} = k \quad (\text{常数})
$$

其中 $I$ 是背景亮度，$\Delta I$ 是可察觉的亮度差异（Just Noticeable Difference, JND）。

**直观类比**：在明亮的房间里，一支蜡烛的增加几乎察觉不到；但在黑暗中，同一支蜡烛却非常明显。我们感知的是"比例变化"，而非"绝对变化"。

#### Fechner定律

将Weber定律积分，得到感知亮度与物理亮度的关系：

$$
S = k \cdot \ln(I) + C
$$

感知是对数函数，这就是为什么：
- 显示器使用Gamma编码
- 音频使用分贝（dB）标度
- 相机曝光以"档"为单位

#### CSF在图像处理中的应用

| 应用 | 如何利用CSF |
|-----|------------|
| **JPEG压缩** | 高频DCT系数更粗量化 |
| **视频编码** | 根据CSF分配比特率 |
| **显示器校准** | 确定分辨率需求 |
| **降噪** | 在不敏感区域更多滤波 |
| **锐化** | 增强敏感频率范围 |

```python
def csf_mannos_sakrison(freq_cpd):
    """
    Mannos-Sakrison CSF模型
    freq_cpd: 空间频率（cycles per degree）
    返回: 对比度敏感度
    """
    # 参数
    a = 2.6
    b = 0.0192
    c = 0.114
    f0 = 8.0  # 峰值频率
    
    if freq_cpd == 0:
        return 0
    
    csf = a * (b + c * freq_cpd) * np.exp(-(c * freq_cpd) ** 1.1)
    return csf

# 计算不同频率的敏感度
frequencies = [0.5, 1, 2, 4, 8, 16, 32]
for f in frequencies:
    sensitivity = csf_mannos_sakrison(f)
    print(f"{f:4.1f} cpd: CSF = {sensitivity:.3f}")
```

## 2. CIE色度学

### 2.1 CIE 1931 XYZ标准观察者

#### XYZ三刺激值

任何颜色可以用三个数值X、Y、Z来描述：

$$
X = \int_{\lambda} S(\lambda) \cdot \bar{x}(\lambda) \, d\lambda
$$

$$
Y = \int_{\lambda} S(\lambda) \cdot \bar{y}(\lambda) \, d\lambda
$$

$$
Z = \int_{\lambda} S(\lambda) \cdot \bar{z}(\lambda) \, d\lambda
$$

其中 $S(\lambda)$ 是颜色的光谱功率分布。

**重要**：Y值同时代表**亮度**（luminance）。

```python
import numpy as np

def spectrum_to_xyz(spectrum, wavelengths, cmf_x, cmf_y, cmf_z):
    """
    将光谱转换为XYZ三刺激值
    spectrum: 光谱功率分布
    wavelengths: 对应波长
    cmf_x, cmf_y, cmf_z: 色匹配函数
    """
    # 数值积分
    X = np.trapz(spectrum * cmf_x, wavelengths)
    Y = np.trapz(spectrum * cmf_y, wavelengths)
    Z = np.trapz(spectrum * cmf_z, wavelengths)
    
    return X, Y, Z

def xyz_to_xy(X, Y, Z):
    """
    从XYZ计算xy色度坐标
    """
    total = X + Y + Z
    if total == 0:
        return 0, 0
    x = X / total
    y = Y / total
    return x, y
```

```python
def xy_to_cct_approximation(x, y):
```

## 3. 色差度量

### 3.1 CIE ΔE*ab（CIE76）

#### CIELAB颜色空间

Lab空间设计为感知均匀，L表示亮度，a和b表示色度：

$$
L^* = 116 \cdot f\left(\frac{Y}{Y_n}\right) - 16
$$

$$
a^* = 500 \cdot \left[f\left(\frac{X}{X_n}\right) - f\left(\frac{Y}{Y_n}\right)\right]
$$

$$
b^* = 200 \cdot \left[f\left(\frac{Y}{Y_n}\right) - f\left(\frac{Z}{Z_n}\right)\right]
$$

其中：

$$
f(t) = \begin{cases}
t^{1/3} & t > \delta^3 \\
\frac{t}{3\delta^2} + \frac{4}{29} & t \leq \delta^3
\end{cases}, \quad \delta = \frac{6}{29}
$$

#### CIE76色差公式

最简单的欧氏距离：

$$
\Delta E^*_{ab} = \sqrt{(\Delta L^*)^2 + (\Delta a^*)^2 + (\Delta b^*)^2}
$$

```
┌───────────────────────────────────────────────────────┐
│              CIELAB颜色空间结构                         │
├───────────────────────────────────────────────────────┤
│                                                       │
│               L* (亮度)                               │
│                 ↑ 100 (白)                            │
│                 │                                     │
│                 │       ●某颜色                       │
│                 │      ╱│                             │
│                 │    ╱  │ C*ab (彩度)                │
│                 │  ╱    │                             │
│   -a*(绿)←──────●──────→+a*(红)                       │
│                ╱│╲                                    │
│              ╱  │  ╲                                  │
│            ╱    │    ╲                                │
│          ╱      │      ╲                              │
│        ↓        │        ↓                            │
│    -b*(蓝)      0 (黑)    +b*(黄)                     │
│                                                       │
│   C*ab = √(a*² + b*²)  彩度（饱和度）                  │
│   h°ab = arctan(b*/a*) 色相角                         │
│                                                       │
└───────────────────────────────────────────────────────┘
```

```python
import numpy as np

def xyz_to_lab(X, Y, Z, Xn=95.047, Yn=100.0, Zn=108.883):
    """
    XYZ转CIELAB (D65白点)
    """
    def f(t):
        delta = 6/29
        if t > delta**3:
            return t**(1/3)
        else:
            return t / (3 * delta**2) + 4/29
    
    L = 116 * f(Y/Yn) - 16
    a = 500 * (f(X/Xn) - f(Y/Yn))
    b = 200 * (f(Y/Yn) - f(Z/Zn))
    
    return L, a, b

def delta_e_76(Lab1, Lab2):
    """
    CIE76色差公式
    """
    L1, a1, b1 = Lab1
    L2, a2, b2 = Lab2
    
    dE = np.sqrt((L2-L1)**2 + (a2-a1)**2 + (b2-b1)**2)
    return dE
```

|-------|---------|
| 0-1 | 难以察觉的差异 |
| 1-2 | 仔细观察可察觉 |
| 2-3.5 | 明显可察觉 |
| 3.5-5 | 显著差异 |
| >5 | 强烈差异 |

### 3.2 CIEDE2000

#### 改进动机

CIE76在蓝色区域和低彩度区域不够准确。CIEDE2000是目前最准确的色差公式。

$$
\Delta E_{00} = \sqrt{\left(\frac{\Delta L'}{k_L S_L}\right)^2 + \left(\frac{\Delta C'}{k_C S_C}\right)^2 + \left(\frac{\Delta H'}{k_H S_H}\right)^2 + R_T \frac{\Delta C'}{k_C S_C}\frac{\Delta H'}{k_H S_H}}
$$

```python
import numpy as np

def delta_e_2000(Lab1, Lab2, kL=1, kC=1, kH=1):
    """
    CIEDE2000色差公式
    Lab1, Lab2: (L*, a*, b*) 元组
    """
    L1, a1, b1 = Lab1
    L2, a2, b2 = Lab2
    
    # 计算平均L和彩度
    L_bar = (L1 + L2) / 2
    C1 = np.sqrt(a1**2 + b1**2)
    C2 = np.sqrt(a2**2 + b2**2)
    C_bar = (C1 + C2) / 2
    
    # G因子（调整a轴）
    G = 0.5 * (1 - np.sqrt(C_bar**7 / (C_bar**7 + 25**7)))
    
    a1_prime = a1 * (1 + G)
    a2_prime = a2 * (1 + G)
    
    C1_prime = np.sqrt(a1_prime**2 + b1**2)
    C2_prime = np.sqrt(a2_prime**2 + b2**2)
    C_bar_prime = (C1_prime + C2_prime) / 2
    
    # 色相角
    h1_prime = np.degrees(np.arctan2(b1, a1_prime)) % 360
    h2_prime = np.degrees(np.arctan2(b2, a2_prime)) % 360
    
    # 色相差
    if abs(h1_prime - h2_prime) <= 180:
        dh_prime = h2_prime - h1_prime
    elif h2_prime <= h1_prime:
        dh_prime = h2_prime - h1_prime + 360
    else:
        dh_prime = h2_prime - h1_prime - 360
    
    dH_prime = 2 * np.sqrt(C1_prime * C2_prime) * np.sin(np.radians(dh_prime/2))
    
    # 平均色相
    if abs(h1_prime - h2_prime) <= 180:
        H_bar_prime = (h1_prime + h2_prime) / 2
    elif h1_prime + h2_prime < 360:
        H_bar_prime = (h1_prime + h2_prime + 360) / 2
    else:
        H_bar_prime = (h1_prime + h2_prime - 360) / 2
    
    # 亮度、彩度、色相差
    dL_prime = L2 - L1
    dC_prime = C2_prime - C1_prime
    
    # 加权函数
    T = 1 - 0.17*np.cos(np.radians(H_bar_prime - 30)) + \
        0.24*np.cos(np.radians(2*H_bar_prime)) + \
        0.32*np.cos(np.radians(3*H_bar_prime + 6)) - \
        0.20*np.cos(np.radians(4*H_bar_prime - 63))
    
    SL = 1 + (0.015 * (L_bar - 50)**2) / np.sqrt(20 + (L_bar - 50)**2)
    SC = 1 + 0.045 * C_bar_prime
    SH = 1 + 0.015 * C_bar_prime * T
    
    # 旋转函数
    dTheta = 30 * np.exp(-((H_bar_prime - 275)/25)**2)
    RC = 2 * np.sqrt(C_bar_prime**7 / (C_bar_prime**7 + 25**7))
    RT = -RC * np.sin(np.radians(2*dTheta))
    
    # 最终色差
    dE = np.sqrt(
        (dL_prime / (kL * SL))**2 +
        (dC_prime / (kC * SC))**2 +
        (dH_prime / (kH * SH))**2 +
        RT * (dC_prime / (kC * SC)) * (dH_prime / (kH * SH))
    )
    
    return dE
```

### 3.3 感知均匀色彩空间（CIELAB/CIELUV）

#### CIELUV空间

另一种感知均匀空间，更适合发光体颜色：

$$
u' = \frac{4X}{X + 15Y + 3Z}, \quad v' = \frac{9Y}{X + 15Y + 3Z}
$$

$$
L^* = 116 \cdot f(Y/Y_n) - 16
$$

$$
u^* = 13L^*(u' - u'_n), \quad v^* = 13L^*(v' - v'_n)
$$

| 特性 | CIELAB | CIELUV |
|------|--------|--------|
| **设计目标** | 表面色 | 发光体 |
| **色差** | 更常用 | 较少使用 |
| **彩度定义** | $\sqrt{a^{*2}+b^{*2}}$ | $\sqrt{u^{*2}+v^{*2}}$ |
| **饱和度** | 无明确定义 | $s = 13\sqrt{(u'-u'_n)^2 + (v'-v'_n)^2}$ |
| **加法混色** | 不直观 | u'v'平面上线性 |

### 4.3 时域掩蔽

#### 原理

视觉系统对快速变化的内容不敏感，运动和场景切换可以掩蔽失真。

| 时域掩蔽类型 | 描述 | 应用 |
|-------------|------|------|
| **前向掩蔽** | 强刺激后一段时间内敏感度降低 | 场景切换后降低质量 |
| **后向掩蔽** | 强刺激前的弱刺激被掩盖 | 运动前几帧可压缩更多 |
| **运动掩蔽** | 快速运动区域失真不敏感 | 运动补偿编码优化 |

```python
def compute_masking_threshold(block, block_size=8):
    """
    计算DCT块的掩蔽阈值
    用于JPEG量化调整
    """
    # 计算块的亮度（DC分量近似）
    luminance = np.mean(block)
    
    # 计算块的活动度（纹理复杂度）
    activity = np.std(block)
    
    # 亮度掩蔽因子
    if luminance < 25:
        lum_mask = 1.0 + (25 - luminance) / 25  # 暗区提高阈值
    elif luminance > 230:
        lum_mask = 1.0 + (luminance - 230) / 25  # 亮区提高阈值
    else:
        lum_mask = 1.0
    
    # 纹理掩蔽因子（活动度越高，阈值越高）
    tex_mask = 1.0 + activity / 128
    
    # 组合掩蔽因子
    total_mask = lum_mask * tex_mask
    
    return total_mask

def adaptive_quantization(dct_block, base_qtable, masking_factor):
    """
    自适应量化：根据掩蔽因子调整量化步长
    """
    adjusted_qtable = base_qtable / masking_factor
    quantized = np.round(dct_block / adjusted_qtable)
    return quantized
```

#### CAT02色彩适应变换

将XYZ转换为LMS锥细胞响应，应用适应因子，再转回XYZ：

$$
\begin{bmatrix} L \\ M \\ S \end{bmatrix} = M_{CAT02} \begin{bmatrix} X \\ Y \\ Z \end{bmatrix}
$$

适应后：

$$
\begin{bmatrix} L_c \\ M_c \\ S_c \end{bmatrix} = \begin{bmatrix} D_{L} & 0 & 0 \\ 0 & D_{M} & 0 \\ 0 & 0 & D_{S} \end{bmatrix} \begin{bmatrix} L \\ M \\ S \end{bmatrix}
$$

```python
import numpy as np

# CAT02矩阵
M_CAT02 = np.array([
    [0.7328, 0.4296, -0.1624],
    [-0.7036, 1.6975, 0.0061],
    [0.0030, 0.0136, 0.9834]
])

M_CAT02_inv = np.linalg.inv(M_CAT02)

def chromatic_adaptation_cat02(XYZ, XYZ_w_src, XYZ_w_dst, D=1.0):
    """
    CAT02色彩适应变换
    XYZ: 待转换颜色的XYZ
    XYZ_w_src: 源光源白点XYZ
    XYZ_w_dst: 目标光源白点XYZ
    D: 适应程度 (0=无适应, 1=完全适应)
    """
    # 转到LMS
    LMS = M_CAT02 @ XYZ
    LMS_w_src = M_CAT02 @ XYZ_w_src
    LMS_w_dst = M_CAT02 @ XYZ_w_dst
    
    # 计算适应因子
    D_factors = D * (LMS_w_dst / LMS_w_src) + (1 - D)
    
    # 应用适应
    LMS_c = LMS * D_factors
    
    # 转回XYZ
    XYZ_c = M_CAT02_inv @ LMS_c
    
    return XYZ_c

# 示例：将D50光源下的颜色转换到D65
D50_white = np.array([96.422, 100.0, 82.521])
D65_white = np.array([95.047, 100.0, 108.883])
sample_xyz = np.array([50.0, 50.0, 50.0])

adapted = chromatic_adaptation_cat02(sample_xyz, D50_white, D65_white)
print(f"原始XYZ: {sample_xyz}")
print(f"适应后XYZ: {adapted}")
```

### 5.2 CIECAM02模型概述

#### 模型架构

CIECAM02是目前最先进的色彩外观模型，输出多个感知属性：

```

├───────────────────────────────────────────────────────┤

│   │ XYZ │  │ XYZ_w│  │ 观察条件参数  │               │
│   │试样 │  │白点 │  │ (La, Yb, c等) │               │
│   └──┬──┘  └──┬──┘  └───────┬───────┘               │
│      │        │              │                        │
│      ▼        ▼              ▼                        │
│   ┌────────────────────────────────┐                 │

│   └────────────────────────────────┘                 │
│                    │                                  │

│   ┌────────────────────────────────┐                 │
```

#### CIECAM02属性定义

| 属性 | 符号 | 描述 |
|------|------|------|
| **明度 (Lightness)** | J | 相对白点的亮度感知 |
| **亮度 (Brightness)** | Q | 绝对亮度感知 |
| **彩度 (Chroma)** | C | 相对白点的色彩强度 |
| **鲜艳度 (Colourfulness)** | M | 绝对色彩强度 |
| **饱和度 (Saturation)** | s | 色彩浓度 (M/Q) |
| **色相角 (Hue Angle)** | h | 色相位置 (0-360°) |
| **色相组成 (Hue Quadrature)** | H | 感知均匀的色相标度 |

```python
def ciecam02_forward(XYZ, XYZ_w, L_A=64, Y_b=20, c=0.69, N_c=1.0, F=1.0):
    """
    CIECAM02正向计算（简化版）
    XYZ: 试样XYZ
    XYZ_w: 白点XYZ
    L_A: 适应场亮度
    Y_b: 背景相对亮度
    c, N_c, F: 观察条件参数
    """
    # 1. 色彩适应
    D = F * (1 - (1/3.6) * np.exp((-L_A - 42) / 92))
    D = np.clip(D, 0, 1)
    
    # 转到LMS并适应
    LMS = M_CAT02 @ XYZ
    LMS_w = M_CAT02 @ XYZ_w
    
    LMS_c = LMS * ((100 * D / LMS_w) + 1 - D)
    LMS_w_c = LMS_w * ((100 * D / LMS_w) + 1 - D)
    
    # 2. 非线性响应压缩
    k = 1 / (5 * L_A + 1)
    F_L = 0.2 * k**4 * 5 * L_A + 0.1 * (1 - k**4)**2 * (5*L_A)**(1/3)
    
    def compress(LMS_c, F_L):
        LMS_p = (400 * (F_L * LMS_c / 100)**0.42) / (27.13 + (F_L * LMS_c / 100)**0.42) + 0.1
        return LMS_p
    
    LMS_p = compress(np.abs(LMS_c), F_L)
    
    # 3. 计算对手色通道
    a = LMS_p[0] - 12*LMS_p[1]/11 + LMS_p[2]/11
    b = (LMS_p[0] + LMS_p[1] - 2*LMS_p[2]) / 9
    
    # 4. 色相角
    h = np.degrees(np.arctan2(b, a)) % 360
    
    # 5. 无彩响应
    A = (2*LMS_p[0] + LMS_p[1] + LMS_p[2]/20 - 0.305)
    A_w = (2*compress(np.abs(LMS_w_c), F_L)[0] + 
           compress(np.abs(LMS_w_c), F_L)[1] + 
           compress(np.abs(LMS_w_c), F_L)[2]/20 - 0.305)
    
    # 6. 明度 J
    n = Y_b / XYZ_w[1]
    z = 1.48 + np.sqrt(n)
    J = 100 * (A / A_w)**(c * z)
    
    # 7. 彩度 C
    e_t = 0.25 * (np.cos(np.radians(h) + 2) + 3.8)
    t = (5e4/13 * N_c * e_t * np.sqrt(a**2 + b**2)) / (LMS_p[0] + LMS_p[1] + 21*LMS_p[2]/20)
    C = t**0.9 * (J/100)**0.5 * (1.64 - 0.29**n)**0.73
    
    return {'J': J, 'C': C, 'h': h}
```

## 1. 色彩管理系统概述

### 1.1 端到端色彩一致性的目标

色彩管理系统（CMS, Color Management System）的核心目标是确保图像在不同设备间传递时**颜色外观一致**。

```
┌─────────────────────────────────────────────────────────────────┐
│                色彩管理的必要性                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│   无色彩管理:                                                    │
│   ┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐     │
│   │ 相机    │ →  │ 显示器A │ ≠  │ 显示器B │ ≠  │ 打印机  │     │
│   │ RGB=    │    │ 偏暖    │    │ 偏冷    │    │ 暗淡    │     │
│   │(200,100,│    │         │    │         │    │         │     │
│   │ 50)     │    │         │    │         │    │         │     │
│   └─────────┘    └─────────┘    └─────────┘    └─────────┘     │
│        ↓              ↓              ↓              ↓           │
│      原始色         色彩1          色彩2          色彩3         │
│      (一个)        (不同)         (不同)         (不同)         │
│                                                                  │
│   有色彩管理:                                                    │
│   ┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐     │
│   │ 相机    │ →  │ 显示器A │ ≈  │ 显示器B │ ≈  │ 打印机  │     │
│   │ Profile │    │ Profile │    │ Profile │    │ Profile │     │
│   │    ↓    │    │    ↓    │    │    ↓    │    │    ↓    │     │
│   │   PCS   │ →  │   PCS   │ →  │   PCS   │ →  │   PCS   │     │
│   └─────────┘    └─────────┘    └─────────┘    └─────────┘     │
│        ↓              ↓              ↓              ↓           │
│      原始色         色彩1          色彩2          色彩3         │
│     (感知)        (相似)         (相似)         (相似)          │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

```python
import numpy as np

# Bradford色适应矩阵
BRADFORD_MATRIX = np.array([
    [0.8951,  0.2664, -0.1614],
    [-0.7502, 1.7135,  0.0367],
    [0.0389, -0.0685,  1.0296]
])

def chromatic_adaptation_bradford(xyz, src_white, dst_white):
    """
    Bradford色适应变换
    
    参数:
        xyz: 输入XYZ颜色
        src_white: 源白点XYZ
        dst_white: 目标白点XYZ
    """
    # 转换到LMS空间
    lms_src = BRADFORD_MATRIX @ src_white
    lms_dst = BRADFORD_MATRIX @ dst_white
    
    # 缩放矩阵
    scale = np.diag(lms_dst / lms_src)
    
    # 完整变换矩阵
    M = np.linalg.inv(BRADFORD_MATRIX) @ scale @ BRADFORD_MATRIX
    
    return M @ xyz

# D65到D50
D65 = np.array([0.95047, 1.00000, 1.08883])
D50 = np.array([0.96422, 1.00000, 0.82521])
```

## 3. 3D LUT

### 3.1 3D LUT的数据结构

3D LUT（Look-Up Table）是将RGB输入映射到RGB输出的三维查找表：

```
┌─────────────────────────────────────────────────────────────────┐
│                    3D LUT结构                                    │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│   概念:                                                          │
│   输入RGB (r, g, b) → 查表 → 输出RGB (r', g', b')              │
│                                                                  │
│   3D网格结构 (以17点LUT为例):                                   │
│                                                                  │
│          B                                                       │
│          ↑                                                       │
│          │    ┌───────────────────┐                             │
│          │   ╱                   ╱│                             │
│          │  ╱   17×17×17网格   ╱ │                             │
│          │ ╱                   ╱  │                             │
│          │┌───────────────────┐   │                             │
│          ││                   │   │                             │
│          ││    每个节点存储   │   │                             │
│          ││    (R', G', B')   │   │                             │
```

### 3.2 三线性插值算法

对于非网格点的输入，需要进行插值：

```
┌─────────────────────────────────────────────────────────────────┐
│                   三线性插值                                     │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│   给定点P(r,g,b)，找到包含它的立方体:                           │
│                                                                  │
│         V011 ●─────────────────● V111                           │
│             ╱│                ╱│                                │
│            ╱ │               ╱ │                                │
│           ╱  │              ╱  │                                │
│     V001 ●─────────────────● V101                               │
│          │   │             │   │                                │
│          │   │      ● P    │   │                                │
│          │   │             │   │                                │
│          │ V010 ●──────────│───● V110                           │
│          │  ╱              │  ╱                                 │
│          │ ╱               │ ╱                                  │
│          │╱                │╱                                   │
│     V000 ●─────────────────● V100                               │
│                                                                  │
│   插值步骤:                                                      │
│   1. 计算局部坐标 (u, v, w) ∈ [0,1]³                           │
│   2. 沿R方向插值 (4次)                                          │
│   3. 沿G方向插值 (2次)                                          │
│   4. 沿B方向插值 (1次)                                          │
│                                                                  │
│   公式:                                                          │
│   f(u,v,w) = (1-u)(1-v)(1-w)V₀₀₀ + u(1-v)(1-w)V₁₀₀            │
│            + (1-u)v(1-w)V₀₁₀     + uv(1-w)V₁₁₀                 │
│            + (1-u)(1-v)w V₀₀₁    + u(1-v)w V₁₀₁                │
│            + (1-u)vw V₀₁₁        + uvw V₁₁₁                    │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 3.3 四面体插值算法

四面体插值比三线性更高效（6次插值vs 7次）：

```python
def tetrahedral_interpolation(lut, r, g, b, lut_size):
    """
    四面体插值实现
    
    参数:
        lut: 3D LUT数组 [size, size, size, 3]
        r, g, b: 归一化输入 [0, 1]
        lut_size: LUT尺寸
    """
    # 缩放到LUT索引
    r_scaled = r * (lut_size - 1)
    g_scaled = g * (lut_size - 1)
    b_scaled = b * (lut_size - 1)
    
    # 基础索引
    r0 = int(r_scaled)
    g0 = int(g_scaled)
    b0 = int(b_scaled)
    
    # 小数部分
    fr = r_scaled - r0
    fg = g_scaled - g0
    fb = b_scaled - b0
    
    # 确保不越界
    r1 = min(r0 + 1, lut_size - 1)
    g1 = min(g0 + 1, lut_size - 1)
    b1 = min(b0 + 1, lut_size - 1)
    
    # 获取立方体顶点
    c000 = lut[r0, g0, b0]
    c001 = lut[r0, g0, b1]
    c010 = lut[r0, g1, b0]
    c011 = lut[r0, g1, b1]
    c100 = lut[r1, g0, b0]
    c101 = lut[r1, g0, b1]
    c110 = lut[r1, g1, b0]
    c111 = lut[r1, g1, b1]
    
    # 确定四面体并插值
    if fr > fg:
        if fg > fb:
            # 四面体 1: c000, c100, c110, c111
            return (1-fr)*c000 + (fr-fg)*c100 + (fg-fb)*c110 + fb*c111
        elif fr > fb:
            # 四面体 2: c000, c100, c101, c111
            return (1-fr)*c000 + (fr-fb)*c100 + (fb-fg)*c101 + fg*c111
        else:
            # 四面体 3: c000, c001, c101, c111
            return (1-fb)*c000 + (fb-fr)*c001 + (fr-fg)*c101 + fg*c111
    else:
        if fg > fb:
            if fr > fb:
                # 四面体 4: c000, c010, c110, c111
                return (1-fg)*c000 + (fg-fr)*c010 + (fr-fb)*c110 + fb*c111
            else:
                # 四面体 5: c000, c010, c011, c111
                return (1-fg)*c000 + (fg-fb)*c010 + (fb-fr)*c011 + fr*c111
        else:
            # 四面体 6: c000, c001, c011, c111
            return (1-fb)*c000 + (fb-fg)*c001 + (fg-fr)*c011 + fr*c111
```

### 3.4 LUT生成流程

```
┌─────────────────────────────────────────────────────────────────┐
│              从ICC Profile生成3D LUT                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│   输入: 源Profile, 目标Profile, 渲染意图                        │
│                                                                  │
│   ┌─────────────────────────────────────────────────────────┐   │
│   │  for r in range(lut_size):                              │   │
│   │      for g in range(lut_size):                          │   │
│   │          for b in range(lut_size):                      │   │
│   │              # 1. 归一化输入                             │   │
│   │              rgb_in = [r, g, b] / (lut_size - 1)        │   │
│   │                                                          │   │
│   │              # 2. 源Profile: RGB → PCS                  │   │
│   │              pcs = source_profile.to_pcs(rgb_in)         │   │
│   │                                                          │   │
│   │              # 3. 应用渲染意图/色域映射                  │   │
```

### 5.3 CUSP算法

```python
def cusp_gamut_mapping(lab_color, source_gamut, dest_gamut):
    """
    CUSP (Chroma Uniform Sampling Point) 色域映射
    
    沿着指向L*轴的方向映射，保持色相不变
    """
    L, a, b = lab_color
    
    # 计算色相和饱和度
    h = np.arctan2(b, a)
    C = np.sqrt(a**2 + b**2)
    
    # 获取该色相下两个色域的边界
    C_src_max = source_gamut.get_max_chroma(L, h)
    C_dst_max = dest_gamut.get_max_chroma(L, h)
    
    if C <= C_dst_max:
        # 在目标色域内，不变
        return lab_color
    elif C <= C_src_max:
        # 在源色域内但超出目标色域
        # 线性压缩到目标边界
        C_new = C_dst_max * (C / C_src_max)
    else:
        # 超出源色域（异常情况），裁剪
        C_new = C_dst_max
    
    # 重建Lab
    a_new = C_new * np.cos(h)
    b_new = C_new * np.sin(h)
    
    return [L, a_new, b_new]
```

### 5.4 HPMinDE算法

HPMinDE（Hue-Preserving Minimum Delta E）最小化感知色差：

$$
\min_{\mathbf{c}_{dst}} \Delta E(\mathbf{c}_{src}, \mathbf{c}_{dst}) \quad \text{s.t.} \quad \mathbf{c}_{dst} \in \text{DestGamut}, h_{dst} = h_{src}
$$

```swift
func convertColor(
    from sourceProfile: ColorSyncProfile,
    to destProfile: ColorSyncProfile,
    color: [CGFloat]

        MemoryLayout<CGFloat>.stride * 3,
        nil
    )

}
```

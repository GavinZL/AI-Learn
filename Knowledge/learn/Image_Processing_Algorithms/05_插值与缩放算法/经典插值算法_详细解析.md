## 1. 插值问题的数学本质

### 1.1 连续信号的离散重采样

图像插值的核心问题是：**如何从离散采样点重建连续信号，然后在新位置重新采样**。

```
原始图像（离散）→ 连续函数重建 → 新位置采样 → 目标图像（离散）

┌─────────────────────────────────────────────────────────────┐
│              图像缩放的本质过程                               │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│   原始网格 (M×N)           目标网格 (P×Q)                    │
│   ┌───┬───┬───┬───┐        ┌───┬───┬───┬───┬───┬───┐        │
│   │ ● │ ● │ ● │ ● │        │ ○ │ ○ │ ○ │ ○ │ ○ │ ○ │        │
│   ├───┼───┼───┼───┤   →    ├───┼───┼───┼───┼───┼───┤        │
│   │ ● │ ● │ ● │ ● │        │ ○ │ ○ │ ○ │ ○ │ ○ │ ○ │        │
│   ├───┼───┼───┼───┤        ├───┼───┼───┼───┼───┼───┤        │
│   │ ● │ ● │ ● │ ● │        │ ○ │ ○ │ ○ │ ○ │ ○ │ ○ │        │
│   └───┴───┴───┴───┘        └───┴───┴───┴───┴───┴───┘        │
│                                                              │
│   ● = 原始像素点           ○ = 需要插值计算的新像素点         │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

#### 坐标映射关系

目标图像坐标 $(x', y')$ 对应原始图像坐标 $(x, y)$ 的映射：

$$
x = x' \times \frac{W_{src}}{W_{dst}}, \quad y = y' \times \frac{H_{src}}{H_{dst}}
$$

其中 $(x, y)$ 通常是非整数坐标，需要通过插值计算对应的像素值。

### 1.2 插值核函数理论

所有插值算法都可以统一表示为**卷积形式**：

$$
f(x, y) = \sum_{i} \sum_{j} f[i, j] \cdot h(x - i, y - j)
$$

其中：
- $f[i, j]$ 是原始离散像素值
- $h(x, y)$ 是**插值核函数**（Interpolation Kernel）
- $f(x, y)$ 是重建的连续函数在 $(x, y)$ 处的值

#### 插值核的基本性质

| 性质 | 数学表达 | 物理意义 |
|------|---------|---------|
| 归一化 | $\int_{-\infty}^{\infty} h(x)dx = 1$ | 保持亮度不变 |
| 插值条件 | $h(0) = 1, h(n) = 0$ for $n \neq 0$ | 在采样点保持原值 |
| 对称性 | $h(x) = h(-x)$ | 各向同性 |
| 紧支撑 | $h(x) = 0$ for $|x| > r$ | 有限计算范围 |

### 1.3 理想插值与sinc函数

根据**奈奎斯特-香农采样定理**，带限信号可以完美重建。理想的插值核是**sinc函数**：

$$
h_{ideal}(x) = \text{sinc}(x) = \frac{\sin(\pi x)}{\pi x}
$$

```
              sinc函数图形
        1.0 ┤        ●
            │       ╱│╲
        0.5 ┤      ╱ │ ╲
            │     ╱  │  ╲
        0.0 ┼────●───●───●────●───●───●────
            │  ╱      ╲       ╱ ╲
       -0.5 ┤ ●        ●     ●   ●
            ├──┬──┬──┬──┬──┬──┬──┬──┬──┬──┤
           -4 -3 -2 -1  0  1  2  3  4
```

#### sinc函数的问题

| 问题 | 描述 | 后果 |
|------|------|------|
| **无限支撑** | $h(x) \neq 0$ 对于所有 $x$ | 需要无限多采样点 |
| **慢衰减** | 按 $1/x$ 衰减 | 截断造成严重振铃 |
| **计算量大** | 需要大量乘加运算 | 实时应用困难 |

因此实际应用中需要对sinc函数进行**近似**或**窗口截断**。

## 2. 最近邻插值

### 2.1 零阶保持原理

最近邻插值是最简单的插值方法，本质是**零阶保持**（Zero-Order Hold）：

$$
f(x, y) = f[\text{round}(x), \text{round}(y)]
$$

即直接取最近的整数坐标像素值。

```
┌─────────────────────────────────────────────────────────────┐
│                  最近邻插值原理                              │
├─────────────────────────────────────────────────────────────┤

│   ┌─────┬─────┐              ┌─────┬─────┐                  │
│   │     │     │              │     │     │                  │
│   │  A  │  B  │              │  A  │  B  │                  │
│   │     │  ×  │  ──────→     │     │  ●  │  选择最近的B      │
│   ├─────┼─────┤              ├─────┼─────┤                  │
│   │     │     │              │     │     │                  │
│   │  C  │  D  │              │  C  │  D  │                  │
│   │     │     │              │     │     │                  │
│   └─────┴─────┘              └─────┴─────┘                  │
```

### 2.2 坐标映射公式

```python
def nearest_neighbor(src, dst_width, dst_height):
    """
    最近邻插值实现
    
    参数:
        src: 源图像 (H, W, C)
        dst_width: 目标宽度
        dst_height: 目标高度
    返回:
        dst: 目标图像 (dst_height, dst_width, C)
    """
    src_height, src_width = src.shape[:2]
    dst = np.zeros((dst_height, dst_width, src.shape[2]), dtype=src.dtype)
    
    # 缩放因子
    scale_x = src_width / dst_width
    scale_y = src_height / dst_height
    
    for y in range(dst_height):
        for x in range(dst_width):
            # 映射到源图像坐标
            src_x = int(round(x * scale_x))
            src_y = int(round(y * scale_y))
            
            # 边界处理
            src_x = min(src_x, src_width - 1)
            src_y = min(src_y, src_height - 1)
            
            dst[y, x] = src[src_y, src_x]
    
    return dst
```

## 3. 双线性插值

### 3.1 数学推导

双线性插值通过**两次线性插值**的组合实现：

```
┌─────────────────────────────────────────────────────────────┐
│                    双线性插值原理                            │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│          f(0,0)=A              f(1,0)=B                      │
│              ●──────────────────────●                        │
│              │                      │                        │
│              │    ●f₁              │                        │
│              │    │                 │                        │
│              │    │     ●P(x,y)    │                        │
│              │    │                 │                        │
│              │    ●f₂              │                        │
│              │                      │                        │
│              ●──────────────────────●                        │
│          f(0,1)=C              f(1,1)=D                      │
│                                                              │
│   步骤1: 上边水平插值 f₁ = A(1-x) + Bx                       │
│   步骤2: 下边水平插值 f₂ = C(1-x) + Dx                       │
│   步骤3: 垂直插值    P = f₁(1-y) + f₂y                       │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 权重计算公式

设待插值点坐标为 $(x, y)$，其中 $x = i + u$，$y = j + v$（$i, j$ 为整数部分，$u, v \in [0,1)$ 为小数部分）：

$$
f(x, y) = (1-u)(1-v) \cdot f[i,j] + u(1-v) \cdot f[i+1,j] + (1-u)v \cdot f[i,j+1] + uv \cdot f[i+1,j+1]
$$

权重矩阵表示：

$$
\begin{bmatrix} w_{00} & w_{10} \\ w_{01} & w_{11} \end{bmatrix} = 
\begin{bmatrix} (1-u)(1-v) & u(1-v) \\ (1-u)v & uv \end{bmatrix}
$$

### 3.3 代码实现

```python
import numpy as np

def bilinear_interpolation(src, dst_width, dst_height):
    """
    双线性插值实现
    
    参数:
        src: 源图像 (H, W, C) 或 (H, W)
        dst_width: 目标宽度
        dst_height: 目标高度
    返回:
        dst: 目标图像
    """
    src_height, src_width = src.shape[:2]
    
    # 处理单通道和多通道
    if len(src.shape) == 2:
        src = src[:, :, np.newaxis]
    channels = src.shape[2]
    
    dst = np.zeros((dst_height, dst_width, channels), dtype=np.float32)
    
    # 缩放因子
    scale_x = src_width / dst_width
    scale_y = src_height / dst_height
    
    for y in range(dst_height):
        for x in range(dst_width):
            # 映射到源图像坐标（中心对齐）
            src_x = (x + 0.5) * scale_x - 0.5
            src_y = (y + 0.5) * scale_y - 0.5
            
            # 整数和小数部分
            x0 = int(np.floor(src_x))
            y0 = int(np.floor(src_y))
            x1 = x0 + 1
            y1 = y0 + 1
            
            # 计算权重
            u = src_x - x0
            v = src_y - y0
            
            # 边界裁剪
            x0 = max(0, min(x0, src_width - 1))
            x1 = max(0, min(x1, src_width - 1))
            y0 = max(0, min(y0, src_height - 1))
            y1 = max(0, min(y1, src_height - 1))
            
            # 双线性插值
            dst[y, x] = (
                (1 - u) * (1 - v) * src[y0, x0] +
                u * (1 - v) * src[y0, x1] +
                (1 - u) * v * src[y1, x0] +
                u * v * src[y1, x1]
            )
    
    return dst.squeeze().astype(src.dtype)
```

C语言优化版本：

```c
void bilinear_resize(
    const uint8_t* src, int src_w, int src_h,
    uint8_t* dst, int dst_w, int dst_h,
    int channels
) {
    float scale_x = (float)src_w / dst_w;
    float scale_y = (float)src_h / dst_h;
    
    for (int y = 0; y < dst_h; y++) {
        for (int x = 0; x < dst_w; x++) {
            // 源坐标（中心对齐）
            float src_x = (x + 0.5f) * scale_x - 0.5f;
            float src_y = (y + 0.5f) * scale_y - 0.5f;
            
            int x0 = (int)floorf(src_x);
            int y0 = (int)floorf(src_y);
            
            float u = src_x - x0;
            float v = src_y - y0;
            
            // 边界处理
            int x1 = x0 + 1 < src_w ? x0 + 1 : x0;
            int y1 = y0 + 1 < src_h ? y0 + 1 : y0;
            x0 = x0 >= 0 ? x0 : 0;
            y0 = y0 >= 0 ? y0 : 0;
            
            // 预计算权重
            float w00 = (1 - u) * (1 - v);
            float w10 = u * (1 - v);
            float w01 = (1 - u) * v;
            float w11 = u * v;
            
            for (int c = 0; c < channels; c++) {
                float val = 
                    w00 * src[(y0 * src_w + x0) * channels + c] +
                    w10 * src[(y0 * src_w + x1) * channels + c] +
                    w01 * src[(y1 * src_w + x0) * channels + c] +
                    w11 * src[(y1 * src_w + x1) * channels + c];
                
                dst[(y * dst_w + x) * channels + c] = (uint8_t)(val + 0.5f);
            }
        }
    }
}
```

### 3.4 频域分析

双线性插值核函数为**三角形函数**（Tent Function）：

$$
h_{bilinear}(x) = \begin{cases}
1 - |x| & |x| < 1 \\
0 & |x| \geq 1
\end{cases}
$$

其傅里叶变换为：

$$
H_{bilinear}(\omega) = \text{sinc}^2\left(\frac{\omega}{2}\right)
$$

## 4. 双三次插值

### 4.1 三次样条基函数

双三次插值使用**4×4邻域**进行加权，核函数通常采用以下形式：

$$
h(x) = \begin{cases}
(a+2)|x|^3 - (a+3)|x|^2 + 1 & |x| \leq 1 \\
a|x|^3 - 5a|x|^2 + 8a|x| - 4a & 1 < |x| < 2 \\
0 & |x| \geq 2
\end{cases}
$$

其中参数 $a$ 控制核函数的形状。

### 4.2 α参数的影响

| 参数 | 名称 | 特点 |
|------|------|------|
| $a = 0$ | B-spline | 最平滑，不满足插值条件 |
| $a = -0.5$ | Catmull-Rom | 平衡锐度和平滑度，**推荐** |
| $a = -0.75$ | 常用值 | 略增加锐度 |
| $a = -1$ | 类似sinc | 最锐利，但可能过冲 |

```

            │        a=-1.0 ╱╲
```

### 4.3 4×4邻域插值实现

```python
def cubic_kernel(x, a=-0.5):
    """
    双三次核函数 (Keys' kernel)
    
    参数:
        x: 距离（可以是数组）
        a: 形状参数，默认-0.5 (Catmull-Rom)
    """
    x = np.abs(x)
    result = np.zeros_like(x, dtype=np.float32)
    
    # |x| <= 1
    mask1 = x <= 1
    result[mask1] = (a + 2) * x[mask1]**3 - (a + 3) * x[mask1]**2 + 1
    
    # 1 < |x| < 2
    mask2 = (x > 1) & (x < 2)
    result[mask2] = a * x[mask2]**3 - 5*a * x[mask2]**2 + 8*a * x[mask2] - 4*a
    
    return result

def bicubic_interpolation(src, dst_width, dst_height, a=-0.5):
    """
    双三次插值实现
    """
    src_height, src_width = src.shape[:2]
    
    if len(src.shape) == 2:
        src = src[:, :, np.newaxis]
    channels = src.shape[2]
    
    dst = np.zeros((dst_height, dst_width, channels), dtype=np.float32)
    
    scale_x = src_width / dst_width
    scale_y = src_height / dst_height
    
    for y in range(dst_height):
        for x in range(dst_width):
            # 源坐标
            src_x = (x + 0.5) * scale_x - 0.5
            src_y = (y + 0.5) * scale_y - 0.5
            
            x0 = int(np.floor(src_x))
            y0 = int(np.floor(src_y))
            
            # 计算4x4邻域的权重
            weights_x = np.array([cubic_kernel(src_x - (x0 + i), a) for i in range(-1, 3)])
            weights_y = np.array([cubic_kernel(src_y - (y0 + i), a) for i in range(-1, 3)])
            
            # 归一化权重
            weights_x /= weights_x.sum()
            weights_y /= weights_y.sum()
            
            for c in range(channels):
                val = 0
                for j in range(4):
                    for i in range(4):
                        px = max(0, min(x0 - 1 + i, src_width - 1))
                        py = max(0, min(y0 - 1 + j, src_height - 1))
                        val += weights_x[i] * weights_y[j] * src[py, px, c]
                
                dst[y, x, c] = np.clip(val, 0, 255)
    
    return dst.squeeze().astype(np.uint8)
```

## 5. Lanczos插值

### 5.1 sinc函数的窗口截断

Lanczos插值通过对sinc函数施加**Lanczos窗口**实现截断：

$$
h_{Lanczos}(x) = \begin{cases}
\text{sinc}(x) \cdot \text{sinc}(x/a) & |x| < a \\
0 & |x| \geq a
\end{cases}
$$

其中 $\text{sinc}(x) = \frac{\sin(\pi x)}{\pi x}$，$a$ 是窗口半径。

### 5.2 Lanczos-2/Lanczos-3核函数

libswscale中的Lanczos实现核心：

```c
// FFmpeg swscale Lanczos核函数
static double lanczos_kernel(double x, double param) {
    if (x == 0.0)
        return 1.0;
    if (fabs(x) >= param)
        return 0.0;
    
    double pi_x = M_PI * x;
    return sin(pi_x) / pi_x * sin(pi_x / param) / (pi_x / param);
}
```

### 6.1 预滤波的必要性

```
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 6.2 区域平均法（Area/Box）

最简单的抗混叠下采样：对每个输出像素，计算其覆盖的所有输入像素的平均值。

```python
def area_downscale(src, scale_factor):
    """
    区域平均下采样（整数倍）
    
    参数:
        src: 源图像
        scale_factor: 下采样倍数（整数）
    """
    src_h, src_w = src.shape[:2]
    dst_h = src_h // scale_factor
    dst_w = src_w // scale_factor
    
    if len(src.shape) == 3:
        # 重塑为块状，然后求平均
        dst = src[:dst_h*scale_factor, :dst_w*scale_factor]
        dst = dst.reshape(dst_h, scale_factor, dst_w, scale_factor, -1)
        dst = dst.mean(axis=(1, 3))
    else:
        dst = src[:dst_h*scale_factor, :dst_w*scale_factor]
        dst = dst.reshape(dst_h, scale_factor, dst_w, scale_factor)
        dst = dst.mean(axis=(1, 3))
    
    return dst.astype(src.dtype)
```

### 6.3 高斯预滤波+降采样

```python
import cv2

def gaussian_downscale(src, scale_factor, sigma=None):
    """
    高斯预滤波下采样
    
    参数:
        src: 源图像
        scale_factor: 下采样倍数
        sigma: 高斯核标准差，默认为 0.5 * scale_factor
    """
    if sigma is None:
        sigma = 0.5 * scale_factor
    
    # 高斯核大小（至少覆盖3σ）
    ksize = int(np.ceil(sigma * 6)) | 1  # 确保奇数
    
    # 高斯滤波
    blurred = cv2.GaussianBlur(src, (ksize, ksize), sigma)
    
    # 下采样（可以使用最近邻，因为已经滤波）
    dst_h = src.shape[0] // scale_factor
    dst_w = src.shape[1] // scale_factor
    
    return cv2.resize(blurred, (dst_w, dst_h), interpolation=cv2.INTER_NEAREST)
```

```python
def generate_mipmap(src, levels=None):
    """
    生成Mipmap金字塔
    
    参数:
        src: 源图像
        levels: 层数，默认到1×1
    返回:
        list of images: Mipmap金字塔
    """
    mipmap = [src]
    
    if levels is None:
        levels = int(np.log2(min(src.shape[:2]))) + 1
    
    current = src.astype(np.float32)
    
    for i in range(1, levels):
        # Box filter (2x2平均) + 下采样
        h, w = current.shape[:2]
        if h < 2 or w < 2:
            break
        
        # 2x2区域平均
        next_level = (current[0::2, 0::2] + current[1::2, 0::2] +
                      current[0::2, 1::2] + current[1::2, 1::2]) / 4.0
        
        mipmap.append(next_level.astype(src.dtype))
        current = next_level
    
    return mipmap
```

## 8. 定点化与硬件实现

### 8.1 整数权重近似

浮点运算在嵌入式/FPGA中代价高，需要转换为定点数：

```c
// 双线性插值的定点实现 (Q8格式，8位小数)
#define Q_BITS 8
#define Q_ONE (1 << Q_BITS)  // 256

void bilinear_fixed_point(
    const uint8_t* src, int src_w, int src_h,
    uint8_t* dst, int dst_w, int dst_h
) {
    // 缩放因子 (Q16格式)
    uint32_t scale_x = (src_w << 16) / dst_w;
    uint32_t scale_y = (src_h << 16) / dst_h;
    
    for (int y = 0; y < dst_h; y++) {
        uint32_t src_y_q16 = y * scale_y;
        int y0 = src_y_q16 >> 16;
        int v = (src_y_q16 >> 8) & 0xFF;  // 小数部分 Q8
        
        for (int x = 0; x < dst_w; x++) {
            uint32_t src_x_q16 = x * scale_x;
            int x0 = src_x_q16 >> 16;
            int u = (src_x_q16 >> 8) & 0xFF;  // 小数部分 Q8
            
            int x1 = (x0 + 1 < src_w) ? x0 + 1 : x0;
            int y1 = (y0 + 1 < src_h) ? y0 + 1 : y0;
            
            // 定点权重
            int w00 = (Q_ONE - u) * (Q_ONE - v);  // Q16
            int w10 = u * (Q_ONE - v);
            int w01 = (Q_ONE - u) * v;
            int w11 = u * v;
            
            // 加权求和 (结果为Q16)
            int val = (w00 * src[y0 * src_w + x0] +
                       w10 * src[y0 * src_w + x1] +
                       w01 * src[y1 * src_w + x0] +
                       w11 * src[y1 * src_w + x1]);
            
            // 四舍五入并转回整数
            dst[y * dst_w + x] = (val + (1 << 15)) >> 16;
        }
    }
}
```

### 8.2 查找表优化

对于固定缩放比例，可以预计算权重查找表：

```c
// 预计算双三次核函数查找表
#define LUT_SIZE 256  // 对应小数部分的256级
#define KERNEL_SIZE 4

int16_t bicubic_lut[LUT_SIZE][KERNEL_SIZE];

void init_bicubic_lut(float a) {
    for (int i = 0; i < LUT_SIZE; i++) {
        float t = i / 256.0f;  // 小数部分 [0, 1)
        
        // 4个采样点的权重 (位置 -1, 0, 1, 2)
        float w[4];
        w[0] = cubic_weight(1.0f + t, a);
        w[1] = cubic_weight(t, a);
        w[2] = cubic_weight(1.0f - t, a);
        w[3] = cubic_weight(2.0f - t, a);
        
        // 归一化并转为Q14定点
        float sum = w[0] + w[1] + w[2] + w[3];
        for (int j = 0; j < 4; j++) {
            bicubic_lut[i][j] = (int16_t)((w[j] / sum) * 16384 + 0.5f);
        }
    }
}
```

| `INTER_CUBIC` | 双三次 | 高质量上采样 |
| `INTER_LANCZOS4` | Lanczos-4 | 最高质量 |

### 9.4 FFmpeg sws_scale

```c
#include <libswscale/swscale.h>

// 创建缩放上下文
struct SwsContext* sws_ctx = sws_getContext(
    src_width, src_height, AV_PIX_FMT_RGB24,   // 源格式
    dst_width, dst_height, AV_PIX_FMT_RGB24,   // 目标格式
    SWS_LANCZOS,                                // 缩放算法
    NULL, NULL, NULL
);

// 可用的缩放算法标志
// SWS_FAST_BILINEAR  - 快速双线性
// SWS_BILINEAR       - 标准双线性
// SWS_BICUBIC        - 双三次
// SWS_POINT          - 最近邻
// SWS_AREA           - 区域平均
// SWS_LANCZOS        - Lanczos

// 执行缩放
sws_scale(sws_ctx,
    (const uint8_t* const*)src_data, src_linesize,
    0, src_height,
    dst_data, dst_linesize
);

// 释放
sws_freeContext(sws_ctx);
```

命令行使用：

```bash
# 使用不同算法缩放视频
ffmpeg -i input.mp4 -vf "scale=1280:720:flags=bilinear" output_bilinear.mp4
ffmpeg -i input.mp4 -vf "scale=1280:720:flags=bicubic" output_bicubic.mp4
ffmpeg -i input.mp4 -vf "scale=1280:720:flags=lanczos" output_lanczos.mp4

# 高质量下采样（先模糊再缩放）
ffmpeg -i input.mp4 -vf "scale=640:360:flags=lanczos+accurate_rnd" output.mp4
```

## 1. 超分辨率问题定义

### 1.1 从低分辨率到高分辨率的逆问题

超分辨率（Super-Resolution, SR）的目标是从低分辨率（LR）图像重建高分辨率（HR）图像：

```
┌─────────────────────────────────────────────────────────────┐
│                  超分辨率问题建模                            │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│   正向退化过程（已知）:                                       │
│   ┌────────┐    模糊     ┌────────┐  下采样   ┌────────┐    │
│   │   HR   │  ────────→  │ 模糊HR │ ────────→ │   LR   │    │
│   │  图像  │     k       │        │    ↓s     │  图像  │    │
│   └────────┘             └────────┘           └────────┘    │
│                                                    ↓ +n      │
│   数学表示:                                    ┌────────┐    │
│   y = (x ⊗ k)↓s + n                          │ 噪声LR │    │
│                                               └────────┘    │
│   逆向重建过程（需求解）:                                     │
│   ┌────────┐            ？？？？            ┌────────┐      │
│   │   HR   │  ←──────────────────────────  │   LR   │      │
│   │  图像  │         超分辨率算法          │  图像  │      │
│   └────────┘                                └────────┘      │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

其中：
- $x$ 是高分辨率图像
- $k$ 是模糊核（如高斯模糊）
- $\downarrow_s$ 是下采样操作（因子为 $s$）
- $n$ 是噪声
- $y$ 是观测到的低分辨率图像

### 1.2 与传统插值的本质区别

| 特性 | 传统插值 | 超分辨率 |
|------|----------|----------|
| **信息来源** | 仅利用输入像素 | 利用先验知识/训练数据 |
| **高频恢复** | 无法恢复丢失的高频 | 尝试"幻想"合理的高频 |
| **数学性质** | 确定性变换 | 概率/统计推断 |
| **计算复杂度** | O(N) | O(N²) 到 O(N³) |
| **放大极限** | 2-4x 可接受 | 4-8x 仍可用 |

```
              频谱对比
        
   高分辨率原图        低分辨率图          插值放大          超分辨率
   ┌─────────┐       ┌─────────┐       ┌─────────┐       ┌─────────┐
   │ ████████│       │ ████    │       │ ████▒▒▒▒│       │ ████████│
   │ ████████│  →    │ ████    │  →    │ ████▒▒▒▒│  or   │ ████▓▓▓▓│
   │ ████████│       │         │       │ ▒▒▒▒▒▒▒▒│       │ ▓▓▓▓▓▓▓▓│
   │ ████████│       │         │       │ ▒▒▒▒▒▒▒▒│       │ ▓▓▓▓▓▓▓▓│
   └─────────┘       └─────────┘       └─────────┘       └─────────┘
   完整频谱          丢失高频           高频仍缺失         恢复高频
                                       （模糊）           （清晰）
```

$$

$$

## 2. 传统超分方法

### 2.1 基于重建的方法：IBP迭代反投影

**迭代反投影**（Iterative Back-Projection, IBP）通过迭代优化重建误差：

$$
x^{(k+1)} = x^{(k)} + \lambda \cdot \text{BackProject}(y - \text{Degrade}(x^{(k)}))
$$

```
┌─────────────────────────────────────────────────────────────┐
│                     IBP算法流程                              │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│   初始估计 x⁰ (双三次插值)                                   │
│        │                                                     │
│        ▼                                                     │
│   ┌─────────────────────────────────────────┐               │
│   │  迭代循环:                               │               │
│   │                                          │               │
│   │  1. 模拟退化: ŷ = Degrade(xᵏ)           │               │
│   │                                          │               │
│   │  2. 计算残差: r = y - ŷ                 │               │
│   │                                          │               │
│   │  3. 反投影: Δx = Upsample(r)            │               │
│   │                                          │               │
│   │  4. 更新: xᵏ⁺¹ = xᵏ + λ·Δx             │               │
│   │                                          │               │
│   │  5. 收敛检查: ‖Δx‖ < ε ?                │               │
│   │                                          │               │
│   └─────────────────────────────────────────┘               │
│        │                                                     │
│        ▼                                                     │
│   输出重建结果 x*                                            │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

```python
def ibp_super_resolution(lr_image, scale_factor, iterations=20, lambda_=0.5):
    """
    迭代反投影超分辨率
    
    参数:
        lr_image: 低分辨率输入
        scale_factor: 放大倍数
        iterations: 迭代次数
        lambda_: 更新步长
    """
    import cv2
    import numpy as np
    
    # 初始估计：双三次插值
    hr_estimate = cv2.resize(lr_image, None, fx=scale_factor, fy=scale_factor,
                              interpolation=cv2.INTER_CUBIC)
    
    for i in range(iterations):
        # 1. 模拟退化过程
        # 高斯模糊
        blurred = cv2.GaussianBlur(hr_estimate, (5, 5), 1.0)
        # 下采样
        simulated_lr = cv2.resize(blurred, (lr_image.shape[1], lr_image.shape[0]),
                                   interpolation=cv2.INTER_AREA)
        
        # 2. 计算残差
        residual = lr_image.astype(np.float32) - simulated_lr.astype(np.float32)
        
        # 3. 反投影（上采样残差）
        back_projected = cv2.resize(residual, (hr_estimate.shape[1], hr_estimate.shape[0]),
                                     interpolation=cv2.INTER_CUBIC)
        
        # 4. 更新估计
        hr_estimate = np.clip(hr_estimate + lambda_ * back_projected, 0, 255).astype(np.uint8)
    
    return hr_estimate
```

### 2.3 基于边缘的方法

**Edge-Directed Interpolation** 沿边缘方向而非固定网格方向插值：

```
┌─────────────────────────────────────────────────────────────┐
│                边缘导向插值原理                              │

│                                                              │

│   ● ─ ─ ─ ● ─ ─ ─ ●          ● ─ ─ ─ ● ─ ─ ─ ●             │
│   │       │       │          │ ╲       │     ╱ │             │
│   │   ×   │       │          │   ╲ ×   │   ╱   │             │
│   │       │       │          │     ╲   │ ╱     │   沿边缘    │
│   ● ─ ─ ─ ● ─ ─ ─ ●          ● ─ ─ ─ ● ─ ─ ─ ●   方向加权   │
│   │       │       │          │     ╱   │ ╲     │             │
│   │       │       │          │   ╱     │   ╲   │             │
│   │       │       │          │ ╱       │     ╲ │             │
│   ● ─ ─ ─ ● ─ ─ ─ ●          ● ─ ─ ─ ● ─ ─ ─ ●             │
│                                                              │

│   ● = 已知像素                                               │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

典型算法：**NEDI**（New Edge-Directed Interpolation）

def edge_directed_interpolation(image, x, y):
    """
    边缘导向插值示例（简化版）
    
    通过局部协方差估计边缘方向，沿边缘方向加权插值
    """
    # 1. 收集局部窗口像素
    window_size = 5
    half = window_size // 2
    
    # 2. 计算梯度协方差矩阵
    gx = cv2.Sobel(image, cv2.CV_64F, 1, 0, ksize=3)
    gy = cv2.Sobel(image, cv2.CV_64F, 0, 1, ksize=3)
    
    # 局部协方差
    local_gx = gx[y-half:y+half+1, x-half:x+half+1]
    local_gy = gy[y-half:y+half+1, x-half:x+half+1]
    
    C = np.array([
        [np.sum(local_gx**2), np.sum(local_gx*local_gy)],
        [np.sum(local_gx*local_gy), np.sum(local_gy**2)]
    ])
    
    # 3. 特征分解得到主方向
    eigenvalues, eigenvectors = np.linalg.eig(C)
    edge_direction = eigenvectors[:, np.argmin(eigenvalues)]

## 3. 深度学习超分

### 3.1 SRCNN：开创性CNN超分

**SRCNN**（Super-Resolution Convolutional Neural Network, 2014）是第一个端到端的深度学习超分网络：

```
┌─────────────────────────────────────────────────────────────┐
│                    SRCNN网络结构                             │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│   输入: 双三次上采样的LR图像 (已放大到目标尺寸)              │
│                                                              │
│   ┌─────────────────────────────────────────────────────┐   │
│   │                                                      │   │
│   │  ┌───────────┐   ┌───────────┐   ┌───────────┐     │   │
│   │  │ Conv1     │   │ Conv2     │   │ Conv3     │     │   │
│   │  │ 9×9, 64   │ → │ 1×1, 32   │ → │ 5×5, 1    │     │   │
│   │  │ + ReLU    │   │ + ReLU    │   │           │     │   │
│   │  └───────────┘   └───────────┘   └───────────┘     │   │
│   │                                                      │   │
│   │  块提取/表示     非线性映射       重建               │   │
│   │                                                      │   │
│   └─────────────────────────────────────────────────────┘   │
│                                                              │
│   输出: 重建的HR图像                                         │
│                                                              │
│   损失函数: MSE Loss                                         │
│   L = (1/n) Σ ‖F(Y;θ) - X‖²                                │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

```python
import torch
import torch.nn as nn

class SRCNN(nn.Module):
    """
    SRCNN网络实现
    """
    def __init__(self, num_channels=1):
        super(SRCNN, self).__init__()
        
        # 特征提取层
        self.conv1 = nn.Conv2d(num_channels, 64, kernel_size=9, padding=4)
        
        # 非线性映射层
        self.conv2 = nn.Conv2d(64, 32, kernel_size=1, padding=0)
        
        # 重建层
        self.conv3 = nn.Conv2d(32, num_channels, kernel_size=5, padding=2)
        
        self.relu = nn.ReLU(inplace=True)
    
    def forward(self, x):
        x = self.relu(self.conv1(x))
        x = self.relu(self.conv2(x))
        x = self.conv3(x)
        return x

# 训练策略
def train_srcnn(model, train_loader, epochs=100, lr=1e-4):
    criterion = nn.MSELoss()
    optimizer = torch.optim.Adam(model.parameters(), lr=lr)
    
    for epoch in range(epochs):
        for lr_imgs, hr_imgs in train_loader:
            # 前向传播
            outputs = model(lr_imgs)
            loss = criterion(outputs, hr_imgs)
            
            # 反向传播
            optimizer.zero_grad()
            loss.backward()
            optimizer.step()
```

│                                                              │

│   ┌─────────────────────────────────────────────────┐       │
│   │                                                  │       │

│   │    │                                         │  │       │

│   └─────────────────────────────────────────────────┘       │
│                                                              │
│   EDSR整体结构:                                              │
│   ┌──────┐   ┌─────┐         ┌─────┐   ┌─────────┐         │
│   │ Conv │ → │ RB  │ → ... → │ RB  │ → │UpSample │ → HR   │
│   │ 3×3  │   │ ×32 │         │     │   │ (Pixel  │         │

**RCAN**（Residual Channel Attention Networks）引入通道注意力机制：

```
┌─────────────────────────────────────────────────────────────┐
│                 通道注意力模块 (CA)                          │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│   输入特征 F (C×H×W)                                         │
│        │                                                     │
│        ▼                                                     │
│   ┌─────────────────┐                                        │
│   │ Global Average  │   → (C×1×1)                           │
│   │    Pooling      │                                        │
│   └────────┬────────┘                                        │
│            │                                                 │
│            ▼                                                 │
│   ┌─────────────────┐                                        │
│   │   FC → ReLU     │   C → C/r → C                         │
│   │   → FC → Sigmoid│                                        │
│   └────────┬────────┘                                        │
│            │                                                 │
│            ▼                                                 │
│        Scale (C×1×1)   通道权重                              │
│            │                                                 │
│            ▼                                                 │
│     F × Scale = F'     通道加权                              │
│                                                              │
│   公式: CA(F) = F · σ(W₂ · ReLU(W₁ · GAP(F)))              │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

```python
class ChannelAttention(nn.Module):
    """通道注意力模块"""
    def __init__(self, num_features, reduction=16):
        super().__init__()
        self.avg_pool = nn.AdaptiveAvgPool2d(1)
        self.fc = nn.Sequential(
            nn.Linear(num_features, num_features // reduction, bias=False),
            nn.ReLU(inplace=True),
            nn.Linear(num_features // reduction, num_features, bias=False),
            nn.Sigmoid()
        )
    
    def forward(self, x):
        b, c, _, _ = x.size()
        y = self.avg_pool(x).view(b, c)
        y = self.fc(y).view(b, c, 1, 1)
        return x * y.expand_as(x)

class RCAB(nn.Module):
    """残差通道注意力块"""
    def __init__(self, num_features, reduction=16):
        super().__init__()
        self.conv1 = nn.Conv2d(num_features, num_features, 3, padding=1)
        self.conv2 = nn.Conv2d(num_features, num_features, 3, padding=1)
        self.ca = ChannelAttention(num_features, reduction)
        self.relu = nn.ReLU(inplace=True)
    
    def forward(self, x):
        residual = x
        x = self.relu(self.conv1(x))
        x = self.conv2(x)
        x = self.ca(x)
        return x + residual
```

GAN训练损失：

$$
\mathcal{L}_{total} = \mathcal{L}_{pixel} + \lambda_1 \mathcal{L}_{perceptual} + \lambda_2 \mathcal{L}_{GAN}
$$

| 损失项 | 公式 | 作用 |
|--------|------|------|
| 像素损失 | $L_1(G(x), y)$ | 保真度 |
| 感知损失 | $\|VGG(G(x)) - VGG(y)\|_2$ | 视觉质量 |
| GAN损失 | $-\log D(G(x))$ | 真实感 |

import tensorflow as tf

# 加载TFLite模型
interpreter = tf.lite.Interpreter(model_path="espcn_x2.tflite")
interpreter.allocate_tensors()

input_details = interpreter.get_input_details()
output_details = interpreter.get_output_details()

def run_sr_inference(lr_image):
    # 预处理
    input_data = lr_image.astype(np.float32) / 255.0
    input_data = np.expand_dims(input_data, axis=0)
    
    # 推理
    interpreter.set_tensor(input_details[0]['index'], input_data)
    interpreter.invoke()
    
    # 后处理
    output_data = interpreter.get_tensor(output_details[0]['index'])
    hr_image = (output_data[0] * 255).clip(0, 255).astype(np.uint8)
    
    return hr_image

## 5. 质量评估

### 5.1 客观指标

| 指标 | 公式/说明 | 范围 | 特点 |
|------|-----------|------|------|
| **PSNR** | $10\log_{10}(\frac{MAX^2}{MSE})$ | ~30-40dB | 像素级保真度 |
| **SSIM** | 结构相似性 | 0-1 | 感知结构相似 |
| **LPIPS** | 深度特征距离 | 0-1 | 感知质量 |
| **NIQE** | 自然图像质量 | 越小越好 | 无参考指标 |

```python
import torch
import lpips

# PSNR计算
def calculate_psnr(img1, img2, max_val=255):
    mse = np.mean((img1.astype(float) - img2.astype(float)) ** 2)
    if mse == 0:
        return float('inf')
    return 10 * np.log10(max_val ** 2 / mse)

# SSIM计算
from skimage.metrics import structural_similarity as ssim

def calculate_ssim(img1, img2):
    return ssim(img1, img2, channel_axis=2, data_range=255)

# LPIPS计算
lpips_model = lpips.LPIPS(net='alex')

def calculate_lpips(img1, img2):
    # 转为tensor，归一化到[-1, 1]
    img1_tensor = torch.from_numpy(img1).permute(2, 0, 1).unsqueeze(0).float() / 127.5 - 1
    img2_tensor = torch.from_numpy(img2).permute(2, 0, 1).unsqueeze(0).float() / 127.5 - 1
    return lpips_model(img1_tensor, img2_tensor).item()
```

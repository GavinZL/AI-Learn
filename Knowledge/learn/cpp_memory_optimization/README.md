# C++ 内存优化技术

## 主题概述

本知识库从 C++ 应用开发角度出发，系统性地介绍内存优化技术，重点结合 Android（ARM架构）和 iOS（ARM64架构）移动平台的特点。涵盖从硬件原理到工程实践的完整技术栈，适用于音视频处理、图形渲染、高性能计算等对内存性能敏感的应用场景。

**核心目标**：通过深入理解内存工作原理，掌握系统性的优化方法，实现应用的低延迟、高吞吐、低内存占用。

## 核心概念速览

| 概念 | 一句话总结 |
|------|-----------|
| **内存模型** | C++11定义的多线程内存访问规则，决定可见性与有序性 |
| **虚拟内存** | OS提供的抽象层，实现进程隔离与内存超额分配 |
| **内存对齐** | 数据按特定边界存放，提升CPU和SIMD访问效率 |
| **内存碎片** | 频繁分配/释放导致的空闲内存不连续问题 |
| **CPU Cache** | 多级缓存层次，局部性原理是优化核心 |
| **DMA** | 绕过CPU直接访问内存，降低CPU负载 |
| **内存池** | 预分配固定大小内存块，消除分配开销 |
| **RAII** | 资源获取即初始化，C++内存安全的基石 |
| **mimalloc** | 微软高性能分配器，分段架构+无锁线程本地分配 |
| **jemalloc** | Meta 开发，多级缓存架构，适合高并发服务 |
| **Scudo** | LLVM 安全分配器，Android 11+ 默认，检测 UAF/溢出 |

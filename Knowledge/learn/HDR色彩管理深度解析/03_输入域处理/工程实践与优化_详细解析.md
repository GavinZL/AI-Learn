            weightSum += w;

## 3. 多帧合成技术

### 3.1 多帧降噪

#### 原理

```
多帧降噪原理：

单帧噪点：随机分布，帧间不相关
信号：静态场景帧间相同

N帧平均后：
- 信号：保持不变
- 噪点：标准差降低为1/√N

SNR提升：10·log10(N) dB
```

#### 实现流程

```python
class MultiFrameDenoiser:
    def __init__(self, num_frames=8):
        self.num_frames = num_frames
        self.frame_buffer = []
        self.motion_threshold = 0.05
    
    def process(self, new_raw_frame):
        # 1. 添加到缓冲区
        self.frame_buffer.append(new_raw_frame)
        if len(self.frame_buffer) > self.num_frames:
            self.frame_buffer.pop(0)
        
        # 2. 对齐（简化版：假设小运动）
        aligned_frames = self.align_frames(self.frame_buffer)
        
        # 3. 时域滤波
        denoised = self.temporal_filter(aligned_frames)
        
        return denoised
    
    def temporal_filter(self, frames):
        """加权时域平均"""
        # 以最新帧为参考
        reference = frames[-1]
        
        weights = []
        for i, frame in enumerate(frames):
            # 计算与参考帧的差异
            diff = np.abs(frame - reference)
            
            # 根据差异计算权重
            # 差异小的区域（静态）权重高
            w = np.exp(-diff / self.motion_threshold)
            weights.append(w)
        
        # 归一化权重
        total_weight = sum(weights)
        
        # 加权平均
        result = sum(f * w for f, w in zip(frames, weights)) / total_weight
        
        return result
```

### 3.2 HDR合成

#### 曝光包围

```
HDR合成流程：

短曝光帧（-2EV）：保留高光细节
中曝光帧（0EV）：正常曝光
长曝光帧（+2EV）：保留阴影细节
    ↓
对齐（考虑运动）
    ↓
合并（加权平均）
    ↓
色调映射
```

```python
class HDRMerger:
    def merge_exposure_stack(self, raw_frames, exposures):
        """
        合并多曝光RAW帧
        
        Args:
            raw_frames: 不同曝光的RAW帧列表
            exposures: 对应的曝光值（EV）
        """
        # 1. 对齐
        aligned = self.align_stack(raw_frames)
        
        # 2. 计算权重
        weights = []
        for frame in aligned:
            # 基于亮度的权重
            # 避免过曝和欠曝区域
            w = self.compute_weight(frame)
            weights.append(w)
        
        # 3. 合并
        merged = np.zeros_like(aligned[0], dtype=np.float32)
        total_weight = np.zeros_like(aligned[0], dtype=np.float32)
        
        for frame, w in zip(aligned, weights):
            # 转换到线性空间（考虑曝光差异）
            linear = frame * (2.0 ** exposures[i])
            merged += linear * w
            total_weight += w
        
        merged /= total_weight
        
        return merged
    
    def compute_weight(self, frame):
        """计算像素权重"""
        # 基于像素值的钟形曲线
        # 中间调权重最高，两端权重低
        z = frame / 4095.0  # 归一化
        w = np.exp(-4 * (z - 0.5)**2 / 0.25)
        return w
```

class YUVFormatConverter:
    """YUV格式转换器"""
    
    @staticmethod
    def nv12_to_i420(nv12_data, width, height):
        """NV12转I420"""
        y_size = width * height
        uv_size = width * height // 4
        
        y = nv12_data[:y_size]
        uv = nv12_data[y_size:]
        
        # 分离UV
        u = uv[0::2]
        v = uv[1::2]
        
        return y.tobytes() + u.tobytes() + v.tobytes()
    
    @staticmethod
    def i420_to_nv12(i420_data, width, height):
        """I420转NV12"""
        y_size = width * height
        uv_size = width * height // 4
        
        y = i420_data[:y_size]
        u = i420_data[y_size:y_size+uv_size]
        v = i420_data[y_size+uv_size:]
        
        # 交错UV
        uv = np.empty(uv_size * 2, dtype=np.uint8)
        uv[0::2] = u
        uv[1::2] = v
        
        return y.tobytes() + uv.tobytes()
    
    @staticmethod
    def yuv420p_to_yuv444(y, u, v, width, height):
        """YUV 4:2:0上采样到4:4:4"""
        # Y保持原样
        y_444 = y.reshape(height, width)
        
        # UV上采样
        u_420 = u.reshape(height//2, width//2)
        v_420 = v.reshape(height//2, width//2)
        
        u_444 = cv2.resize(u_420, (width, height), interpolation=cv2.INTER_LINEAR)
        v_444 = cv2.resize(v_420, (width, height), interpolation=cv2.INTER_LINEAR)
        
        return y_444, u_444, v_444

import time
import functools

def profile(func):
    """性能分析装饰器"""
    @functools.wraps(func)
    def wrapper(*args, **kwargs):
        start = time.perf_counter()
        result = func(*args, **kwargs)
        elapsed = time.perf_counter() - start
        print(f"{func.__name__}: {elapsed*1000:.2f}ms")
        return result
    return wrapper

class ISPProfiler:
    def __init__(self):

    def record(self, stage_name, elapsed_ms):
        if stage_name not in self.timings:

    def report(self):
        print("\n=== ISP性能报告 ===")

        for stage, times in self.timings.items():
            avg = np.mean(times)
            total += avg
            print(f"{stage}: {avg:.2f}ms")

### 6.2 瓶颈识别

```
常见性能瓶颈：

1. 内存带宽瓶颈
   症状：CPU/GPU利用率低，等待内存
   解决：分块处理、压缩数据、减少拷贝

2. 算法复杂度瓶颈
   症状：CPU/GPU满载，处理时间长
   解决：简化算法、使用近似、并行化

3. 同步瓶颈
   症状：CPU和GPU互相等待
   解决：异步处理、双缓冲、流水线

4. 缓存未命中
   症状：访问模式不规则
   解决：调整访问模式、预取、分块
```

## 总结

RAW/YUV处理的工程实践涉及多个层面的优化：

1. **算法层面**：在质量和速度间找到平衡
2. **硬件层面**：充分利用专用ISP、GPU、NPU
3. **系统层面**：优化内存、带宽、同步
4. **跨平台**：抽象差异，统一接口
5. **调试评估**：客观指标+主观评估

核心原则：
- **理解约束**：功耗、延迟、内存
- **测量优先**：用数据指导优化
- **权衡取舍**：没有完美的方案，只有合适的方案

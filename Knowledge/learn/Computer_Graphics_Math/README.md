## 学习路径

根据不同的学习目标，推荐以下学习路径：

### 路径一：快速入门（1-2天）

适合：想快速了解图形学数学全貌的开发者

1. 阅读 [Computer_Graphics_Math_深度解析.md](./Computer_Graphics_Math_深度解析.md) 获取全景视图
2. 重点理解矩阵变换的基本流程（MVP矩阵）
3. 了解旋转表示的选择策略

### 路径二：系统学习（1周）

适合：需要深入理解图形学数学基础的开发者

**第一阶段：矩阵基础（2-3天）**
1. [图形学矩阵基础_详细解析.md](./01_矩阵与变换/图形学矩阵基础_详细解析.md)
   - 理解列向量/行向量约定差异
   - 掌握平移、旋转、缩放矩阵的构造
   - 学习视图矩阵和投影矩阵原理
   - 了解矩阵分解方法（SVD、QR、LU）

**第二阶段：旋转表示（2-3天）**
2. [四元数与旋转表示_详细解析.md](./02_旋转表示与四元数/四元数与旋转表示_详细解析.md)
   - 理解欧拉角与万向锁问题
   - 掌握轴角表示与罗德里格斯公式
   - 深入学习四元数数学与运算
   - 实践四元数插值（SLERP/NLERP）

**第三阶段：渲染管线架构（2-3天）**
3. [IMR与TBDR渲染管线_详细解析.md](./03_渲染管线架构/IMR与TBDR渲染管线_详细解析.md)
   - 理解 GPU 渲染管线的完整流程
   - 掌握 IMR、TBR、TBDR 三种架构的差异
   - 学习 Apple Metal 的 TBDR 实现机制
   - 了解渲染优化实践（HSR、MSAA、延迟渲染）

**第四阶段：渲染优化与多线程（3-4天）**
4. [移动端图形渲染优化_详细解析.md](./04_渲染优化实践/移动端图形渲染优化_详细解析.md)
   - 深入理解移动端 GPU 架构特性
   - 掌握渲染状态管理与 Draw Call 批处理
   - 学习内存带宽优化与 Overdraw 消除技术
   - 了解 Tile Memory 与 Render Pass 优化

5. [多线程图形编程实践_详细解析.md](./05_多线程图形编程/多线程图形编程实践_详细解析.md)
   - 理解图形 API 多线程模型
   - 掌握 Metal 多线程命令编码
   - 学习 GPU-CPU 同步机制
   - 了解资源生命周期与跨线程共享

**第五阶段：渲染系统设计（2-3天）**
6. [高性能渲染系统设计_详细解析.md](./06_渲染系统设计/高性能渲染系统设计_详细解析.md)
   - 理解渲染管线架构设计（Forward/Deferred/Hybrid）
   - 掌握资源管理系统与内存预算控制
   - 学习场景图与可见性管理
   - 了解 Frame Graph 调度与自适应系统

**第六阶段：综合应用（1-2天）**
7. 回顾 [Computer_Graphics_Math_深度解析.md](./Computer_Graphics_Math_深度解析.md) 的知识关联部分
8. 尝试实现简单的变换系统或渲染管线

### 路径三：专题突破（按需）

适合：针对特定问题寻求解决方案

| 问题场景 | 推荐阅读 |

| 优化大规模矩阵运算 | 图形学矩阵基础 → 数值稳定性与性能优化 |
| 理解移动端 GPU 渲染机制 | 渲染管线架构 → TBR/TBDR 架构详解 |

## 2. 欧拉角（Euler Angles）

### 2.1 欧拉角的定义

欧拉角使用三个角度 $(\alpha, \beta, \gamma)$ 来描述三维旋转。根据旋转轴的选择和旋转顺序，共有**12种有效约定**（不考虑重复轴的24种中，12种是有效的）。

#### 内旋（Intrinsic Rotation）vs 外旋（Extrinsic Rotation）

- **内旋**：绕物体自身的坐标轴旋转（随物体转动的坐标系）
- **外旋**：绕固定世界坐标轴旋转

**关键性质**：内旋的 $XYZ$ 等价于外旋的 $ZYX$（顺序相反）。

```
┌─────────────────────────────────────────────────────────────────┐
│              内旋 vs 外旋 直观对比                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   内旋 (Intrinsic)              外旋 (Extrinsic)                │
│   ─────────────────             ─────────────────               │
│                                                                 │
│   初始: 飞机平飞                 初始: 飞机平飞                  │
│      ↑ Y                          ↑ Y                           │
│      │                            │                             │
│      └──→ X                       └──→ X                        │
│     /                              /                            │
│    Z                               Z                            │
│                                                                 │
│   1. 绕X轴滚转90°                1. 绕固定X轴滚转90°             │
│      (飞机侧飞)                   (飞机侧飞)                     │
│      → Y                          → Y                           │
│      │                            │                             │
│      ↓ Z                          ↓ Z (仍指向地面)               │
│                                                                 │
│   2. 绕新的Y轴俯仰               2. 绕固定Y轴俯仰                │
│      (Y已随飞机转动)              (Y仍指向上方)                  │
│      结果: 机头下倾               结果: 机头侧向偏转             │
│                                                                 │
│   结论: 内旋的轴随物体变化，外旋的轴固定在世界坐标系               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 常见约定

| 约定名称 | 旋转顺序 | 典型应用 | 说明 |
|---------|---------|---------|------|
| **XYZ（Tait-Bryan）** | 绕X→Y→Z | 航空、航海 | 滚转(Roll)-俯仰(Pitch)-偏航(Yaw) |
| **ZYX** | 绕Z→Y→X | 机器人学 | 偏航-俯仰-滚转 |
| **ZXZ** | 绕Z→X→Z | 经典欧拉角 | 天文、陀螺仪 |
| **ZYZ** | 绕Z→Y→Z | 经典欧拉角 | 球坐标系 |

#### XYZ约定（航空标准）

$$
R_{XYZ}(\alpha, \beta, \gamma) = R_X(\alpha) \cdot R_Y(\beta) \cdot R_Z(\gamma)
$$

其中：

$$
R_X(\alpha) = \begin{bmatrix} 1 & 0 & 0 \\ 0 & \cos\alpha & -\sin\alpha \\ 0 & \sin\alpha & \cos\alpha \end{bmatrix}
$$

$$
R_Y(\beta) = \begin{bmatrix} \cos\beta & 0 & \sin\beta \\ 0 & 1 & 0 \\ -\sin\beta & 0 & \cos\beta \end{bmatrix}
$$

$$
R_Z(\gamma) = \begin{bmatrix} \cos\gamma & -\sin\gamma & 0 \\ \sin\gamma & \cos\gamma & 0 \\ 0 & 0 & 1 \end{bmatrix}
$$

```python
import numpy as np

def euler_to_rotation_matrix_xyz(roll, pitch, yaw):
    """
    欧拉角(XYZ)转旋转矩阵
    roll: 绕X轴旋转 (滚转)
    pitch: 绕Y轴旋转 (俯仰)
    yaw: 绕Z轴旋转 (偏航)
    角度单位为弧度
    """
    # 绕X轴 (滚转)
    Rx = np.array([
        [1, 0, 0],
        [0, np.cos(roll), -np.sin(roll)],
        [0, np.sin(roll), np.cos(roll)]
    ])
    
    # 绕Y轴 (俯仰)
    Ry = np.array([
        [np.cos(pitch), 0, np.sin(pitch)],
        [0, 1, 0],
        [-np.sin(pitch), 0, np.cos(pitch)]
    ])
    
    # 绕Z轴 (偏航)
    Rz = np.array([
        [np.cos(yaw), -np.sin(yaw), 0],
        [np.sin(yaw), np.cos(yaw), 0],
        [0, 0, 1]
    ])
    
    # 注意: 矩阵乘法顺序为 Rz * Ry * Rx (从右到左应用)
    return Rz @ Ry @ Rx

# 示例：45度偏航，30度俯仰，15度滚转
roll = np.radians(15)
pitch = np.radians(30)
yaw = np.radians(45)

R = euler_to_rotation_matrix_xyz(roll, pitch, yaw)
print("旋转矩阵:")
print(R)
```

### 2.3 万向锁问题（Gimbal Lock）深入分析

#### 什么是万向锁？

**万向锁**是欧拉角表示的致命缺陷：当第二个旋转角为 $\pm 90°$ 时，第一次和第三次旋转会绕同一轴进行，导致**丢失一个自由度**。

**直观类比**：想象一个陀螺仪的三环结构。当外环和中环成90°时，内环的轴与外环的轴对齐。此时，转动外环和转动内环产生相同的效果——你失去了一个独立的旋转轴。

```
┌─────────────────────────────────────────────────────────────────┐
│                   万向锁状态示意图                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   正常状态 (pitch ≠ 90°)           万向锁状态 (pitch = 90°)      │
│   ─────────────────────           ─────────────────────         │
│                                                                 │
│        Z (偏航轴)                      Z                        │
│        ↑                               ↑                        │
│        │    Y (俯仰轴)                 │    Y                   │
│        │   ↗                           │   ↗                    │
│        │  /                            │  /                     │
│        │ /                             │/                      │
│        ●──────→ X (滚转轴)             ●──────→ X              │
│                                        \                       │
│                                         \                      │
│                                          ↘                     │
│                                           X 与 Z 对齐！         │
│                                                                 │
│   三个轴相互独立                      X轴和Z轴共线               │
│   可以独立控制滚转、俯仰、偏航         滚转和偏航变成同一操作      │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

#### 数学证明：为什么会丢失一个自由度

考虑ZYX欧拉角（偏航-俯仰-滚转）：

$$
R = R_Z(\psi) \cdot R_Y(\theta) \cdot R_X(\phi)
$$

当 $\theta = 90°$ 时：

$$
R_Y(90°) = \begin{bmatrix} 0 & 0 & 1 \\ 0 & 1 & 0 \\ -1 & 0 & 0 \end{bmatrix}
$$

计算 $R_Z(\psi) \cdot R_Y(90°)$：

$$
\begin{bmatrix} \cos\psi & -\sin\psi & 0 \\ \sin\psi & \cos\psi & 0 \\ 0 & 0 & 1 \end{bmatrix} \cdot \begin{bmatrix} 0 & 0 & 1 \\ 0 & 1 & 0 \\ -1 & 0 & 0 \end{bmatrix} = \begin{bmatrix} 0 & -\sin\psi & \cos\psi \\ 0 & \cos\psi & \sin\psi \\ -1 & 0 & 0 \end{bmatrix}
$$

再乘以 $R_X(\phi)$：

$$
R = \begin{bmatrix} 0 & -\sin\psi\cos\phi - \cos\psi\sin\phi & \cos\psi\cos\phi - \sin\psi\sin\phi \\ 0 & \cos\psi\cos\phi - \sin\psi\sin\phi & \sin\psi\cos\phi + \cos\psi\sin\phi \\ -1 & 0 & 0 \end{bmatrix}
$$

利用三角恒等式 $\sin(A+B) = \sin A\cos B + \cos A\sin B$ 和 $\cos(A+B) = \cos A\cos B - \sin A\sin B$：

$$
R = \begin{bmatrix} 0 & -\sin(\psi + \phi) & \cos(\psi + \phi) \\ 0 & \cos(\psi + \phi) & \sin(\psi + \phi) \\ -1 & 0 & 0 \end{bmatrix}
$$

**关键发现**：结果只依赖于 $(\psi + \phi)$，而不是独立的 $\psi$ 和 $\phi$！

这意味着有无穷多组 $(\psi, \phi)$ 对应同一个旋转矩阵——我们失去了独立控制这两个角的能力。

#### 万向锁的实际影响

```python
import numpy as np

def demonstrate_gimbal_lock():
    """
    演示万向锁问题
    """
    # 万向锁状态: pitch = 90°
    pitch = np.radians(90)
    
    # 不同的yaw和roll组合，但yaw+roll相同
    test_cases = [
        (0, 45),      # yaw=0, roll=45
        (30, 15),     # yaw=30, roll=15
        (45, 0),      # yaw=45, roll=0
    ]
    
    print("万向锁演示 (pitch = 90°):")
    print("不同 (yaw, roll) 组合产生的旋转矩阵:\n")
    
    for yaw_deg, roll_deg in test_cases:
        yaw = np.radians(yaw_deg)
        roll = np.radians(roll_deg)
        R = euler_to_rotation_matrix_xyz(roll, pitch, yaw)
        print(f"yaw={yaw_deg:3}°, roll={roll_deg:3}° (sum={yaw_deg+roll_deg}°):")
        print(R.round(6))
        print()
    
    print("结论: 只要 yaw + roll = 45°，结果矩阵相同！")

demonstrate_gimbal_lock()
```

### 2.4 欧拉角的适用场景与局限性

**适用场景**：
- 用户界面（如3D建模软件中的旋转工具）
- 需要直观理解旋转的应用（如航空姿态显示）
- 旋转范围有限的场景（避免接近万向锁）

**局限性**：
- 万向锁导致失去自由度
- 插值不平滑（最短路径问题）
- 旋转顺序不统一导致混乱
- 累积误差导致数值不稳定

## 3. 轴角表示（Axis-Angle）

### 3.1 轴角表示的数学定义

轴角表示使用一个单位向量 $\hat{n}$（旋转轴）和一个标量 $\theta$（旋转角度）来描述旋转：

$$
(\hat{n}, \theta), \quad \hat{n} \in \mathbb{R}^3, \|\hat{n}\| = 1, \theta \in [0, 2\pi)
$$

**直观类比**：想象用螺丝刀拧螺丝——螺丝刀的方向就是旋转轴，拧动的角度就是旋转角。

### 3.2 欧拉旋转定理

**欧拉旋转定理**：三维空间中，任意两个具有相同原点的坐标系之间的旋转，都可以表示为绕某一固定轴的单一旋转。

**数学表述**：对于任意 $R \in SO(3)$，存在单位向量 $\hat{n}$ 和角度 $\theta$ 使得：

$$
R = e^{[\hat{n}]_\times \theta}
$$

其中 $[\hat{n}]_\times$ 是 $\hat{n}$ 的反对称矩阵（叉积矩阵）：

$$
[\hat{n}]_\times = \begin{bmatrix} 0 & -n_z & n_y \\ n_z & 0 & -n_x \\ -n_y & n_x & 0 \end{bmatrix}
$$

### 3.3 轴角 ↔ 旋转矩阵 的转换

#### 轴角转旋转矩阵（罗德里格斯公式）

$$
R = I \cos\theta + (1 - \cos\theta)\hat{n}\hat{n}^T + [\hat{n}]_\times \sin\theta
$$

展开形式：

$$
R = \begin{bmatrix}
\cos\theta + n_x^2(1-\cos\theta) & n_xn_y(1-\cos\theta) - n_z\sin\theta & n_xn_z(1-\cos\theta) + n_y\sin\theta \\
n_yn_x(1-\cos\theta) + n_z\sin\theta & \cos\theta + n_y^2(1-\cos\theta) & n_yn_z(1-\cos\theta) - n_x\sin\theta \\
n_zn_x(1-\cos\theta) - n_y\sin\theta & n_zn_y(1-\cos\theta) + n_x\sin\theta & \cos\theta + n_z^2(1-\cos\theta)
\end{bmatrix}
$$

#### 旋转矩阵转轴角

给定旋转矩阵 $R$，求解 $(\hat{n}, \theta)$：

$$
\theta = \arccos\left(\frac{\text{trace}(R) - 1}{2}\right)
$$

$$
\hat{n} = \frac{1}{2\sin\theta}\begin{bmatrix} R_{32} - R_{23} \\ R_{13} - R_{31} \\ R_{21} - R_{12} \end{bmatrix}
$$

**特殊情况处理**：
- 当 $\theta = 0$：无旋转，旋转轴任意
- 当 $\theta = \pi$：$\sin\theta = 0$，使用对角元素求解

```python
import numpy as np

def axis_angle_to_rotation_matrix(axis, angle):
    """
    轴角表示转旋转矩阵 (Rodrigues公式)
    axis: 旋转轴 (3,) 会被归一化
    angle: 旋转角度 (弧度)
    """
    n = axis / np.linalg.norm(axis)
    nx, ny, nz = n
    
    c = np.cos(angle)
    s = np.sin(angle)
    t = 1 - c
    
    R = np.array([
        [t*nx*nx + c,    t*nx*ny - s*nz, t*nx*nz + s*ny],
        [t*nx*ny + s*nz, t*ny*ny + c,    t*ny*nz - s*nx],
        [t*nx*nz - s*ny, t*ny*nz + s*nx, t*nz*nz + c   ]
    ])
    
    return R

def rotation_matrix_to_axis_angle(R):
    """
    旋转矩阵转轴角表示
    返回: (axis, angle)
    """
    trace = np.trace(R)
    angle = np.arccos(np.clip((trace - 1) / 2, -1, 1))
    
    if np.isclose(angle, 0):
        # 无旋转
        return np.array([1, 0, 0]), 0.0
    
    if np.isclose(angle, np.pi):
        # 180度旋转，使用对角元素
        # 此时 R = I + 2nn^T，所以 n_i^2 = (R_ii + 1) / 2
        nx = np.sqrt((R[0,0] + 1) / 2)
        ny = np.sqrt((R[1,1] + 1) / 2)
        nz = np.sqrt((R[2,2] + 1) / 2)
        
        # 根据非对角元素确定符号
        if R[0,1] < 0: ny = -ny
        if R[0,2] < 0: nz = -nz
        
        return np.array([nx, ny, nz]), angle
    
    # 一般情况
    axis = np.array([
        R[2,1] - R[1,2],
        R[0,2] - R[2,0],
        R[1,0] - R[0,1]
    ]) / (2 * np.sin(angle))
    
    return axis, angle

# 测试
axis = np.array([1, 1, 1])  # 绕对角线旋转
angle = np.radians(60)

R = axis_angle_to_rotation_matrix(axis, angle)
axis_back, angle_back = rotation_matrix_to_axis_angle(R)

print(f"原始轴角: axis={axis/np.linalg.norm(axis)}, angle={np.degrees(angle):.2f}°")
print(f"恢复轴角: axis={axis_back}, angle={np.degrees(angle_back):.2f}°")
```

### 3.4 轴角表示的优缺点

**优点**：
- 直观易懂（有明确的旋转轴和角度）
- 最小表示（3个自由度）
- 无奇异性

**缺点**：
- 旋转→轴角不唯一（$(\hat{n}, \theta)$ 和 $(-\hat{n}, -\theta)$ 表示相同旋转）
- 插值需要特殊处理（角度周期性）
- 组合两个旋转较复杂

## 4. 罗德里格斯旋转公式（Rodrigues' Rotation Formula）

### 4.1 公式的推导过程（几何推导法）

罗德里格斯公式描述了向量绕任意轴旋转后的结果。

**问题设定**：将向量 $\mathbf{v}$ 绕单位轴 $\hat{n}$ 旋转角度 $\theta$，求旋转后的向量 $\mathbf{v}'$。

```
┌─────────────────────────────────────────────────────────────────┐
│              罗德里格斯公式几何推导                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│                        n̂                                        │
│                        ↑                                        │
│                        │                                        │
│                        │                                        │
│                        ●──────────→ v_∥ (平行分量)              │
│                       /                                         │
│                      /                                          │
│           v ────────●                                           │
│            \       / \                                          │
│             \     /   \                                         │
│              \   /     \                                        │
│               \ /       \                                       │
│                ×         ●──────→ v'                            │
│               / \       /                                       │
│              /   \     /                                        │
│             /     \   /                                         │
│            /       \ /                                          │
│           ●─────────●                                           │
│        v_⊥ (垂直分量)  w (旋转平面内正交向量)                     │
│                                                                 │
│   分解: v = v_∥ + v_⊥                                           │
│   其中: v_∥ = (v·n̂)n̂  (平行于旋转轴的分量)                       │
│        v_⊥ = v - v_∥ = -n̂×(n̂×v)  (垂直于旋转轴的分量)          │
│        w = n̂ × v  (与v_⊥正交的向量)                              │
│                                                                 │
│   旋转后: v' = v_∥ + v_⊥cosθ + wsinθ                            │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**推导步骤**：

1. **分解向量**：将 $\mathbf{v}$ 分解为平行于和垂直于 $\hat{n}$ 的分量

$$
\mathbf{v}_\parallel = (\mathbf{v} \cdot \hat{n})\hat{n}
$$

$$
\mathbf{v}_\perp = \mathbf{v} - \mathbf{v}_\parallel = -\hat{n} \times (\hat{n} \times \mathbf{v})
$$

2. **构造旋转平面内的正交基**：

$$
\mathbf{w} = \hat{n} \times \mathbf{v}
$$

$\mathbf{v}_\perp$ 和 $\mathbf{w}$ 构成旋转平面内的一组正交基。

3. **旋转垂直分量**：

在旋转平面内，$\mathbf{v}_\perp$ 旋转 $\theta$ 角度后：

$$
\mathbf{v}'_\perp = \mathbf{v}_\perp \cos\theta + \mathbf{w} \sin\theta
$$

4. **组合结果**：

$$
\mathbf{v}' = \mathbf{v}_\parallel + \mathbf{v}'_\perp = (\mathbf{v} \cdot \hat{n})\hat{n} + (\mathbf{v} - (\mathbf{v} \cdot \hat{n})\hat{n})\cos\theta + (\hat{n} \times \mathbf{v})\sin\theta
$$

整理得**罗德里格斯旋转公式**：

$$
\mathbf{v}' = \mathbf{v}\cos\theta + (\hat{n} \times \mathbf{v})\sin\theta + \hat{n}(\hat{n} \cdot \mathbf{v})(1 - \cos\theta)
$$

### 4.2 向量形式与矩阵形式

#### 向量形式

$$
\mathbf{v}' = \mathbf{v}\cos\theta + (\hat{n} \times \mathbf{v})\sin\theta + \hat{n}(\hat{n} \cdot \mathbf{v})(1 - \cos\theta)
$$

#### 矩阵形式

利用叉积矩阵 $[\hat{n}]_\times$：

$$
\mathbf{v}' = R\mathbf{v}
$$

其中：

$$
R = I\cos\theta + [\hat{n}]_\times\sin\theta + \hat{n}\hat{n}^T(1 - \cos\theta)
$$

这与第3节中的轴角转旋转矩阵公式一致。

### 4.3 指数映射（Exponential Map）与李群/李代数初步

#### 李群与李代数

- **$SO(3)$**（三维旋转群）：所有 $3 \times 3$ 正交矩阵且行列式为1的集合，构成一个李群
- **$\mathfrak{so}(3)$**（$SO(3)$的李代数）：所有 $3 \times 3$ 反对称矩阵的集合

#### 指数映射

罗德里格斯公式实际上是矩阵指数映射的闭式解：

$$
R = \exp([\hat{n}]_\times \theta) = I + [\hat{n}]_\times\sin\theta + [\hat{n}]_\times^2(1 - \cos\theta)
$$

**直观理解**：
- 李代数 $\mathfrak{so}(3)$ 是旋转的"切空间"（角速度空间）
- 指数映射将李代数中的元素（角速度）映射到李群中的元素（旋转矩阵）
- 对数映射是其逆运算

```
┌─────────────────────────────────────────────────────────────────┐
│              指数映射与李群/李代数关系                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   李代数 so(3)                    李群 SO(3)                    │
│   (切空间)                        (旋转流形)                     │
│                                                                 │
│   ω ∈ R³  ──────→  [ω]× ∈ so(3)                                │
│   角速度向量       反对称矩阵                                    │
│                                                                 │
│        ↓ exp                      ↓ exp                         │
│                                                                 │
│   R = exp([ω]×)  ←──────  R ∈ SO(3)                            │
│   旋转矩阵           旋转矩阵                                    │
│                                                                 │
│   其中: ω = θn̂  (轴角表示)                                       │
│                                                                 │
│   exp([ω]×) = I + sinθ/θ · [ω]× + (1-cosθ)/θ² · [ω]×²          │
│                                                                 │
│   当 θ→0 时，exp([ω]×) ≈ I + [ω]×  (小角度近似)                 │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 4.4 罗德里格斯公式在计算机视觉中的应用

**主要应用**：
1. **相机姿态估计**：PnP问题的求解
2. **光束法平差**（Bundle Adjustment）：优化相机参数
3. **视觉SLAM**：位姿图优化
4. **刚体运动学**：角速度积分

### 4.5 代码实现

```python
import numpy as np

def rodrigues_rotation(v, axis, angle):
    """
    使用罗德里格斯公式旋转向量
    v: 待旋转向量 (3,)
    axis: 旋转轴 (3,) 会被归一化
    angle: 旋转角度 (弧度)
    返回: 旋转后的向量
    """
    n = axis / np.linalg.norm(axis)
    
    cos_theta = np.cos(angle)
    sin_theta = np.sin(angle)
    
    # 罗德里格斯公式: v' = v*cosθ + (n×v)*sinθ + n*(n·v)*(1-cosθ)
    term1 = v * cos_theta
    term2 = np.cross(n, v) * sin_theta
    term3 = n * np.dot(n, v) * (1 - cos_theta)
    
    return term1 + term2 + term3

def rodrigues_to_rotation_matrix(rvec):
    """
    罗德里格斯向量转旋转矩阵
    rvec: 罗德里格斯向量 (3,) = θ * n̂
    返回: 3×3旋转矩阵
    """
    theta = np.linalg.norm(rvec)
    
    if theta < 1e-10:
        return np.eye(3)
    
    n = rvec / theta
    return axis_angle_to_rotation_matrix(n, theta)

def rotation_matrix_to_rodrigues(R):
    """
    旋转矩阵转罗德里格斯向量
    返回: 罗德里格斯向量 (3,)
    """
    axis, angle = rotation_matrix_to_axis_angle(R)
    return axis * angle

# 测试
v = np.array([1, 0, 0])
axis = np.array([0, 0, 1])  # 绕Z轴
angle = np.radians(90)

v_rotated = rodrigues_rotation(v, axis, angle)
print(f"原始向量: {v}")
print(f"旋转后向量: {v_rotated}")
print(f"期望结果: [0, 1, 0]")
```

## 5. 四元数（Quaternion）

### 5.1 四元数的数学定义

#### 复数的扩展：从实数到四元数

**复数回顾**：
复数 $z = a + bi$，其中 $i^2 = -1$。复数可以表示二维旋转。

**四元数**：
四元数是复数到四维的扩展，由哈密顿（Hamilton）于1843年发现：

$$
q = w + xi + yj + zk = [w, x, y, z] = [w, \mathbf{v}]
$$

其中 $w$ 是实部（标量部分），$\mathbf{v} = (x, y, z)$ 是虚部（向量部分）。

#### 四元数代数

四元数乘法遵循以下规则（哈密顿关系）：

$$
i^2 = j^2 = k^2 = ijk = -1
$$

由此可导出：

$$
ij = k, \quad jk = i, \quad ki = j
$$

$$
ji = -k, \quad kj = -i, \quad ik = -j
$$

**关键性质**：四元数乘法**不满足交换律**！

```
┌─────────────────────────────────────────────────────────────────┐
│                   四元数乘法表                                    │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│          ×  │   1    i    j    k                                │
│        ─────┼─────────────────────                               │
│          1  │   1    i    j    k                                │
│          i  │   i   -1    k   -j                                │
│          j  │   j   -k   -1    i                                │
│          k  │   k    j   -i   -1                                │
│                                                                 │
│   记忆口诀: ijk = -1，循环顺序 i→j→k→i 为正，逆序为负             │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

#### 标量部分与向量部分

四元数可以表示为标量-向量对：

$$
q = [w, \mathbf{v}] = [w, (x, y, z)]
$$

- **标量部分**：$w$（实部）
- **向量部分**：$\mathbf{v} = (x, y, z)$（虚部）

### 5.2 四元数的基本运算

#### 加法

$$
q_1 + q_2 = [w_1 + w_2, \mathbf{v}_1 + \mathbf{v}_2]
$$

#### 乘法（Hamilton积）

$$
q_1 \cdot q_2 = [w_1w_2 - \mathbf{v}_1 \cdot \mathbf{v}_2, w_1\mathbf{v}_2 + w_2\mathbf{v}_1 + \mathbf{v}_1 \times \mathbf{v}_2]
$$

展开形式：

$$
\begin{bmatrix} w_1 \\ x_1 \\ y_1 \\ z_1 \end{bmatrix} \cdot \begin{bmatrix} w_2 \\ x_2 \\ y_2 \\ z_2 \end{bmatrix} = \begin{bmatrix} w_1w_2 - x_1x_2 - y_1y_2 - z_1z_2 \\ w_1x_2 + x_1w_2 + y_1z_2 - z_1y_2 \\ w_1y_2 - x_1z_2 + y_1w_2 + z_1x_2 \\ w_1z_2 + x_1y_2 - y_1x_2 + z_1w_2 \end{bmatrix}
$$

#### 共轭

$$
q^* = [w, -\mathbf{v}] = [w, -x, -y, -z]
$$

性质：$(q_1 \cdot q_2)^* = q_2^* \cdot q_1^*$

#### 模（Norm）

$$
\|q\| = \sqrt{w^2 + x^2 + y^2 + z^2} = \sqrt{q \cdot q^*}
$$

#### 归一化

$$
\hat{q} = \frac{q}{\|q\|}
$$

#### 逆四元数

$$
q^{-1} = \frac{q^*}{\|q\|^2}
$$

对于单位四元数：$q^{-1} = q^*$

```python
import numpy as np

class Quaternion:
    """
    四元数类实现
    """
    def __init__(self, w=1.0, x=0.0, y=0.0, z=0.0):
        self.w = w
        self.x = x
        self.y = y
        self.z = z
    
    @property
    def scalar(self):
        return self.w
    
    @property
    def vector(self):
        return np.array([self.x, self.y, self.z])
    
    def __repr__(self):
        return f"Quaternion({self.w:.4f}, {self.x:.4f}, {self.y:.4f}, {self.z:.4f})"
    
    def __add__(self, other):
        return Quaternion(
            self.w + other.w,
            self.x + other.x,
            self.y + other.y,
            self.z + other.z
        )
    
    def __mul__(self, other):
        """Hamilton积"""
        if isinstance(other, Quaternion):
            w = self.w * other.w - self.x * other.x - self.y * other.y - self.z * other.z
            x = self.w * other.x + self.x * other.w + self.y * other.z - self.z * other.y
            y = self.w * other.y - self.x * other.z + self.y * other.w + self.z * other.x
            z = self.w * other.z + self.x * other.y - self.y * other.x + self.z * other.w
            return Quaternion(w, x, y, z)
        else:
            # 标量乘法
            return Quaternion(self.w * other, self.x * other, self.y * other, self.z * other)
    
    def conjugate(self):
        """共轭"""
        return Quaternion(self.w, -self.x, -self.y, -self.z)
    
    def norm(self):
        """模"""
        return np.sqrt(self.w**2 + self.x**2 + self.y**2 + self.z**2)
    
    def normalize(self):
        """归一化"""
        n = self.norm()
        if n < 1e-10:
            return Quaternion(1, 0, 0, 0)
        return Quaternion(self.w/n, self.x/n, self.y/n, self.z/n)
    
    def inverse(self):
        """逆"""
        n2 = self.norm()**2
        conj = self.conjugate()
        return Quaternion(conj.w/n2, conj.x/n2, conj.y/n2, conj.z/n2)
    
    def to_array(self):
        return np.array([self.w, self.x, self.y, self.z])

# 测试基本运算
q1 = Quaternion(1, 2, 3, 4)
q2 = Quaternion(0, 1, 0, 0)

print(f"q1 = {q1}")
print(f"q2 = {q2}")
print(f"q1 + q2 = {q1 + q2}")
print(f"q1 * q2 = {q1 * q2}")
print(f"q1的共轭 = {q1.conjugate()}")
print(f"q1的模 = {q1.norm():.4f}")
print(f"q1归一化 = {q1.normalize()}")
print(f"q1的逆 = {q1.inverse()}")
```

### 5.3 单位四元数与旋转

#### 旋转四元数的定义

单位四元数可以表示三维旋转：

$$
q = \cos\frac{\theta}{2} + \sin\frac{\theta}{2}(x\mathbf{i} + y\mathbf{j} + z\mathbf{k}) = [\cos\frac{\theta}{2}, \sin\frac{\theta}{2}\hat{n}]
$$

其中：
- $\theta$ 是旋转角度
- $\hat{n} = (x, y, z)$ 是单位旋转轴

#### 旋转操作：$\mathbf{p}' = q\mathbf{p}q^*$

要旋转一个向量（或点）$\mathbf{p}$，首先将其表示为纯虚四元数：

$$
\mathbf{p} = [0, p_x, p_y, p_z] = [0, \mathbf{p}]
$$

旋转后的结果为：

$$
\mathbf{p}' = q \cdot \mathbf{p} \cdot q^*
$$

#### 为什么是 $\theta/2$？（双覆盖性的直观解释）

```
┌─────────────────────────────────────────────────────────────────┐
│              为什么旋转角是θ/2？双覆盖性解释                       │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   四元数旋转公式: p' = qpq*                                     │
│                                                                 │
│   想象一个点在三维空间中旋转:                                     │
│                                                                 │
│        q = cos(θ/2) + sin(θ/2)n̂                                 │
│                                                                 │
│   在 qpq* 中，q 作用一次，q* 作用一次，总共"旋转"了两次            │
│   每次旋转 θ/2，总共旋转 θ                                        │
│                                                                 │
│   ┌─────────────────────────────────────────┐                  │
│   │                                         │                  │
│   │     p ────→ qp ────→ qpq* = p'          │                  │
│   │          (θ/2)    (θ/2)                 │                  │
│   │           ↓        ↓                    │                  │
│   │         第一次    第二次                │                  │
│   │         旋转      旋转                  │                  │
│   │                                         │                  │
│   │     总旋转角度 = θ/2 + θ/2 = θ          │                  │
│   │                                         │                  │
│   └─────────────────────────────────────────┘                  │
│                                                                 │
│   双覆盖性:                                                     │
│   q 和 -q 表示相同的旋转！                                       │
│   因为: (-q)p(-q)* = (-q)p(-q*) = qpq* = p'                     │
│                                                                 │
│   这对应于SU(2) → SO(3)的2对1映射                               │
│   拓扑上，四元数构成3维球面S³，旋转群SO(3)是实射影空间RP³        │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**直观类比**：想象你站在地球北极，要告诉朋友你面向的方向。你可以说"面向格林尼治"（0°经度），也可以说"背向国际日期变更线"（180°经度）。这两种描述在球面上是相对的点，但表示你面向的同一方向。四元数 $q$ 和 $-q$ 就像这两个相对的点，表示同一个旋转。

```python
def quaternion_from_axis_angle(axis, angle):
    """
    从轴角创建单位四元数
    axis: 旋转轴 (3,)
    angle: 旋转角度 (弧度)
    """
    n = axis / np.linalg.norm(axis)
    half_angle = angle / 2
    w = np.cos(half_angle)
    v = n * np.sin(half_angle)
    return Quaternion(w, v[0], v[1], v[2])

def quaternion_rotate_vector(q, v):
    """
    使用四元数旋转向量
    q: 单位四元数
    v: 向量 (3,)
    返回: 旋转后的向量
    """
    # 将向量转为纯虚四元数
    p = Quaternion(0, v[0], v[1], v[2])
    
    # p' = qpq*
    q_conj = q.conjugate()
    p_rotated = q * p * q_conj
    
    return np.array([p_rotated.x, p_rotated.y, p_rotated.z])

# 测试：绕Z轴旋转90度
axis = np.array([0, 0, 1])
angle = np.radians(90)
q = quaternion_from_axis_angle(axis, angle)

v = np.array([1, 0, 0])
v_rotated = quaternion_rotate_vector(q, v)

print(f"旋转轴: {axis}")
print(f"旋转角: {np.degrees(angle)}°")
print(f"四元数: {q}")
print(f"原始向量: {v}")
print(f"旋转后向量: {v_rotated}")
print(f"期望结果: [0, 1, 0]")

# 验证双覆盖性
q_neg = Quaternion(-q.w, -q.x, -q.y, -q.z)
v_rotated_neg = quaternion_rotate_vector(q_neg, v)
print(f"\n使用-q旋转结果: {v_rotated_neg}")
print(f"q和-q表示相同旋转: {np.allclose(v_rotated, v_rotated_neg)}")
```

### 5.4 四元数 ↔ 旋转矩阵 互转

#### 四元数转旋转矩阵

给定单位四元数 $q = [w, x, y, z]$：

$$
R = \begin{bmatrix}
1 - 2(y^2 + z^2) & 2(xy - zw) & 2(xz + yw) \\
2(xy + zw) & 1 - 2(x^2 + z^2) & 2(yz - xw) \\
2(xz - yw) & 2(yz + xw) & 1 - 2(x^2 + y^2)
\end{bmatrix}
$$

#### 旋转矩阵转四元数（Shepperd方法）

给定旋转矩阵 $R$，求四元数 $q = [w, x, y, z]$：

**Shepperd方法**（数值稳定）：

计算对角元素：

$$
t_0 = 1 + R_{00} + R_{11} + R_{22}
$$

$$
t_1 = 1 + R_{00} - R_{11} - R_{22}
$$

$$
t_2 = 1 - R_{00} + R_{11} - R_{22}
$$

$$
t_3 = 1 - R_{00} - R_{11} + R_{22}
$$

选择最大的 $t_i$ 来计算四元数分量：

- 如果 $t_0$ 最大：$w = \frac{1}{2}\sqrt{t_0}$，$x = \frac{R_{21} - R_{12}}{4w}$，$y = \frac{R_{02} - R_{20}}{4w}$，$z = \frac{R_{10} - R_{01}}{4w}$
- 如果 $t_1$ 最大：$x = \frac{1}{2}\sqrt{t_1}$，$w = \frac{R_{21} - R_{12}}{4x}$，$y = \frac{R_{01} + R_{10}}{4x}$，$z = \frac{R_{02} + R_{20}}{4x}$
- 类似处理 $t_2$ 和 $t_3$

```python
def quaternion_to_rotation_matrix(q):
    """
    四元数转旋转矩阵
    q: Quaternion对象
    """
    w, x, y, z = q.w, q.x, q.y, q.z
    
    R = np.array([
        [1 - 2*(y*y + z*z), 2*(x*y - z*w),     2*(x*z + y*w)    ],
        [2*(x*y + z*w),     1 - 2*(x*x + z*z), 2*(y*z - x*w)    ],
        [2*(x*z - y*w),     2*(y*z + x*w),     1 - 2*(x*x + y*y)]
    ])
    
    return R

def rotation_matrix_to_quaternion(R):
    """
    旋转矩阵转四元数 (Shepperd方法，数值稳定)
    """
    t0 = 1.0 + R[0,0] + R[1,1] + R[2,2]
    t1 = 1.0 + R[0,0] - R[1,1] - R[2,2]
    t2 = 1.0 - R[0,0] + R[1,1] - R[2,2]
    t3 = 1.0 - R[0,0] - R[1,1] + R[2,2]
    
    # 选择最大的t
    max_t = max(t0, t1, t2, t3)
    
    if max_t == t0:
        w = 0.5 * np.sqrt(t0)
        x = (R[2,1] - R[1,2]) / (4*w)
        y = (R[0,2] - R[2,0]) / (4*w)
        z = (R[1,0] - R[0,1]) / (4*w)
    elif max_t == t1:
        x = 0.5 * np.sqrt(t1)
        w = (R[2,1] - R[1,2]) / (4*x)
        y = (R[0,1] + R[1,0]) / (4*x)
        z = (R[0,2] + R[2,0]) / (4*x)
    elif max_t == t2:
        y = 0.5 * np.sqrt(t2)
        w = (R[0,2] - R[2,0]) / (4*y)
        x = (R[0,1] + R[1,0]) / (4*y)
        z = (R[1,2] + R[2,1]) / (4*y)
    else:
        z = 0.5 * np.sqrt(t3)
        w = (R[1,0] - R[0,1]) / (4*z)
        x = (R[0,2] + R[2,0]) / (4*z)
        y = (R[1,2] + R[2,1]) / (4*z)
    
    q = Quaternion(w, x, y, z)
    return q.normalize()
```

### 5.5 四元数 ↔ 欧拉角 互转

#### 四元数转欧拉角（ZYX顺序）

$$
\text{roll} = \arctan2(2(w x + y z), 1 - 2(x^2 + y^2))
$$

$$
\text{pitch} = \arcsin(2(w y - z x))
$$

$$
\text{yaw} = \arctan2(2(w z + x y), 1 - 2(y^2 + z^2))
$$

**注意**：需要处理 pitch = ±90° 的万向锁情况。

#### 欧拉角转四元数

$$
q = q_z(\text{yaw}) \cdot q_y(\text{pitch}) \cdot q_x(\text{roll})
$$

其中：

$$
q_x(\alpha) = [\cos\frac{\alpha}{2}, \sin\frac{\alpha}{2}, 0, 0]
$$

$$
q_y(\beta) = [\cos\frac{\beta}{2}, 0, \sin\frac{\beta}{2}, 0]
$$

$$
q_z(\gamma) = [\cos\frac{\gamma}{2}, 0, 0, \sin\frac{\gamma}{2}]
$$

```python
def quaternion_to_euler_zyx(q):
    """
    四元数转欧拉角 (ZYX顺序: yaw-pitch-roll)
    返回: (roll, pitch, yaw) 单位为弧度
    """
    w, x, y, z = q.w, q.x, q.y, q.z
    
    # Roll (X轴)
    sinr_cosp = 2 * (w * x + y * z)
    cosr_cosp = 1 - 2 * (x * x + y * y)
    roll = np.arctan2(sinr_cosp, cosr_cosp)
    
    # Pitch (Y轴)
    sinp = 2 * (w * y - z * x)
    if abs(sinp) >= 1:
        # 万向锁情况
        pitch = np.copysign(np.pi / 2, sinp)
    else:
        pitch = np.arcsin(sinp)
    
    # Yaw (Z轴)
    siny_cosp = 2 * (w * z + x * y)
    cosy_cosp = 1 - 2 * (y * y + z * z)
    yaw = np.arctan2(siny_cosp, cosy_cosp)
    
    return roll, pitch, yaw

def euler_to_quaternion_zyx(roll, pitch, yaw):
    """
    欧拉角转四元数 (ZYX顺序)
    roll, pitch, yaw: 单位为弧度
    """
    cy = np.cos(yaw * 0.5)
    sy = np.sin(yaw * 0.5)
    cp = np.cos(pitch * 0.5)
    sp = np.sin(pitch * 0.5)
    cr = np.cos(roll * 0.5)
    sr = np.sin(roll * 0.5)
    
    w = cr * cp * cy + sr * sp * sy
    x = sr * cp * cy - cr * sp * sy
    y = cr * sp * cy + sr * cp * sy
    z = cr * cp * sy - sr * sp * cy
    
    return Quaternion(w, x, y, z)
```

$$
\theta = 2 \arccos(w)
$$

$$
\hat{n} = \frac{(x, y, z)}{\sin(\theta/2)} = \frac{(x, y, z)}{\sqrt{1 - w^2}}
$$

```python
def quaternion_to_axis_angle(q):
    """
    四元数转轴角
    返回: (axis, angle) angle为弧度
    """
    q = q.normalize()
    angle = 2 * np.arccos(np.clip(q.w, -1, 1))
    s = np.sqrt(1 - q.w**2)
    
    if s < 1e-10:
        # 无旋转或180度
        axis = np.array([1, 0, 0])
    else:
        axis = np.array([q.x, q.y, q.z]) / s
    
    return axis, angle
```

## 6. 四元数插值

### 6.1 线性插值（LERP）及其问题

**LERP定义**：

$$
\text{LERP}(q_1, q_2, t) = (1-t)q_1 + tq_2, \quad t \in [0, 1]
$$

**问题**：
1. 插值结果不是单位四元数，需要归一化
2. 插值速度不均匀（在两端快，中间慢）
3. 不遵循球面最短路径

```
┌─────────────────────────────────────────────────────────────────┐
│              LERP的问题示意图                                     │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   在单位球面上:                                                  │
│                                                                 │
│                    q1                                            │
│                     ●                                            │
│                    /|                                            │
│                   / |                                            │
│                  /  |                                            │
│                 /   |  LERP路径 (球内直线)                        │
│                /    |  ↘                                         │
│               /     |    ● t=0.5 (不在球面上!)                   │
│              /      |   /                                        │
│             /       |  /                                         │
│            /        | /                                          │
│           ●─────────●/                                           │
│          q2         q1'                                          │
│                                                                 │
│   问题:                                                          │
│   1. t=0.5的点在球内，不在单位球面上                              │
│   2. 需要归一化: q' = q / ||q||                                  │
│   3. 归一化后，插值参数t与实际旋转角度不成正比                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 6.2 归一化线性插值（NLERP）

**NLERP定义**：

$$
\text{NLERP}(q_1, q_2, t) = \frac{(1-t)q_1 + tq_2}{\|(1-t)q_1 + tq_2\|}
$$

**优点**：结果在单位球面上

**缺点**：速度仍不均匀，但比LERP好

```python
def quaternion_lerp(q1, q2, t):
    """线性插值 (LERP)"""
    return Quaternion(
        (1-t)*q1.w + t*q2.w,
        (1-t)*q1.x + t*q2.x,
        (1-t)*q1.y + t*q2.y,
        (1-t)*q1.z + t*q2.z
    )

def quaternion_nlerp(q1, q2, t):
    """归一化线性插值 (NLERP)"""
    lerp = quaternion_lerp(q1, q2, t)
    return lerp.normalize()
```

### 6.3 球面线性插值（SLERP）

#### SLERP的几何意义

SLERP在单位四元数球面上沿大圆弧插值，保证：
1. 路径最短（恒定角速度）
2. 结果始终在单位球面上
3. 插值参数 $t$ 与实际旋转角度成正比

#### SLERP公式推导

设 $q_1$ 和 $q_2$ 之间的夹角为 $\Omega$，则：

$$
\cos\Omega = q_1 \cdot q_2 = w_1w_2 + x_1x_2 + y_1y_2 + z_1z_2
$$

SLERP公式：

$$
\text{SLERP}(q_1, q_2, t) = \frac{\sin((1-t)\Omega)}{\sin\Omega} q_1 + \frac{\sin(t\Omega)}{\sin\Omega} q_2
$$

**几何解释**：这是球面上的线性插值，类似于平面上的三角函数插值。

```
┌─────────────────────────────────────────────────────────────────┐
│              SLERP几何意义                                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   在单位四元数球面S³上:                                          │
│                                                                 │
│                    q1                                            │
│                     ●                                            │
│                    / \                                           │
│                   /   \                                          │
│                  /     \                                         │
│                 /       \  大圆弧 (最短路径)                      │
│                /    ●    \                                       │
│               /   q(t)    \                                      │
│              /             \                                     │
│             /               \                                    │
│            ●─────────────────●                                   │
│           q2                 q1'                                 │
│                                                                 │
│   q(t) = sin((1-t)Ω)/sin(Ω) * q1 + sin(tΩ)/sin(Ω) * q2          │
│                                                                 │
│   性质:                                                          │
│   - 路径在球面上                                                 │
│   - 恒定角速度                                                   │
│   - 最短路径                                                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

#### 数值稳定性处理（当角度接近0时）

当 $\Omega \approx 0$ 时，$\sin\Omega \approx 0$，直接计算会导致数值不稳定。

**解决方案**：
- 如果 $\Omega$ 很小，回退到NLERP
- 或者使用泰勒展开近似

```python
def quaternion_slerp(q1, q2, t):
    """
    球面线性插值 (SLERP)
    返回: 插值后的单位四元数
    """
    q1 = q1.normalize()
    q2 = q2.normalize()
    
    dot = q1.w*q2.w + q1.x*q2.x + q1.y*q2.y + q1.z*q2.z
    
    # 如果点积为负，取反q2以走最短路径
    if dot < 0:
        q2 = Quaternion(-q2.w, -q2.x, -q2.y, -q2.z)
        dot = -dot
    
    # 如果四元数非常接近，使用NLERP避免数值不稳定
    DOT_THRESHOLD = 0.9995
    if dot > DOT_THRESHOLD:
        # 线性插值并归一化
        result = quaternion_lerp(q1, q2, t)
        return result.normalize()
    
    # 计算夹角
    theta_0 = np.arccos(dot)  # θ0 = 夹角
    theta = theta_0 * t       # θ = 插值角度
    
    sin_theta = np.sin(theta)
    sin_theta_0 = np.sin(theta_0)
    
    s0 = np.cos(theta) - dot * sin_theta / sin_theta_0
    s1 = sin_theta / sin_theta_0
    
    return Quaternion(
        q1.w * s0 + q2.w * s1,
        q1.x * s0 + q2.x * s1,
        q1.y * s0 + q2.y * s1,
        q1.z * s0 + q2.z * s1
    )
```

### 6.4 SQUAD（球面样条插值）

SQUAD（Spherical Quadrangle Interpolation）是SLERP的扩展，用于通过多个关键四元数的平滑插值。

**应用场景**：
- 骨骼动画的关键帧插值
- 相机路径的平滑运动
- 机器人轨迹规划

**基本思想**：
- 使用四个控制点 $q_{i-1}, q_i, q_{i+1}, q_{i+2}$
- 计算中间控制点 $a_i$ 和 $b_i$
- 在区间 $[q_i, q_{i+1}]$ 上使用SLERP两次

```python
def quaternion_squad(q0, q1, q2, q3, t):
    """
    SQUAD插值
    q0, q1, q2, q3: 连续四个控制点
    t: 在[q1, q2]区间内的插值参数
    """
    # 计算中间控制点
    def intermediate(qi, qj, qk):
        """计算中间控制点"""
        qi_inv = qi.inverse()
        qj_inv = qj.inverse()
        
        # 使用对数映射计算
        # 简化实现: 使用SLERP近似
        return quaternion_slerp(
            quaternion_slerp(qi, qk, 0.5),
            qj,
            0.5
        )
    
    a = intermediate(q0, q1, q2)
    b = intermediate(q1, q2, q3)
    
    # SQUAD = SLERP(SLERP(q1, q2, t), SLERP(a, b, t), 2t(1-t))
    slerp_q1_q2 = quaternion_slerp(q1, q2, t)
    slerp_a_b = quaternion_slerp(a, b, t)
    
    return quaternion_slerp(slerp_q1_q2, slerp_a_b, 2*t*(1-t))
```

### 6.5 插值方法对比表

| 方法 | 路径 | 速度 | 计算成本 | 适用场景 |
|-----|------|-----|---------|---------|
| **LERP** | 直线（球内） | 不均匀 | 低 | 不推荐 |
| **NLERP** | 近似球面 | 近似均匀 | 低 | 实时性要求高的场景 |
| **SLERP** | 大圆弧 | 恒定 | 中（含三角函数） | 两个旋转之间插值 |
| **SQUAD** | 平滑曲线 | 连续 | 高 | 多关键帧动画 |

### 7.1 3D图形渲染中的应用

#### 相机控制（ArcBall、FPS相机）

**ArcBall控制器**：使用四元数实现直观的3D旋转控制

```python
class ArcBall:
    """
    ArcBall相机控制器
    """
    def __init__(self):
        self.current_rotation = Quaternion(1, 0, 0, 0)
        self.start_rotation = Quaternion(1, 0, 0, 0)
    
    def screen_to_sphere(self, x, y, width, height):
        """
        将屏幕坐标映射到单位球面
        (x, y): 屏幕坐标，范围[-1, 1]
        """
        # 映射到[-1, 1]
        nx = 2 * x / width - 1
        ny = 1 - 2 * y / height  # Y轴翻转
        
        # 计算Z坐标
        length2 = nx*nx + ny*ny
        if length2 <= 1:
            nz = np.sqrt(1 - length2)
        else:
            # 在球外，投影到球面
            norm = np.sqrt(length2)
            nx /= norm
            ny /= norm
            nz = 0
        
        return np.array([nx, ny, nz])
    
    def begin_drag(self, x, y, width, height):
        """开始拖拽"""
        self.start_vector = self.screen_to_sphere(x, y, width, height)
        self.start_rotation = self.current_rotation
    
    def drag(self, x, y, width, height):
        """拖拽中"""
        current_vector = self.screen_to_sphere(x, y, width, height)
        
        # 计算旋转轴和角度
        axis = np.cross(self.start_vector, current_vector)
        axis_norm = np.linalg.norm(axis)
        
        if axis_norm > 1e-10:
            axis = axis / axis_norm
            # 点积给出cos(θ)
            dot = np.clip(np.dot(self.start_vector, current_vector), -1, 1)
            angle = np.arccos(dot)
            
            # 创建增量旋转四元数
            delta_q = quaternion_from_axis_angle(axis, angle)
            
            # 组合旋转
            self.current_rotation = delta_q * self.start_rotation
            self.current_rotation = self.current_rotation.normalize()
    
    def get_rotation_matrix(self):
        """获取当前旋转矩阵"""
        return quaternion_to_rotation_matrix(self.current_rotation)
```

#### 骨骼动画中的关节旋转

```python
class Joint:
    """
    骨骼动画关节
    """
    def __init__(self, name, parent=None):
        self.name = name
        self.parent = parent
        self.local_rotation = Quaternion(1, 0, 0, 0)
        self.local_position = np.array([0.0, 0.0, 0.0])
        self.children = []
    
    def get_global_rotation(self):
        """获取全局旋转"""
        if self.parent is None:
            return self.local_rotation
        return self.parent.get_global_rotation() * self.local_rotation
    
    def set_rotation_keyframe(self, rotation, time):
        """设置关键帧旋转"""
        # rotation可以是欧拉角、四元数或轴角
        if isinstance(rotation, np.ndarray) and rotation.shape == (3,):
            # 欧拉角
            roll, pitch, yaw = rotation
            self.local_rotation = euler_to_quaternion_zyx(roll, pitch, yaw)
        elif isinstance(rotation, Quaternion):
            self.local_rotation = rotation
    
    def interpolate_keyframes(self, keyframes, time):
        """
        在关键帧之间插值
        keyframes: [(time0, q0), (time1, q1), ...]
        """
        # 找到当前时间所在的区间
        for i in range(len(keyframes) - 1):
            t0, q0 = keyframes[i]
            t1, q1 = keyframes[i + 1]
            
            if t0 <= time <= t1:
                # 计算插值参数
                t = (time - t0) / (t1 - t0)
                # 使用SLERP插值
                self.local_rotation = quaternion_slerp(q0, q1, t)
                return
```

#### 物理引擎中的刚体旋转

```python
class RigidBody:
    """
    刚体物理对象
    """
    def __init__(self):
        self.position = np.array([0.0, 0.0, 0.0])
        self.orientation = Quaternion(1, 0, 0, 0)
        self.velocity = np.array([0.0, 0.0, 0.0])
        self.angular_velocity = np.array([0.0, 0.0, 0.0])  # 角速度 (rad/s)
        self.mass = 1.0
        self.inertia_tensor = np.eye(3)  # 惯性张量
    
    def integrate(self, dt):
        """
        积分更新位置和旋转
        使用半隐式欧拉法
        """
        # 更新位置
        self.position += self.velocity * dt
        
        # 更新旋转 (使用角速度)
        # ω = 角速度向量
        omega = self.angular_velocity
        omega_norm = np.linalg.norm(omega)
        
        if omega_norm > 1e-10:
            # 将角速度转换为轴角，然后转四元数
            # dq/dt = 0.5 * q * [0, ω]
            # 使用一阶近似
            axis = omega / omega_norm
            angle = omega_norm * dt
            delta_q = quaternion_from_axis_angle(axis, angle)
            
            self.orientation = delta_q * self.orientation
            self.orientation = self.orientation.normalize()
    
    def apply_torque(self, torque, dt):
        """
        施加力矩
        torque: 力矩向量 (3,)
        """
        # τ = I * α  =>  α = I^(-1) * τ
        angular_acceleration = np.linalg.solve(self.inertia_tensor, torque)
        self.angular_velocity += angular_acceleration * dt
```

### 7.3 航空航天中的姿态表示

```python
class AircraftAttitude:
    """
    飞行器姿态表示
    """
    def __init__(self):
        # 使用四元数存储姿态
        self.orientation = Quaternion(1, 0, 0, 0)
        # 角速度 (滚转率、俯仰率、偏航率)
        self.pqr = np.array([0.0, 0.0, 0.0])
    
    def update_from_gyro(self, p, q, r, dt):
        """
        根据陀螺仪数据更新姿态
        p, q, r: 滚转率、俯仰率、偏航率 (rad/s)
        """
        self.pqr = np.array([p, q, r])
        
        # 角速度向量
        omega = np.array([p, q, r])
        omega_norm = np.linalg.norm(omega)
        
        if omega_norm > 1e-10:
            axis = omega / omega_norm
            angle = omega_norm * dt
            delta_q = quaternion_from_axis_angle(axis, angle)
            self.orientation = delta_q * self.orientation
            self.orientation = self.orientation.normalize()
    
    def get_euler_angles(self):
        """获取欧拉角 (用于显示)"""
        roll, pitch, yaw = quaternion_to_euler_zyx(self.orientation)
        return np.degrees(roll), np.degrees(pitch), np.degrees(yaw)
    
    def get_direction_cosine_matrix(self):
        """获取方向余弦矩阵 (DCM)"""
        return quaternion_to_rotation_matrix(self.orientation)
```

### 7.4 性能考量（四元数 vs 旋转矩阵 运算量对比）

| 操作 | 旋转矩阵 | 四元数 | 优势 |
|-----|---------|-------|-----|
| **存储** | 9 floats | 4 floats | 四元数（节省55%） |
| **向量旋转** | 9 mul + 6 add | 15-18 mul + 12-15 add | 旋转矩阵 |
| **旋转组合** | 27 mul + 18 add | 16 mul + 12 add | 四元数 |
| **归一化** | 复杂（SVD） | 简单（sqrt + div） | 四元数 |
| **插值** | 困难 | 自然（SLERP） | 四元数 |
| **累积误差** | 大（需正交化） | 小（只需归一化） | 四元数 |

**结论**：
- **频繁组合旋转**：使用四元数
- **频繁变换向量**：使用旋转矩阵（或预计算）
- **骨骼动画/插值**：使用四元数
- **渲染管线**：转换为矩阵后使用
